青衫之友交流網－老遊戲攻略總集
注意事項
  1. 本站主要收錄2004年以前的PC中文單機遊戲，2005年以後或是英文遊戲就隨緣，2010年以後的新遊戲基本上就不收了。非PC、非單機或非中英文的遊戲必定不收。
  2. 本站提供的攻略，會儘可能鑽研出遊戲的各種詳細資訊，方便玩家研究與過關。遊戲名稱後面加註[待重整]者，為以前玩過但尚未重新深入研究並改寫的遊戲攻略，未來會陸續加以改寫。遊戲名稱後面加註[待補]者（沒有標示評價），為青衫持有但尚未玩的遊戲，以便了解還有多少遊戲尚未整理（持續增加中），雖沒有攻略資訊，但仍有啟動或破解、問題修正的相關資訊可參考。
待處理的遊戲
  1. 以下攻略均暫停，等整理完遊戲庫再繼續。預計大約需要4～10年的時間，可能在2028～2034年期間完成。屆時青衫也即將邁入古稀之年，能活多久都在未定之數。因此除了已答應的事項會在有生之年繼續陸續完成外，不再接受其他的預約（很可能實現不了了）。
  2. 正在改寫的遊戲攻略：魔法世紀1,2、鹿鼎記1,2、飛鷹騎士、仙劍2、仙劍3外傳、軒轅劍2、軒轅劍2外傳楓之舞
  3. 準備撰寫的遊戲攻略：大富翁4、仙劍奇俠傳3、軒轅劍3 Steam版、瘋狂主題樂園、暗黑破壞神1,2、黑暗王座2、柏德之門1,2
  4. 目前優先處理的遊戲類別：麻將、棋牌、益智
徵求遊戲或更新檔
  1. 待購光碟與徵求更新檔清單，歡迎提供或交流。
未解攻略疑惑
  1. 台灣麻將全民開打：「鍾沒情」對應的政治人物。
  2. 聲控麻將王、麻將教師：大智若愚、過門不入、意亂神迷秘技作用。
  3. 蜘蛛女郎撲克：實際的接龍規則（說明中的規則似乎有誤）。
  4. 激鬥戰士：歐魯達、沙奇爾部份招式的按鍵正確組合。
