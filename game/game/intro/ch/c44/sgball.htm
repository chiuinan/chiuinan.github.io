名稱：三國連環炮 (中文版)
簡介：極真科技2005年出品的益智遊戲，玩家需在球移至城門前清光，方式為控制方向射出
      球，只要有3個以上同色球相連，即可消去 [待補]。



保護：光碟，破解碼：
      Ball.exe
        83 F8 05 74 20
        -- -- -- EB --
      必須拷貝光碟中的Anim、Datas、Mus目錄到安裝目錄裡。另外，GAME.CFG開頭需改成
      2E 5C 00，將光碟目錄換成目前目錄。
檔案：patch101.rar = 1.01更新檔
操作：

滑鼠左鍵 = 射出球
滑鼠右鍵 = 更換目前待射球的顏色
