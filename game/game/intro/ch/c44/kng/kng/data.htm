主角屬性
對自己按下滑鼠右鍵，可出現人物履歷表視窗：

（一）資料

年齡：主角的年紀。某些職業會有年齡的限制。
相貌：高顏值對於從事演藝圈演員工作時，較佔優勢。某些女性也只喜歡帥哥。若不想進入
      演藝圈，或追求喜好外貌的女性，這個值其實是不用在意的。
身高、體重：會隨年齡增長而改變，上健身房運動也可以改善此數值。良好的身材對於演員
      來說，會比較佔優勢。
體力：上限值為健康值*100。超時工作時體力會下降，若長期體力低落，便會影響到健康值，
      因此遊戲過程中儘量不要讓主角過度操勞。
學歷：可在成就網中直接升學，或利用生活圈的補校補修學歷。許多職業有一定的學歷要求，
      最好是在成就網中直接升學會比較快些。
職業：主角目前從事的職業，可到生活圈的職業介紹所進行變更。

（二）能力

學習：會影響主角在學習國文、英文、數學與教育技能時的速度。這個部份在高中畢業前會
      影響到學習的速度，到大學或工作後，除非是學術界或是一些特殊的行業，否則並沒
      有太大的影響。
創作：對於培養貿易公司開發人員與演藝圈幕後人員的專業技能非常重要。
表達：演技、教育、政治和商務都需要較高的表達力。
領導：任何領域的管理階級都需要，也是政治家、導演重要的能力之一。
親和：對於教育者、政治家或任何管理者都極為重要。
應變：會影響學習演技、商務、球技時的效率。
健康：會影響體力上限值，也是鍛練球技時的重要條件之一。30歲後逐年減一。
意志：對於棒球選手或軍人特別重要。

（四）成就

快樂：初值1000，為人生目標之一。
名譽：初值1000，為人生目標之一。
智識：初值0，為人生目標之一。
現金：初值1000，為人生目標之一，演藝界製片與電影公司老闆必備的條件。
活期儲蓄：在銀行的活期存款金額。
定期存款：在銀行的3年定存金額。
銀行借貸：向銀行借貸的金額。
親友借貸：向親友借貸的總金額。

（五）經驗

括號中為該技能提升時，所依據的人物能力值，多個能力則取其平均值。

國文（學習）：升學的基本條件，演藝界幕後人員與政治界部份職業的技能。
英文（學習）：升學的基本條件，貿易公司業務部的技能。
數學（學習）：升學的基本條件，貿易公司財務部的技能。
教育（學習、親和、表達）：學術界的專業技能。
商務（應變、表達）：貿易公司的專業技能。
管理（領導、親和）：所有管理階級均必備的技能。
政治（領導、表達、親和）：政治界的專業技能。
球技（健康、應變、意志）：棒球選手的專業技能。
設計（創作）：貿易公司設計開發部門的專業技能。
戰技（意志）：軍人的的專業技能。
演技（應變、表達）：演藝界演員的專業技能。
編導（創作、領導）：演藝界幕後人員的專業技能。
聲望：唯一無法於任何場所提升的屬性，會隨著現有成就而日益增長。由於並沒有靠聲望值
      高低決定的職業，因此可以不用管這個屬性值。
