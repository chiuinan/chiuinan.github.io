操作：

M = 船艦移動路線

R = 派出偵察機
O = 設定空偵路線
P = 設定空攻路線
S = 搜索記錄
D = 分編
J = 合流
B = 開始戰鬥
C = 命令取消
I = 檢視游標所在我方機船艦或基地狀況
U = 檢視敵我艦隊大本營狀況
F1～F4 = 地圖比例切換
F5 = 檔案功能表
F6 = 統帥功能表
F7 = 戰力功能表
F8 = 行動功能表
F9 = 戰術功能表
F10 = 設定功能表
