名稱：洛克人X1 (Rockman X／Mega Man X，中文／英文版)
簡介：CAPCOM 1993年出品的動作遊戲，屬遊戲系列中的「X系列」分支。1995年移植至PC，
      2018年經由Steam重新移植，收錄在「遺產合集（Legacy Collection）」中 [待補]。
 
備註：本版為Steam合集版（缺動畫，無法直接截圖，可使用Fraps軟體）。
保護：原始版無保護
檔案：mmx1-dos.rar = 原始英文DOS版
操作：

一、DOS版配置

Space = 射擊（按久一點為集氣）
Alt = 跳躍
Ctrl = 衝刺（持有腳部配件時）
Tab = 變更武器
Esc = 選擇武器
F10 = 系統選單

二、Steam版配置

X = 射擊（按久一點為集氣）
Z = 跳躍
A = 衝刺（持有腳部配件時）
D/C = 變更武器
Space = 選擇武器
Tab = 系統選單
