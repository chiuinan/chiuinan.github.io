名稱：e路狂飆 (Speed Mania，中文版)
簡介：瑪吉斯2001年出品的賽車遊戲，最初只有「陽金公路篇」。2002年、2003年分別推出
      「中橫公路篇」、「屏鵝公路篇（墾丁杯挑戰賽）」資料片 [待補]。



備註：本版為陽金公路篇，尚未安裝資料片，目前缺主程式與屏鵝公路篇光碟。
保護：無
備註：若主程式（陽金公路篇）不是安裝在內定目錄，資料片安裝時要選擇自訂，然後變更
      主程式實際安裝的目錄，否則會直接安裝在內定目錄裡。
備註：遊戲需以"vp4x.exe" "SpeedMania_menu.vmo" -f -b=32啟動。
操作：

左右 = 左右轉
上下 = 換檔
A = 加速
Z = 剎車
X = 倒車
S = 手剎車
