名稱：e路狂飆 (Speed Mania，中文版)
簡介：瑪吉斯2001年出品的賽車遊戲，最初只有「陽金公路篇」。2002年、2003年分別推出
      「中橫公路篇」、「屏鵝公路篇（墾丁杯挑戰賽）」資料片 [待補]。



保護：無
備註：若主程式（陽金公路篇）不是安裝在內定目錄，資料片安裝時要選擇自訂，然後變更
      主程式實際安裝的目錄，否則會直接安裝在內定目錄裡。
備註：遊戲需以"vp4x.exe" "SpeedMania_menu.vmo" -f -b=32啟動。
備註：本遊戲另有中華汽車限量典藏版，內含主程式+兩個資料片，但陽金公路只有二、三段
      （少了一段）；中橫公路只有一、二、三段（少了四段）；屏鵝公路只有二段（少了
      一、三段）。
操作：

左右 = 左右轉
上下 = 換檔
A = 加速
Z = 剎車
X = 倒車
S = 手剎車
1 = 第一人稱視線
2 = 後側視線
3 = 左前方視線
4 = 右前方視線
G = 儀錶開關
T = 計時器開關
M = 小地圖開關
B = 後照鏡開關
O = 過油打滑開關
F = 特殊景物開關
R = 車輛重置（翻車或卡住時使用）
Pause = 暫停／繼續
Esc = 離開遊戲
