名稱：電梯大作戰／電梯迷宮／電梯大盜／電梯大戰 (Elevator Action，中文版)
簡介：TAITO 1983年出品的動作遊戲，最初在大型機台上，其後移植到各平台。2000年時由
      普威爾中文化並代理發行。玩家扮演一名間諜，潛入電梯大樓，竊取所有機密文件並
      離開 [待補]。



保護：無
備註：遊戲安裝完畢後，還要執行光碟上的chinese.exe進行中文化，才會有中文界面。
操作：

X = 射擊
Z = 跳躍
F2 = 增加打數（COIN數）
F3 = 開始單人遊戲
F4 = 開始雙人遊戲（非同時進行，而是玩家1先玩，結束後換玩家2再玩）
F9 = 重新開始遊戲
F10 = 暫停遊戲
F11 = 視窗模式／全螢幕模式切換
Alt-F4 = 結束遊戲
