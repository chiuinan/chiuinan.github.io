名稱：勇者泡泡龍 (Brave Bubble Dragon，中文版)
簡介：龍愛1997年出品的動作射擊遊戲，遊戲的規則便是讓主角吐泡泡將敵人
      包住後，再將其碰撞消滅，當所有敵人都消滅後，便算過關了。泡泡龍
      這類的遊戲常見於電動遊樂場，有逛過的玩家應該不陌生才對。遊戲中
      一共有６大關，每大關有６～１２不等的小關，最後一小關為魔王。魔
      王都必須拿到上方的特殊瓶子，吹出特殊的氣泡才能加以消滅，因此非
      常難打，而且有些小關的地形設計得變態，要全部過關是非常困難的一
      件事。不過整體而言，這款遊戲還算不錯，當做逍遣輕鬆地玩即可。

資訊：

所有關卡的數目、接續，以及隱藏關卡的條件等，均寫死在程式中，無法變動。

DS:466Ah = 目前的大關卡
DS:466Ch = 目前的小關卡
DS:8184h = 主角X位置，2 byte
DS:8186h = 主角Y位置，2 byte
DS:8E1Eh = 進入隱藏關的泡泡累積次數
DS:4674h = 本關是否進入結束倒數
DS:4676h = 結束前的等待時間，由0累積到100h
DS:8E86h = 本關是否結束

氣泡資訊： XX XX YY YY TT TT
  XX,YY = 位置
  TT = 可捕捉敵人的剩餘時間

密技：

暫停遊戲後，按下：

F1+F3：無敵（每一關都要按一次）
F2：取消無敵
F8+D：無限接關

操作：

左右鍵 = 移動
Ctrl或上鍵 = 跳躍
Alt或Space = 吐泡泡
Esc = 系統選單（音樂開關、離開遊戲）
P = 暫停遊戲／開始遊戲

攻略：（作者－青衫）

1.主角吐泡泡將敵人包住後，碰撞該泡泡即可消滅敵人。
2.消滅所有敵人，即過關。
3.一定時間未消滅所有敵人，則會出現無法消滅的鬼魂出來攻擊。
4.最後一小關的魔王必須先取得魔法泡泡瓶，利用該泡泡釋放出來的魔法（如
  閃電、龍捲風、烈火等）進行攻擊，才能消滅魔王。
