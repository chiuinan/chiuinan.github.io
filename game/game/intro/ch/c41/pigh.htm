名稱：拱豬大賽 (Pig Heart，簡中/英文版)
簡介：北京世紀鼎點1997年出品的拱豬撲克牌遊戲，且採傳統破底定勝負的方
      式相同。亦即玩前設定一負分限制，當任一玩家超過該負分時，遊戲即
      結束，而該破底的玩家便是最輸。整個遊戲相當簡單，可供喜好此類遊
      戲的玩家消磨時間之用。不過本遊戲在豬羊變色後，豬羊的正負分數並
      沒有顛倒過來，為很可惜的一個缺失。



保護：無
備註：本遊戲會依作業系統的語系，自動採用英文或簡體中文的界面。
說明：（作者－青衫）

拱豬遊戲規則如下：

1.四人使用不含鬼牌的52張牌玩，每人發13張，並以順時針方向輪流出牌。
2.每局一開始由持有♠7的人為莊家先行出牌，之後由贏得該墩的人為下一墩的
  莊家，先行出牌。
3.閒家出牌時，必須出和莊家相同花色的牌，並依照牌點大小順序決定輸贏：

    A＞K＞Q＞J＞10＞9＞8＞7＞6＞5＞4＞3＞2

  贏家可獲得該墩的四張牌。若手上沒有相同花色的牌時，可以出任何牌做為
  墊牌，視同輸掉該墩。
4.全部牌出完後，依照各家贏取的牌計算該局得分。
5.最終勝負的決定方法有三種：
  (1)破底：設定一負分數，累積分數先超過該負分者為輸家。
  (2)攻頂：設定一正分數，累積分數先超過該正分者為贏家。
  (3)無勝負：通常由該局最輸的人洗牌、發牌，以做為懲罰。

計分規則如下：

1.♥：負分牌，各牌分數略異（後述）。全吃到稱為豬羊變色，可獲得正200分。
2.♠Q：豬，負100分。豬羊變色後為正100分。
3.♦J：羊，正100分。豬羊變色後為負100分。
4.♣10：得分加倍。若無分數牌，則為正50分。
5.吃到所有分數牌與♣10稱為大滿貫，可獲得正800分（或正1000分）。
6.吃到所有分數牌稱為小滿貫，可獲得正400分（或正500分），但有些規則無
  小滿貫。

負分牌有下列幾種計分方式：

(1)傳統計分法（總分194）

2～10 = 負2分到負10分
    J = 負20分
    Q = 負30分
    K = 負40分
    A = 負50分

(2)第一種新計分法，將不足分補到♥4（總分200）：

    2 = 負2分
    3 = 負3分
    4 = 負10分
5～10 = 負5分到負10分
    J = 負20分
    Q = 負30分
    K = 負40分
    A = 負50分

(3)第二種新計分法，將不足分補到♥9與♥10（總分200）：

2～ 8 = 負2分到負8分
    9 = 負10分
   10 = 負15分
    J = 負20分
    Q = 負30分
    K = 負40分
    A = 負50分

(4)簡化計分法（總分200），為本遊戲採用的計分法：

2～ 4 = 0分
5～10 = 負10分
    J = 負20分
    Q = 負30分
    K = 負40分
    A = 負50分

本遊戲在發牌過程中尚有所謂的亮牌：

1.♥A：亮牌後，所有紅桃的分數加倍。
2.♠Q：亮牌後，豬的分數加倍（負200分）。
3.♦J：亮牌後，羊的分數加倍（正200分）。
4.♣10：亮牌後，♣10變成得分乘4倍，或分數為正100分。
