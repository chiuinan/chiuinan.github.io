名稱：暗棋聖手 (中文版)
簡介：至通1995年出品的象棋暗棋遊戲，即各子均一次走一步，大小則依將士象車馬兵排列，
      包需隔子吃子之暗棋遊戲。遊戲中可兩人對下，亦可單人採用闖關的方式進行，玩者
      必須連闖15關獲得最後的勝利。而遊戲中提供了不少秘技與道具可使用，增加了不少
      趣味，只可惜對弈過程無法隨時存檔，必須在晉級成功後才能存檔，比較不方便些。
      不過整體而言，這款遊戲還蠻不錯的，有興趣的玩家可以玩玩看。



備註：本版已修正錯誤遊戲初始資料，以及使用秘技選擇對應不正確的問題
保護：圖形密碼，破解碼：
      CHESS.EXE (先用UNP解壓)
        83 3E B1 00 00 75 05
        -- -- -- -- -- EB --
備註：直接用DosBox玩時，遊戲會沒有AI（常會下出爛棋），且思考時間會很久，這是因為
      遊戲的AI程序在DosBox裡無法啟用，遊戲一直在等AI程序回應，結果最後逾時隨便下。
      最好採用DosBox Boot方式啟動來改善此一情況，也就是讓原本的MS-DOS系統來取代
      DosBox的DOS系統，以便成功啟用遊戲的AI程序。例如將遊戲做成開機硬碟檔chess.img，
      然後再以下列方式Boot後，才開始遊戲：
        imgmount c .\chess.img
        boot -l c
      成功啟用AI程序後，遊戲的下棋速度明顯快很多，也比較不會下爛棋。

說明：

*** 新遊戲選項

1.賽制
  單人晉級制：晉級後才能存檔，並獲得對手的金錢。
    標準棋盤32格：可使用秘技和法寶，法寶必須事先購買。
    特殊棋盤36格：同上，多出的4格藏有寶物（必須視同敵子吃掉才能獲得）。
  雙人對抗制：
    標準棋盤32格：無法使用秘技和法寶。
2.時間限制：分成30秒、60秒、不限三種。當有時間限制時，超過時間未走子，即視同犯規，
  依後述犯規規定處理。
3.犯規次數：分成3次、7次、不限三種。當有犯規時，超過犯規次數即該局判負，否則便輪
  對方走子（失去走子權利）。
4.勝負場次：分成3場、5場、不限（僅限雙人對抗制）三種。當取得指定的勝利場次時，即
  晉級成功；若失敗達到指定的場次，則晉級失敗。
5.和局設定：分成20步、40步（實際為50步）兩種。在指定的步數內若無勢力消長，即判定
  為和局。
