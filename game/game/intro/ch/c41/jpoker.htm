名稱：美少女撲克 (Girl Poker，中文版)
簡介：大宇1989年出品的撲克牌梭哈遊戲，規則為每人先發５張，然後下注後
      再決定要換那幾張牌，以更換後的５張為準與對手相比決定輸贏。遊戲
      的目標便是挑戰４位美女，當任一美女的錢輸光時，便會顯示一張非限
      制級的美女圖以換取１００元籌碼，不過美女圖實在不怎麼好看。此外，
      電腦有作弊之嫌，使得要贏對手實非易事，常會落得失敗重來。



保護：顏色密碼，破解碼：

操作：

Space = 選擇選項
Enter = 選擇結束或放棄選擇
Ctrl-Q離開

說明：（作者－青衫）

1.初始賭注5元
2.第一次發完牌，會有下列按鍵選擇：
    B = 下賭注，可選擇05、10、15、20、25（對方可能加賭注）
    S = 掀牌，沒有換牌機會，直接比大小（對方可能加賭注）
    D = 放棄，直接認輸
    M = 用密技（只能一次）
3.密技有下列按鍵選擇（注意不是使用Enter選擇）：
    A = 偷看牌，需$25，看一次對方的牌
    B = 透視，需$50，整局對方的牌都翻開
    C = 偷換牌，$75，更換一次牌
    D = 偷調牌，$100，直接選一張想要的牌，上下鍵可換花色
    Enter = 放棄使用密技（該局無法再使用）
  使用密技，對方有可能會抓包並將賭金沒收。
4.對方加賭注時，我方有下列按鍵選擇：
    R = 加賭注，可選擇05、10、15、20、25（對方可能再加賭注）
    C = 跟牌，繼續下一步（換牌或掀牌）
    D = 放棄，直接認輸
5.雙方互加賭注，最多4次。
6.有多加賭注才有一次換牌機會，包括我方下賭注、對方加賭注我方跟牌等。
7.換牌時，使用Space切換是否換牌，有Change字樣表示要更換；Enter表示選
  擇結束。
8.換牌後會再一次下賭注，同第2點，只是無密技選項
