名稱：四玲瓏 (中文版)
簡介：尋夢園1992年出品的四子棋遊戲。遊戲規則是在７ｘ６的棋盤上，以縱
      軸為準，輪流堆上一個己方的子，只要己方的子在橫、直或斜角形成四
      子連線，即算獲勝。遊戲裡提供了兩種模式，一個是四玲瓏教室，可以
      雙人對戰或人機對戰，隨時可回手或重來；另一個是挑戰玲瓏軒，採用
      闖關方式挑戰不同IQ的電腦人物，以前進到最終目標，進入排行榜。雖
      是一個簡單有趣的小品益智遊戲，但本遊戲的棋力並不低，值得喜歡動
      腦的玩家來挑戰。

  SETUP = 設定
  GIVEUP = 重玩
  OOPS = 回上一手
  EXIT = 離開
2.挑戰玲瓏軒：

  左下箭頭 = 起始位置（按下滑鼠左鍵可存檔或離開）
  右上GOAL = 目標位置
  灰色 = 尚未挑戰的人物（挑戰失敗會消失，無法通過）
  綠色 = 挑戰成功的人物
  紫色 = 目前位置（已挑戰成功，可變動）

  功能圖1 = 和局重玩（換人先手）
  功能圖2 = 增加獲勝局數
  功能圖3 = 增加總局數
  功能圖4 = 混亂對手思緒（效果不明顯）
  功能圖5 = 更換對手

  (1)活二：連續或隔空兩子在連線路徑上都是空的可著點，是變成活三必殺棋

  (2)活三：連續三子在連線路徑上左右兩邊都是空的可著點，因為無法同時擋
     住兩邊，死棋。

  (3)死三：可著點落子就可連成四子的情況，為必擋的著點。

  (4)等四：差一子即可連成四子，而該子著點尚未成為可著點，為控制對手的
     勝利點。例如下圖位置(2,4)即為綠方的勝利點：

  (5)連續等四：勝利點為上下連續位置，因為無法一次擋住兩個等四，死棋。


4.己方可做成等四時要優先做出，每個等四所缺的著點，都是己方的勝利點。
  勝利點愈多的一方愈有利，因此也要儘量不讓對手做出等四的情況。如果己
  方等四的著點位置有重複，便不必再做出，改著其他更好的著點。
5.一般玩家常見的獲勝方法有：
  (1)連續等四：只會出現在與低IQ的人物比賽中，高IQ的人物都會事先預防。
  (2)逼迫著子：在與IQ較高的人物比賽中，幾乎都是以此決勝負。因為其他地
     方著子皆滿，不得不下出讓另一方獲勝的著手。先手的一方要儘量將勝利
     點佈在奇數行，後手的一方則要佈在偶數行。
