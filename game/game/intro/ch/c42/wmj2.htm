名稱：政治麻將2 (中文版)
簡介：極真科技2001年出品的16張四人麻將遊戲，採用台灣麻將規則計台，並
      可自行定義。本遊戲以當時最熱門的22位政治人物為主角。遊戲目標為
      在一雀麻將內，提升自己籌碼以打入排行榜，和一代遊戲相同，有點過
      於單調。雖然有分單人賽與團體賽，但目標並沒有變化。一代語音停頓
      的缺點依然沒有改善，而且還取消一代的秘招，娛樂性下降不少。此外，
      對手有偷看之嫌，例如單吊時看到對手打出某張牌，表示該張牌是無用
      的，但讀檔改聽該張牌，在全無人吃碰牌的情況下，對手便不再打出該
      張牌放槍，很明顯是電腦在作弊。同樣情況也常發生在對手單吊自摸的
      比例異常地高，表示也在偷看海底牌。玩家只能透過存取檔來改變劣勢，
      否則正常地玩，勝率並不高。因此整體而言，這款遊戲的娛樂性不算是
      很好，可玩可不玩。



保護：光碟，破解碼：
      WinMJ2.exe
        83 F8 05 74 1B
        -- -- -- EB --
      必須拷貝光碟中的Anim和Mus目錄到安裝目錄裡。另外，TRUETECH.CFG
      開頭需改成2E 5C 00，將光碟目錄換成目前目錄。
備註：若無法執行，或執行時出現"ICDecompressOpen error"，請將執行檔的
      相容性設成XP相容。

修改：存檔????.SAV

位置0164h = 己方的牌組，每個6 byte
位置0350h = 己方籌碼，4 byte
位置056Ch = 下家籌碼，4 byte
位置0788h = 對家籌碼，4 byte
位置09A4h = 上家籌碼，4 byte

牌組資訊：

NN -- CC -- -- --

NN：數字，0～8表示數字1～9，0～6=東南西北中發白
CC：牌色，0=萬、1=條、2=筒、3=字

說明：

1.個人賽為選人，團體賽為選黨，目標都是一樣，在一雀內獲取最多的籌碼，
  打入排行榜。
2.可選擇的政黨與人物：
  民進黨：陳水扁、呂秀蓮、蘇貞昌、謝長廷
  國民黨：連戰、馬英九、朱立倫、胡志強
  親民黨：宋楚瑜、張昭雄、李慶安、秦慧珠
  外國人：布希、賓拉登、海珊、小泉純一郎
  名嘴：陳文茜、汪笨湖、趙少康、鄭弘儀
  台聯：李登輝、黃主文
3.初始籌碼1000000，底/台可調整，內定10000/2000。
4.打完一雀或有人籌碼歸零時結束，統計籌碼決定名次，及是否進入排行榜。
  但團體賽籌碼歸零時不結束，改換該黨下一人物繼續。
