      戲提供了自由對戰的切磋牌技，以及劇情模式的雀王爭霸兩種玩法，且

      能，因此影響並不算大。

資訊：雀王爭霸劇情控制檔script\play\C?0s.csc

?=0～b，代表星仔～校園之花共12位人物。控制資訊由位置100h開始，由多個
命令集所組成，每個命令集的格式為：

  命令集字串 + 多個控制命令

命令集字串一律4個字元，格式為sl??，??為兩個數字。以下為各字串意義：

  sl00 = 開始
  sl11 = 結束
  sl12 = 定義主角人物
  sl13 = 條件執行
  sl14 = 定義頭像
  sl15 = 除去頭像
  sl16 = 顯示文字(指定角色)
  sl17 = 顯示文字(不指定角色)
  sl18 = 指定執行條件
  sl19 = 變更頭像

控制命令的格式為：

  1 byte命令碼 + 命令資料

主要的命令碼有：

  00 = 結束
  01 + 4 byte頭像位置 = 定義人物頭像
  03 + 4 byte頭像位置 = 改變人物頭像
  04 + 4 byte字串大小 + 字串（含00h字元） = 文字命令／顯示文字

文字命令（04命令碼）有：

  c?0 = 指定人物，?=0～b，分別代表星仔～校園之花的12位人物，若是c則
        表示文字方框
  sk1? = 學會秘技，?=0～e，分別表示15個秘技
  cuin = 顯示頭像
  cout = 移除頭像
  norm = 以正常方式顯示文字
