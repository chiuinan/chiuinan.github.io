名稱：四川省 (中文版)
簡介：智冠1990年出品的１８禁四川省麻將遊戲，玩者必須在限時內拿掉所有
      可兩折（三線段）內連接的相同牌組，遊戲中也提供了一些輔助過關的
      功能。當整個牌組解開後，便會顯示一張美女圖做為獎賞。遊戲一共有
      １０個關卡，而且愈到後面限時愈短，不過還不致於無法過關。



保護：圖形數字密碼，破解碼不明。
備註：遊戲結束時會重設環境檔AY&LCC.ALJ為無音樂，因此每次玩前都必須執
      行SETUP.EXE重設環境，否則會沒有音樂。或是將已經設定好的環境檔
      AY&LCC.ALJ備份起來，要玩前再將之還原。

操作：

1.輸入完密碼數字後按Enter鍵
2.遊戲控制鍵：
  Space = 定牌／取牌
  Enter = 取消選擇
  U = 回手
  P = 暫停（但無法看牌）
  / = 音效開關（無音效時處理速度較快）
  Q或Shift-Q（大寫字母Q）= 結束遊戲
3.八卦鍵按鍵（每按一次減少一個八卦）：
  1 = 使用密技：牌形重組
  2 = 使用密技：電腦自動消去3組牌
  3 = 使用密技：任意選取1組牌消去

備註：CG圖無露點。
提示：

1.下方的珠子代表剩餘時間，過關時剩餘時間愈多，得分愈高。
2.每過一段時間，珠子便會減少一顆，而每消去一個牌組，便會增加一顆珠子。
3.初始有8個八卦，每過一關增加一個（最多8個）。
