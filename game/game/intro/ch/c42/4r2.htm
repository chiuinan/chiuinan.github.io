名稱：四川省２ (中文版)
簡介：智冠1992年出品的１８禁四川省麻將遊戲，玩者必須在限時內拿掉所有
      可兩折（三線段）內連接的相同牌組。除了基本的麻將牌組外，還有特
      殊的牌組，拿掉時會隨機出現額外的牌，或增加魔法值。此外，遊戲中
      也加了許多輔助過關的功能，但必須消耗掉魔法值。當整個牌組解開後，
      其底圖即為獎賞用的美女圖。遊戲一共有２０個關卡，而且愈到後面限
      時愈短，短到有點變態，因此要過關變得非常困難。
 
保護：磁片，破解碼：
      4R2.LY
        D2 DB 12 9A E8 11...D2 D7 95 9A 03 C9 67
        -- -- -- -- EF --   57 -- -- -- EF -- --
      然後將之改成.EXE直接執行即可。此改法會造成常遇到除以0錯誤，以
      及下一關背景圖花掉，建議搭配額外的Disk Hunter程式（DHP2.EXE）。
備註：若遊戲執行後沒有音樂，請調降執行速度，或重新執行看看。

操作：

1.遊戲控制鍵：
  Space = 定牌／取牌
  U = 取消選擇
  . = 音樂開關
  / = 音效開關
  Esc = 密技按鍵說明
  E = 結束遊戲
2.密技按鍵（每按一次魔法點數減少一）：
  A = 消牌大法（電腦自動消去2組牌）
  C = 易位大法（更換所有麻將位置）
  B = 霹靂雷電（任意選取1組牌消去）
  M = 移位大法（由電腦任意更換部份麻將位置）

備註：CG圖有露點。
提示：

1.左下炸彈引線代表剩餘時間，過關時剩餘時間愈多，得分愈高。
2.消去骷髏頭牌組，麻將牌會增加，但若是最後一組便不會，因此最好儘量將
  骷髏頭牌組保留到最後再消除（除非想累積更高分）。
3.每消去一個牌組，會增加一些時間，且分數也會增加，而每過一關還有拉Bar
  隨機加分。若3個數字相同，便可再多一次拉Bar隨機加分。
4.初始魔法點數為10，總分每超過600分的倍數，魔法點數便增加一。當消去+1
  牌組時，魔法點數也會增加一。
