名稱：天狐16張 (中文版)
簡介：伊思儷2003年出品的麻將遊戲，以台灣常見的16張麻將玩法計台。遊戲
      提供了五位辣妹做為對手打牌，當玩家胡牌時，辣妹會依照籌碼的多寡
      顯示一定程度的性感圖片。只是這些辣妹的長相平平，並不是那麼吸引
      人。遊戲中的最大缺點便是考慮時限很短，玩家必須很快做出決定，玩
      起來會有一定的壓力。而遊戲的功能也很陽春，並沒有任何的道具或秘
      技做為輔助，再加上電腦仍有不明顯的作弊嫌疑，因此玩家要胡牌得靠
      運氣。幸好遊戲可隨時存檔，可以讀檔後換個打法試試，因此玩家也不
      全然居於劣勢。而遊戲最大的優點便是仿造出實際打牌的場景，看起來
      比較親切。綜合而言，這款遊戲仍有一定的娛樂性，玩家也可試試。



保護：序號：

        P3669-B36HB-GL552-H84G1-2
        HQ1N1-31K52-CCC50-3080G-0
        W1C71-R3WQD-8DQ70-8DQ70-80VQK-2
修改：存檔MJ_Save?.t，每把第一次要摸牌前存檔：
        0004h = 桌上目前還可以摸的牌數
        0008h = 下一張摸牌，2 byte
        00ACh～00CBh = 自己手上的牌，每張牌2 byte
        01CEh = 己方籌碼
        01D2h,01D6h,01DAh = 另三家籌碼
      將另三家籌碼設為0，再把自己手上的牌每3張設成一樣，最後一張設成
      下一張摸牌，讀檔後便可立即自摸，馬上看圖。
操作：右下左邊按鈕為系統選單，右邊按鈕為結束離開
備註：CG圖有露點
說明：

1.遊戲固定底300，台100，籌碼10000。
2.發牌後，所有牌均已固定（包括未摸的牌），此時可先存檔，若無法胡牌，
  便讀檔後換另外一種打法試試。
