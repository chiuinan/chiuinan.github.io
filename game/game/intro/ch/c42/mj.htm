名稱：歡樂麻將 (中文版)
簡介：大宇1990年出品的１３張麻將遊戲，採日本規則計台。遊戲的目的主要
      是挑戰８位對手，順利取得畢業證書。雖然進牌與胡牌率稍低，但因遊
      戲中可利用學分換取特殊牌型，也有存檔的功能，因此要過關其實並不
      困難。



保護：手冊密碼，破解碼：
      MJ.EXE
        E0 1C 74 03 (此破法答什麼均正確，初始一律為六學分)
        -- -- EB --
      或
        83 3E F6 0A 02 7E 03 (此破法答題未必正確，初始最低為三學分)
        C7 06 -- -- 03 00 90
      或
        E8 45 2A (跳過答題)
        90 90 90
        83 3E F6 0A 02 7E 03 (此破法可一開始即有9987個學分)
        B0 03 A3 F6 0A 90 90

修改：記憶體

不打牌時和存檔相同，但打牌時為第三個，和存檔格式不同：

M1 M1 M2 M2

M1 = 己方金錢/10
M2 = 對手金錢/10

學分數在後面70h的位置。

操作：上鍵＝叫出選單
      Ctrl-Q = 離開遊戲

攻略：（作者－青衫）

1.一開始詢問密碼時，每答對一題加二學分。
2.每打完一把牌加一學分。
3.有九學分以上時，可選擇使用下列功能：
  打牌特權（扣一學分）：換牌權、看牌權
  滿貫牌形（扣四學分）：清一色、四暗刻、清老頭、國士無雙、字一色
