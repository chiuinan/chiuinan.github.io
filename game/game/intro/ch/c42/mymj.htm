名稱：我的野蠻麻將 (中文版)
簡介：尼奧科技2003年出品的麻將遊戲，由智冠代理發行，以台灣常見的16張
      麻將玩法計台。遊戲提供了自定牌局和劇情麻將兩種玩法，並提供8位
      人物角色、加上前作少林麻將12位人物角色可供選擇，每個角色都各有
      其對應的語音。事實上，本遊戲打麻將的畫面與操作方式都與少林麻將
      相同，其缺點也相同，就是角色在播放語音時，會中斷牌局的進行，不
      過更新檔已修正此問題，語音不再中斷牌局，打牌過程比少林麻將還要
      順暢。至於遊戲提供的卡片系統並不是很好用，反不如少林麻將的密技。
      本遊戲最讓人喜歡的，還是聽牌時的瞇牌動作。此外，可以自己設定計
      台規則，以及提供胡牌大考驗、小麻雀兩種小遊戲，都還有再加到一點
      分數，因此還算是不錯玩的麻將遊戲。

位置0004h開始9 byte為目前現有的卡片：
  30h = 無
  31h = 加倍金卡
  32h = 免錢金卡
  33h = 加台銀卡
  34h = 減台銀卡
  35h = 二倍紅卡
  36h = 三倍紅卡
位置005Ch為主角總金錢數（自定牌局時）
位置2DF0h開始為四家資訊（主角固定第一家），每家50h=80 byte資料，第二
  個4 byte為剩餘錢數，將任一對手修改成0後進入遊戲，對局即結束。

密技：因系統會自動存檔，故起手牌不好、或打了半天不進張時，可回主選單
      後再進入，重新再玩一次。
操作：按Alt-Enter可切換成全螢幕／視窗模式
說明：

自摸時，將隨機得到一張卡片，最多可累積到九張：

1.加倍金卡：使用後，如果胡牌，由系統額外支付一倍的錢給您。
2.免錢金卡：如果這局您被胡輸錢，則不需要自己出錢，由系統幫您支付。
3.加台銀卡：如果這局您胡牌，除了正常的台數之外，系統會多給您一台的錢。
4.減台銀卡：如果這局您不幸被胡牌，系統會幫您出一台的錢。
5.二倍紅卡：把輸贏的錢數目變成二倍。
6.三倍紅卡：把輸贏的錢數目變成三倍。

劇情麻將為無厘頭的單線劇情，隨著打牌獲勝繼續展開（共有７關）。
