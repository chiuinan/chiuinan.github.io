名稱：四川省３天府之國 (中文版)
簡介：第三波1993年出品的１８禁四川省麻將遊戲，玩者必須在限時內拿掉所
      有相鄰、斜線以及可直線兩折（三線段）內連接的相同牌組。除了基本
      的麻將牌組外，還有特殊的牌組，拿掉時會隨機出現重組功能。此外，
      遊戲中也加了許多輔助過關的功能，但會消耗掉一些時間。當整個牌組
      解開後，便會顯示一個不怎麼好看的美女圖做為獎賞。遊戲一共有３６
      張美女圖，玩者必須在８關內挑選要挑戰的關卡（即美女圖），以最後
      的得分做為排行榜依據。整體而言，這個遊戲算是相當簡單，除了難聽
      的音樂和難看的美女圖外，大致沒什麼缺點，但好像也沒什麼特殊的優
      點，算是一個蠻普通的遊戲吧。

密技：

1.時間增加法
  遊戲中按下 0、1 和 F9 則時間會亂跳，再按一下時間則多的離譜。
2.自動消牌
  在遊戲中按下Esc、L及C鍵，可以看到電腦替你自動消牌。如果是死牌時此
  功能無效，但是可按下M鍵或C鍵來移位，就可以繼續使用此密技。
3.全螢幕看牌
  在遊戲中同時按功能鍵F5及F8，便能夠全螢幕看牌。看完後再按空白鍵跳出
  即可。

操作：

先按Enter再按Space = 跳過版權畫面
A = 神機拿牌（減40秒，自動消去2組牌）
B = 同花消牌（減30秒，消去與選的牌相同的所有牌）
C = 移位大法（減70秒，更換所有麻將位置）
M = 乾坤挪移（減50秒，由電腦任意更換部份麻將位置）
P = 查看按鍵
Ctrl-E = 離開遊戲
1 = 音樂開關
2 = 音效開關

備註：CG圖無露點
說明：

1.每消去一組牌增加4秒時間，但使用秘技消牌不增加時間。
2.剩餘的時間會保留做為下一關的可用時間，每過一關均會隨機增加時間。
