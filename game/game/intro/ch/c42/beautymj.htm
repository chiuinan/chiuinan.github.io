﻿名稱：十六張麻將寫真館 (中文版)
簡介：大野狼工作室2001年出品的麻將遊戲，由智冠代理發行，以16張雙人麻
      將玩法計台。遊戲中提供八位美眉供挑戰，獲勝時可觀賞動畫。遊戲還
      有美眉心防與樂透台數的設定，攻下美眉心防或胡到樂透台數，都還可
      觀賞額外動畫。只是這些動畫其實都只是用一張CG圖放大縮小、局部移
      來移去而已，有點硬充檔案大小的感覺，還不如直接放一張清晰的CG圖。
      不過CG圖畫得還蠻漂亮的，讓人賞心悅目。而打牌過程裡可使用魔法牌，
      還可用手指逗弄美眉，都讓遊戲加分不少。遊戲最大的缺點便是使用魔
      法牌自摸時會誤判某些牌型導致台數計算錯誤，但這也沒什麼太大影響，
      因此算是還不錯的麻將遊戲。



保護：光碟，破解碼不明
修改：

1.最愛排行榜全開

beauty.sav（存檔）
  0010h～002Fh全填FFh
  0030h～004Fh全填00h

2.美眉打牌初始資訊

BeautyMj.exe從A8140h開始，每位20 byte，共8位

W1 W1 W1 W1 W2 W2 W2 W2 M1 M1 M1 M1 M2 M2 M2 M2 DF DF DF DF

W1 = 每底錢數（0BB8h=3000）
W2 = 每台錢數（03E8h=1000）
M1 = 美眉初始金錢
M2 = 玩家初始金錢（4E20h=20000）
DF = 美眉初始心防

密技：

打牌過程按滑鼠右鍵再選回選擇頁後，未使用的魔法牌會保留到新開的一局，
因此重複選新局再回選擇頁，累積到較好且足夠的魔法牌後再玩，會比較容易
獲勝。

說明：（作者－青衫）

1.每一雀玩家的籌碼固定20000，底3000，每台1000。
2.每一把牌雙方各扣1000做為樂透獎金，當胡到的台數和樂透台數相同時，即
  可獲得樂透獎金。當己方獲得樂透獎金時，即可觀看美眉的CG動畫圖。
3.每一把牌一開始雙方各得一張魔法牌，該把未使用的魔法牌可累積到下一把，
  但最多只能保留七張，超過時必須自選一張丟棄。
4.每贏美眉一把，美眉的心防便會下降；美眉每贏一把，心防便會上升。升降
  幅度視輸贏的金錢多寡而定。當任一方輸光籌碼，該雀即結束。
5.贏光美眉籌碼後，即可觀看美眉的CG動畫圖。若此時美眉的心防已降為0，
  可再進一步觀看美眉的CG動畫圖，否則只會顯示動畫的最後一張圖。
6.所有已看過的CG動畫圖，都可在最愛排行榜中隨時重複觀看。
7.重新一雀時，美眉的籌碼與心防均重新開始，不管前一雀心防降多少。
8.打牌時可以用手指逗弄美眉，不同部位會有不同的音效與表情。
