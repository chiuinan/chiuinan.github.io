名稱：政治麻將：凱達格蘭夜市 (中文版)
簡介：極真科技2004年出品的16張四人麻將遊戲，採用台灣麻將規則計台，並
      可自行定義。本遊戲為該公司政治麻將系列的第三部，以當時最熱門的
      10位政治人物為主角。遊戲目標為在一雀麻將內，提升自己籌碼以獲得
      勝利，有點過於單調。前兩代語音停頓的缺點依然沒有改善，沒有一代
      的秘招，也沒有二代的團體戰，更絕的是，將前兩代的排行榜直接拿掉，
      讓玩家完全沒有一再挑戰的目標，娛樂性減低很多。對手一樣有偷看之
      嫌，玩家只能透過存取檔來改變劣勢，否則正常地玩，勝率並不高。因
      此整體而言，這款遊戲的娛樂性不算是很好，可玩可不玩。



保護：光碟，破解碼：
      WMJ.exe
        83 F8 05 75 76
        -- -- -- 90 90
      必須拷貝光碟中的Anim和Mus目錄到安裝目錄裡。另外，TRUETECH.CFG
      開頭需改成2E 5C 00，將光碟目錄換成目前目錄。
備註：光碟裡的Sys目錄約400MB的檔案，其實是無用的假資料。
修改：存檔????.SAV

位置0148h = 己方的牌組，每個6 byte
位置0334h = 己方籌碼，4 byte
位置0550h = 下家籌碼，4 byte
位置076Ch = 對家籌碼，4 byte
位置0988h = 上家籌碼，4 byte

牌組資訊：

NN -- CC -- -- --

NN：數字，0～8表示數字1～9，0～6=東南西北中發白
CC：牌色，0=萬、1=條、2=筒、3=字

說明：

1.可選擇的人物代表：
  左：馬英九、陳水扁
  右：蘇貞昌、鄭弘儀
  上：邱毅、趙建銘
  中：施明德、呂秀蓮
  下：汪笨湖、陳文茜
2.初始籌碼1000000，底/台可調整，內定10000/2000。
3.打完一雀或有人籌碼歸零時結束，統計籌碼以決定名次。
