﻿名稱：正宗四川省麻將 (中文版)
簡介：九藝1994年出品的四川省麻將遊戲，玩者必須在限時內拿掉所有可兩折
      （三線段）內連接的相同牌組。遊戲共提供了14個不同的內定關卡，並
      提供有限的輔助過關功能。只是各牌組的相對位置均固定，因此解開的
      程序也是固定，玩久了便會覺得有點無趣。所幸遊戲中允許玩家自行設
      計關卡，同時遊戲過程還可以存取檔，對於玩家而言算是相當方便，因
      此仍算是可玩的一款遊戲。



限制：必須在Win31/XP系統下執行。
保護：圖案密碼，破解碼：

修改：記憶體

-- -- -- -- -- -- TT TT RR RR N1 N1 N2 N2 -- --

TT = 剩餘時間(秒數)
RR = 剩餘對數
N1 = 自尋次數
N2 = 強迫解牌

操作：

滑鼠左鍵 = 選牌
滑鼠右鍵 = 取消選牌
滑鼠左鍵雙擊 = 確定解牌

說明：

1.遊戲一共有14個不同的內定關卡，第14關之後的牌型便不會再變動。
2.各關卡均有不同的剩餘時間、自尋次數、強迫解牌，不會延續到下一關。
3.發生下列任一情況時，遊戲結束：
  (1)剩餘時間用盡。
  (2)已無可消的牌組，同時亦無強迫解牌魔法。
4.每解掉一組牌組，除了得分外，還增加2秒剩餘時間。
