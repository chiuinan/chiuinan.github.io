名稱：麻辣四川省4 (中文版)
簡介：大點科技1999年出品的18禁四川省麻將遊戲，由第三波代理發行。玩家
      必須在限時內拿掉所有可兩折（三線段）內連接的相同牌組，遊戲中也
      提供了一些輔助過關的功能。當整個牌組解開後，便會顯示一張美女圖
      做為獎賞。遊戲一共有30個關卡，每過過關時，剩餘的時間做為下一關
      使用。整個遊戲基本上有一定的娛樂性，只是選關上的操作不是很順暢。



備註：本版必須將安裝目錄設定成X:硬碟才能正常執行。
保護：光碟（檢查光碟上的4R4\MILESLEE檔案）。破解碼：
      4r4.exe
        8A 01 88 42
        B0 XX -- --  (XX=下列模擬的磁碟編號，58h=X，5Ah=Z)
      再使用下列方式將安裝目錄模擬成硬碟：
        subst x: .
      結束後再以下列方式取消：
        subst x: /d
備註：正常版請執行PLAY.BAT，修改版請執行UPLAY.BAT

資訊：CG圖檔Level\Girl????

1.先解密：各byte左滾（rol）1位元
2.將檔名更為bmp，即可視為圖檔載入

操作：

單擊滑鼠左鍵 = 選牌
雙擊滑鼠左鍵 = 消牌
滑鼠右鍵 = 取消選牌
F1 = 暫停／音樂音效控制
Esc = 離開遊戲
F5 = 自動消掉一組牌（消耗2魔法點數）
F6 = 自動移動部份牌位置（消耗3魔法點數）
F7 = 重新組牌（消耗4魔法點數）

備註：CG圖無露點。
說明：

1.左上為剩餘時間，初值為8:00（480s=1E0h），每消去一組加10秒。
2.下方為魔法點數，初值為10，當消去火紅牌時加1，最多30。
3.過關時，剩餘的時間繼續做為下一關的限時。
4.每一關的難易度不同，且可重複挑戰，因此若剩餘時間不足，可以回頭挑戰
  簡易的關卡，以補充剩餘時間或魔法點數。
