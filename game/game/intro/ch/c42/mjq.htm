﻿名稱：麻將情趣屋 (中文版)
簡介：天堂鳥1994年出品的１８禁１３張麻將遊戲。遊戲的主要目標便是挑戰
      並打敗１０位美女，順利承接麻將屋。而每當打敗一位美女，即會顯示
      一張美女圖，並可隨機得到一些情趣用品使用。遊戲還提供特殊牌形可
      供購買，只是進張與胡牌率稍低，且愈到後面對手愈快胡牌，讓人明顯
      地感到電腦在作弊，有點不爽。

操作：

1.打牌時
  按上鍵叫出選單
  Esc = 離開遊戲
2.看圖時
  1 = 放大鏡
  2 = 搔癢器
  3 = 火柴棒
  4 = 激情藥水
  5 = 搔癢手套
  6 = 水槍
  7 = 透視眼鏡
  8 = 按摩棒
  Esc = 結束看圖

備註：CG圖有露點。
說明：

每打贏一位美女，可以抽三種道具，看圖時使用不同的道具，可以有音效或衣
物或背景的改變。

1.放大鏡：將圖片局部放大。
2.搔癢器：發出笑聲。
3.火柴棒：螢幕慢慢的閃三次。
4.激情藥水：表情變得比較曖昧。
5.搔癢手套：表情變得比較舒服。
6.水槍：讓衣物變得透明。
7.透視眼鏡：全身衣物會消失。
8.按摩棒：激情過後的樣子。
