簡介：大宇2003年出品的麻將遊戲，以台灣常見的16張麻將玩法計台。遊戲提供了自由對戰、
      競賽制的群龍爭霸、以及劇情模式的雀壇風雲等三種玩法，基本上和二代相差不多。
      遊戲計有15位人物角色可供選擇，每個角色都各有其對應的語音，打牌時的仿真程度
      頗高。而在劇情模式裡，主角可學習密技及購買道具，所不同的是，本代的密技必須
      使用一定次數後，才會領悟新的密技，是個很不錯的設計。此外，遊戲另行提供五個
      簡單的賭博遊戲，也能自己設定計台規則，娛樂性相當不錯。唯一缺點是角色在播放
      語音時會中斷牌局的進行，打牌的速度也稍嫌不夠快，不過按個滑鼠就可以略過語音，

*** 正宗賭坊

1.百萬富翁
  規則：更換手中的牌到胡牌牌型，計算總萬數，多者獲勝
  賠率：雙方均胡牌時賠賭金x總萬數差距x(1+剩餘機會數)，雙方皆未胡牌則為平手。只有
        一方胡牌時賠賭金x總萬數差距（總萬數差距少於10萬則當作是10計算，包括胡牌方
        的萬數較少時）
2.黑傑克
  規則：最常見的21點Black Jack賭博遊戲，以最接近且不超過21點者勝。
  賠率：與莊家對賭，勝的話贏回一倍賭金。若一開始兩張牌剛好為21點則贏回1.5倍賭金。
        若拿了五張牌還沒有超過21點過五關則贏回5倍賭金。
3.十八豆
  規則：豹子（三顆相同）>一般（兩顆相同，比另一顆點數）>彆十（123）
  賠率：與莊家對賭，勝的話贏回一倍賭金。
  提示：出現施力圖時，要讓施力點接近上下兩個箭頭（按滑鼠左鍵決定施力點），很接近
        時會出現豹子，太遠的話容易彆十。
4.輪盤
  規則：珠子進入輪盤的數字是自己有押賭金時獲勝
  賠率：單雙、紅黑、大小(押注18個號碼) = 2倍
        第幾排(押注12個號碼) = 3倍
        牌桌上緣的T字交線上(押注6個號碼) = 6倍
        四個號碼間十字交線上(押注4個號碼) = 9倍
        牌桌上緣橫線上(押注3個號碼) = 12倍
        相鄰兩個號碼間線上(押注2個號碼) = 18倍
        數字 = 36倍
        同一賭金押多個號碼時，按比例決定押注賭金
5.雙龍搶珠
  規則：擲骰子拿走對應點數的麻將牌組合，直到無法拿為止，計算剩餘牌點數，少者獲勝。
  賠率：賭金x點數差距，若剛好拿完則賠率再乘3倍
  提示：從大牌先拿，儘量留小牌。
