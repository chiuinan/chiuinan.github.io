名稱：瘋狂政治麻將 (中文版)
簡介：極真科技2001年出品的16張四人麻將遊戲，採用台灣麻將規則計台，並
      可自行定義。本遊戲為該公司政治麻將系列的第一部，以當時最熱門的
      12位政治人物為主角。遊戲目標為在一雀麻將內，提升自己籌碼以打入
      排行榜，和自由對戰沒有兩樣，有點過於單調。而且角色發出語音時無
      法略過，使得遊戲進行常因語音而停頓，停止語音又有點單調，很讓人
      困擾。不過遊戲裡有提供一些秘招輔助，對手不作弊、可隨時存取檔變
      更打法、聽牌有瞇牌動作等優點，因此還是有點娛樂性，可以玩玩看。



保護：光碟，破解碼：
      WinMJ.exe
        83 F8 05 74 1C
        -- -- -- EB --
      必須拷貝光碟中的Anim和Mus目錄到安裝目錄裡。另外，TRUETECH.CFG
      開頭需改成2E 5C 00，將光碟目錄換成目前目錄。
備註：正常版請執行PLAY.BAT，修改版請執行UPLAY.BAT

修改：存檔????.SAV

位置0130h = 法力值，1 byte
位置014Eh = 己方的牌組，每個6 byte
位置033Ah = 己方籌碼，4 byte
位置054Ah = 下家籌碼，4 byte
位置075Ah = 對家籌碼，4 byte
位置096Ah = 上家籌碼，4 byte

牌組資訊：

NN -- CC -- -- --

NN：數字，0～8表示數字1～9，0～6=東南西北中發白
CC：牌色，0=萬、1=條、2=筒、3=字

說明：

1.可選擇的人物代表：
  第一排：呂秀蓮、陳水扁、張俊雄
  第二排：張昭雄、宋楚瑜、連戰、馬英九
  第三排：謝長廷、黃主文、李登輝、李慶安、謝啟大
2.初始籌碼1000000，底/台可調整，內定10000/2000。
3.打牌過程會隨機增加秘招幣（法力），可按下左下錦囊施展秘招。
4.打完一雀或有人籌碼歸零時結束，統計籌碼決定名次，及是否進入排行榜。
