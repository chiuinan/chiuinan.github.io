第二章 山中廢墟
任務： 為了征服大陸，需要大量的軍費，而傳說山中廢墟埋藏著大批寶藏。但是通往廢墟的入口已經被封住了，據說只有精靈長老及矮人王才能打開入口。
金錢： 5000金
角色： （電腦控制的人物與傭兵，其等級均增加2級）
希洛：10級，戰士8級、戰士8級、僧侶8級、法師8級
明 ：10級，獸人8級、獸人8級、獸人8級、獸人8級
凱麗：10級，惡靈8級、鬼魂8級、僵屍8級、僵屍8級
雪莉：10級，鬥士8級、鬥士8級、僧侶8級、僧侶8級
領地： 村落守軍數為1+(0～3)位，等級為8+(0～4)
鄉鎮守軍數為2+(0～3)位，等級為10+(0～4)
城市守軍數為3+(0～2)位，等級為12+(0～6)
城堡守軍數為5位，等級為16+(0～4)
地圖： （點擊可看原圖，或文字版地圖）

過關：

  1. 先到矮人領地或精靈領地接受委託（各角色有各自的委託）。接受委託後，便不能再攻打委託方的領地，只能攻打另一方的領地。
  2. 另一方的領地全部被佔領後（對手佔領的也算，只要不是中立領地即可），再回到委託方的任一城池（包括兩個S記號的地點），即完成委託。幫矮人者可得魔法項鍊，幫精靈者可得精靈靴。完成委託後會打開通往山中廢墟的通道。
  3. 進入山中廢墟的通道時，會遇到一個30級石像巨魔擋道，打敗它。
  4. 來到山中廢墟打敗守軍，即可過關，並得到10000元。

說明：

  1. 本章傭兵的攻防能力，以巨蛇最強，為唯一能一般攻擊全體敵人的傭兵，但它有兩個致命缺點：一是準確度很差，常會打不中敵人，而全體攻擊法術的殺傷力又很低；二則是佔了兩個位置，平均下來反而不如略遜於它的牛頭人。牛頭人除了很強的攻防能力外，也有能攻擊全體的法術，以及強力攻擊單一敵人的法術，常用來打第一章的五隻高等級吸血鬼，因此比較推薦雇用牛頭人。但牛頭人只能用傭兵卡召募到，在沒有傭兵卡前只能照第一章的方法雇用傭兵。
  2. 進獸人部落雇用地精（盜賊也可以，但較貴），在戰鬥中偷取：
    　　戰士、盜賊、地精、獸人：神聖護符
    　　祭司：疾走靴
    偷完後就可以解雇了（攻防能力很差）。如果在第一章已拿到這些裝備，便不用再偷了。
  3. 進入山洞遊樂場，進行對對碰：
    　　希洛：獲取斬鐵劍、龍盾
    　　明：獲取魔法斧、龍盾
    　　凱麗：獲取魔法袍
    如果在第一章已拿到這些裝備，便不用再玩了。
  4. 進入裝備店購買：
    　　希洛、明：戰鬥護甲、魔法盔
    　　凱麗：天使光環
    　　雪莉：戰鬥護甲、天使光環
  5. 最好35級以上時才完成委託任務，前去攻打山中廢墟（雪莉最好到36級），以便下一章時可以簡單攻克對手主城。
  6. 當完成委託後打開通往山中廢墟的通道，此時所有角色均可進入通道，因此在完成委託後要迅速往北過關。注意山中廢墟的通道無法用轉移卡到達，因此務必要備用一張任意步卡。
