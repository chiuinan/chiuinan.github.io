0000 YY YY YY YY MM MM MM MM WW WW WW WW DD DD DD DD
0010 MA MA MA MA ID ID ID ID -- -- -- -- -- -- -- --

YY,MM,DD = 年/月/日
WW = 星期（0～6）
MA = 第幾關，1～4（不可亂改）
ID = 目前行動的人物，0～3

(+04)NN = 姓命
(+13)EX = 經驗值
(+17)LV = 等級
(+1B)H1/H2 = 目前生命/最大生命 = H3+H0*(等級-1)（升級會重算）
     (+23)H0 (+7B)H3
(+27)M1/M2 = 目前法力/最大法力 = M3+M0*(等級-1)（升級會重算）
     (+2F)M0 (+7F)M3
(+33)A1=AP1 (+37)A0=AP0 (+3B)A2=HT1 (+3F)A3=HT0
BASE_AP = A1+A0*等級-1
BASE_HT = A2+A0*等級-1
(+4F)AP = 攻擊力 = BASE_AP+裝備攻擊力（升級會重算）
(+53)HT = 準確度 = BASE_HT+武器準確度（升級會重算）
(+57)DP = 防禦力 = BASE_AP/2+裝備防禦力（升級會重算）
(+5B)EV = 閃避率 = BASE_HT/2+裝備閃避率（升級會重算）
(+5F)AT = 一般攻擊次數，1=1次，2=2次，5=全體攻擊
          （但希洛的資料在進入戰鬥時會修正，修改無用，可進行程式修正）
(+63)E1～E6 = 身上裝備編號，依次為武器、衣服、盾牌、頭盔、項鍊、鞋子
(+83)S1/S2 = 主角法術編號/可施展的最低等級

(+C3)FL = 已完成任務編號，例如第1章1=接受請託、2=拿到神之草、3=拿到真實之珠
(+C7)CT = 是否電腦控制，1表是，0表否，FFFFFFFF表陣亡（必須清除所屬領地）
(+CB)MC = 主城編號
(+D3)XX,YY = 位置，陣亡時需清為FFFFFFFF
(+DF)P1 = 無法行動的剩餘回合
(+E0)P2 = 無法使用卡片的剩餘回合
(+E1)P3 = 英雄卡是否作用，0=否
(+E2)P4 = 防禦卡是否作用，0=否
(+E3)P5 = 祝福卡是否作用，0=否
(+E4)P6 = 平安卡是否作用，0=否
(+FF)T1 = 傭兵種類編號，每4 BYTE一組，共4組
T2 = 傭兵等級，每4 BYTE一組，共4組
T3 = 傭兵目前生命，每4 BYTE一組，共4組
T4 = 傭兵最大生命，每4 BYTE一組，共4組
T5 = 傭兵目前法力，每4 BYTE一組，共4組
T6 = 傭兵最大法力，每4 BYTE一組，共4組
T7 = 傭兵經驗值，每4 BYTE一組，共4組
T8 = 傭兵已工作天數，每4 BYTE一組，共4組

註：因程式誤寫，導致BASE_AP和BASE_HT的值計算錯誤，實際應為：

BASE_AP = A1+A0*(等級-1)
BASE_HT = A2+A3*(等級-1)
