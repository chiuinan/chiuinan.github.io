(+04)NN = 姓名
(+0E)MI = 角色順序編號，由1開始
          （凱麗與雪莉的原始資料錯誤，會造成原本應該雪莉可以裝備的皮甲等護甲，變成無法裝備，而是凱麗可以裝備，可進行程式修正）
(+13)EX = 經驗值
(+17)LV = 等級
(+1B)HP/HX = 目前生命/最大生命 = H1+H0*(LV-1)（升級會重算）
     (+23)H0 (+7B)H1
(+27)MP/MX = 目前法力/最大法力 = M1+M0*(LV-1)（升級會重算）
     (+2F)M0 (+7F)M1
(+33)S1=STR1 (+37)S0=STR0 (+3B)D1=DEX1 (+3F)D0=DEX0
STR = S1+S0*LV-1
DEX = D1+S0*LV-1
(+43)I1=INT1 (+47)I0=INT0 (+4B)LK=LUK
(+4F)AP = 攻擊力 = STR+裝備攻擊力（升級會重算）
(+53)HT = 準確度 = DEX+武器準確度（升級會重算）
(+57)DP = 防禦力 = STR/2+裝備防禦力（升級會重算）
(+5B)EV = 閃避力 = DEX/2+裝備閃避力（升級會重算）
(+5F)AT = 一般攻擊次數，1=1次，2=2次，5=全體攻擊
          （但希洛的資料在進入戰鬥時會修正，修改無用，可進行程式修正）
(+63)E1～E6 = 身上裝備編號，依次為武器、衣服、盾牌、頭盔、項鍊、鞋子
(+83)KD/KL = 主角法術編號/可施展的最低等級

(+AB)CD/CN = 卡片編號/數目
(+BF)MN = 金錢
(+C3)FL = 已完成任務編號，例如第1章1=接受請託、2=拿到神之草、3=拿到真實之珠
(+C7)CT = 是否電腦控制，1表是，0表否，FFFFFFFF表陣亡（必須清除所屬領地）
(+CB)MC = 主城編號
(+D3)XX,YY = 位置，陣亡時需清為FFFFFFFF
(+DB)DI = 方向，0=上、1=右、2=下、3=左
(+DF)P1 = 無法行動的剩餘回合
(+E0)P2 = 無法使用卡片的剩餘回合

(+E2)P4 = 防禦卡是否作用，0=否
(+E3)P5 = 祝福卡是否作用，0=否
(+E4)P6 = 平安卡是否作用，0=否
(+FF)T1 = 傭兵種類編號，每4 BYTE一組，共4組

T3 = 傭兵目前生命，每4 BYTE一組，共4組
T4 = 傭兵最大生命，每4 BYTE一組，共4組
T5 = 傭兵目前法力，每4 BYTE一組，共4組
T6 = 傭兵最大法力，每4 BYTE一組，共4組
T7 = 傭兵經驗值，每4 BYTE一組，共4組
T8 = 傭兵已工作天數，每4 BYTE一組，共4組

註：因程式誤寫，導致STR和DEX的值計算錯誤，實際應為：

STR = S1+S0*(LV-1)
DEX = D1+D0*(LV-1)
