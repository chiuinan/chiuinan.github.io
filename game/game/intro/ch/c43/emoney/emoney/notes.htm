地圖提示：

  1. 每一回合為一天，每週日會進行稅收及薪水結算，每天開始時可能發生隨機的突發事件，接著所有角色輪流行動一次。
  2. 角色在大地圖上有移動，才會結束該回合的行動，否則可以不斷使用卡片或道具。大地圖各位置的作用，請參見地點列表。
  3. 每天結束時，所有人員（含傭兵與領地守軍）均會恢復20%生命與法力。
    • 領地守軍的生命、法力值決定進入攻城戰時的初始值，一般不會改變（攻城戰中領地守軍的受傷、死亡、施法消耗，在戰後都不計），只會受事件或卡片影響而減少。
    • 因程式誤寫，導致實際上傭兵每天只會回復2點法力，可進行程式修正。
    • 後手玩家可利用此特性陷害先手玩家，例如明、凱麗、雪莉可趁希洛剛結束艱困戰鬥未恢復之際，使用大量機運卡攻擊，有相當高的機率引發遇到怪物事件，令其連續戰鬥而戰敗。
  4. 物品賣出的價格為原價70%。
  5. 遊戲在每位主角（包括玩家或電腦操控的）行動前都會自動存檔至GAME088.SAV。萬一忘記存檔或遊戲無預警當掉，將此檔案的數字部分改成001～004就能讀檔接關。

戰鬥提示：

  1. 戰鬥時，由我方按隊員次序先攻一輪，接著才會由敵方進攻。
  2. 戰鬥時，若主角陣亡便立即視為失敗，此時會損失所有卡片、道具裝備（身上的除外）、以及一半的金錢，並回到主城繼續。
  3. 打敗敵人後，經驗值分配的公式為：
    　每人配得經驗值 = 怪物總經驗值 / 隊伍人數
    　隊員實際獲得經驗值 = 每人配得經驗值 / 隊員等級，向下取至2的正整數倍，最低2
  4. 愈高等級的法術，效果未必比較好，例如【神恩】的治療效果便比【回復】差。請參考法術列表中的傷害計算公式，以及種族相性表。
  5. 萬一遭遇打不贏的強大怪物，可善用非戰鬥手段防止敗北：如果怪物是可召募的類型，可透過戰鬥交談要求加入。不缺錢時可贈送禮物，一般贈送5次金錢（好感度期望值100）就有相當高的成功率，贈送10次金錢就必定成功。金錢不足時，如果怪物厭惡主角的機率不高，可嘗試好言相勸，只要沒有立即進入戰鬥，接著重覆好言相勸至10次即可要求加入。如果隊伍空間不足，可先交戰幾回合，待傭兵陣亡釋出空間再交談（主角可防禦以免先陣亡）。如果不是可召募的類型，基本上只能賭運氣拚逃跑（可重覆多次直至成功或陣亡）。此外也可以常備魅惑卡及風神卡，二者皆能100%結束戰鬥。

綜合提示：

  1. 人物若失去所有領地，便算失敗，並從遊戲角色中剔除。所有玩家控制的角色都失敗，則遊戲結束。
  2. 遊戲一開始的首要目標就是提升等級，因為可以召募的傭兵等級、能用的法術、以及財寶事件的金錢數都和等級有關。可盡量獨自作戰，以獲取最多的經驗值，但獨自作戰也會增加戰敗風險，若沒把握，可找1～2個傭兵輔助，但不要太多，否則得到的經驗值會較少。如果是單人遊戲，可善加利用存取檔大法，被打敗便讀檔重來。
  3. 可善加利用步數卡片，儘量遇敵或攻打領地練功。攻打領地前可多用幾次瘟疫卡或淨化卡，把守軍血量削減到能用一發全體攻擊法術擊倒，就能輕易攻下高很多級的領地，並賺取大量經驗值。
  4. 劇情模式下，當等級提升一定程度之後，務必先消滅對手的全部領地，剩下自己一人過關比較節省時間。儘量善用轉移卡，方便直接前往某地而不必擲骰子。建議是身上備著一張外賣卡，等轉移卡用完時，便利用存取檔大法使用外賣卡，當同時出現外賣卡和轉移卡時才購買，如此就會有用不完的轉移卡。
  5. 第一章由於沒有好的裝備，最好雇用5級以上的地精或盜賊在戰鬥中施展盜取物品：
    　戰士、盜賊、地精、獸人：可偷得神聖護符
    　僧侶、法師、精靈：可偷得精靈靴
    第二章或一般模式時：
    　祭司：可偷得疾走靴
  6. 大地圖上的傭兵召募地點，只能召募到一般能力較差的傭兵，想要能力較強的傭兵，必須使用傭兵卡才能召募到，或在戰鬥中透過交談要求加入或用魅惑卡網羅，特別是魅惑卡可無視陣營限制，取得主角無法召募的傭兵。一般而言，隊伍帶4個牛頭人可讓戰力最大化。然而基於其他考量，有時選擇幾種大型怪物會更好：
    • 練功：使用大型怪物時總人數少，分配到的經驗值較多，升級快。
    • 強化：主角若有神力、倍力等強化性法術，施放在大型怪物上較有效率。
    • 折損：牛頭人畢竟沒有幾種大型怪物耐打，且法術抗性低，挑戰強敵仍較易陣亡而難長期陪戰。
    • 取得：牛頭人只能透過傭兵卡召募，想召滿4個牛頭人往往得反覆抽很多次，混搭幾種大型怪物則容易許多。
    大型怪物的選擇可參考以下幾點：
    • 龍：血量、攻防、抗性都高，非常耐打，往往比主角更難陣亡。只要活著，就能有效分散敵人對主角的攻擊，讓主角有餘裕可以治療或使用物品。普攻及全體攻擊法術的傷害也相當高，遇到怕火的敵人還可以用倍力／神力噴火掃蕩。許多時候在隊伍放一隻以上的龍都是好主意。
    • 巨蛇：血量、攻防僅次於龍，且能攻擊全體，但命中率太低，只能打中閃避力差或等級低的敵人，無法做到戰略集火，一般並不實用。然而如果主角能施放神力，則能有效彌補低命中率的缺陷，輸出高達LV*16的全體物理傷害。此外，打魔王或邪神時，可用毒氣攻擊使之每回合大量扣血，即能輕鬆戰勝。
    • 獨眼巨人：體質僅稍遜於巨蛇，單體攻擊技投擲巨石有LV*8的高傷害，且只須4點法力，不易耗盡。缺點是法術抗性差，受到法術攻擊可能來不及治療，此外普攻命中率也較差。
    • 龍人：血防偏低，但有龍族的抗性，整體而言比牛頭人耐打一點，也較易治療，且沒有怕火的弱點。對強敵時，狂龍擊可造成全遊戲最高的LV*9傷害；對弱敵時，二次攻擊能造成LV*12左右的高傷害。缺點是沒有全體攻擊法術。
  7. 隨機遇敵的怪物最高為50級，因此角色超過50級以後會愈來愈難升級。雖然可先在自己的領地召募最高等級、經驗值的守軍，用交換卡換成非我方領地再獨自攻打，每次可得120點以上的經驗值；然而能到達高等級，對手大概都被全滅了，若沒有修正bug，根本無法使用交換卡，即便修正了，能交換的中立領地有限，買必要的卡片花時間，打高等級敵人也較艱難，因此不太推薦此一練功法。不如獨自一人到處踩點遇低等敵輕鬆打，每場還是能有10～60點經驗值進帳。

*** 傷害計算公式

  1. 一般攻擊 (物理攻擊)

    若有倍力、必中、鐵壁、虛幻狀態者，該人員的攻擊力、準確度、防禦力、閃避力加倍
    命中率 = (攻方準確度HIT - 守方閃避力EV)%，最低0%，最高100%
    傷害 = 攻方攻擊力AP - 守方防禦力DP
    實際傷害 = 傷害*相性[守方種族][物理] * 100/(80～119)

  2. 法術攻擊 (必定命中、無視防禦)

    傷害 = 法術傷害計算公式
    施法人員若有倍力狀態時，傷害加倍
    實際傷害 = 傷害*相性[守方種族][法術形態] * 100/(80～119)
    天雷轟不隨機增減傷害

  3. 醫療

    恢復 = 法術恢復計算公式
    實際恢復 = 恢復 * 100/(80～119)

註：

  1. 因遊戲程式誤寫，命中率多了1%，因此即便攻方準確度HIT遠低於守方閃避力EV，依然有1%的命中率。可進行程式修正。
  2. 遊戲原本是要寫成增減20%的隨機傷害或醫療，但除數與被除數寫錯，變成-16%～+25%，可進行程式修正。
  3. 實際傷害，至少1點，即便是免疫或回復的敵人仍會受到1點傷害。

*** 其他

電腦操控的主角，因程式誤寫，會導致一些不合理的行為，可進行程式修正：

  1. 使用物品：戰鬥時，原本設計電腦操控的主角有10%機率使用物品，但因程式誤寫，導致之後會重新決定行動，本來選出要使用的物品則憑空消失。
  2. 逃跑判斷：戰鬥時，電腦操控的主角會以敵我雙方存活隊員的基礎經驗值（四主角使用常數：45,70,25,18）*LV的總和來判斷雙方戰力，我方低於敵方則逃跑（必定100%成功）。但傭兵在進入戰場時，原先載入的基礎經驗值已被重新覆蓋為目前經驗值，原版程式並未注意到此點，導致在戰力評估時，會因為傭兵目前經驗值的高低而高估或低估我方戰力。
  3. 召募守軍：因相關程式判定錯誤，導致電腦操控的主角在自己的領地上會召募出不符合領地守軍型態的守軍種類，且數量可能超出隊伍空間上限。
