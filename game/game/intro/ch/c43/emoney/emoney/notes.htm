地圖提示：

  1. 所有角色輪過一次為一回合，每回合結束時，所有人員（含傭兵與領地守軍）均會恢復20%生命與法力。
  2. 每一回合為1天，每週均會有固定發生的事件。
  3. 物在大地圖上有移動，才會結束該回合的行動，否則可以不斷使用卡片或道具。大地圖各位置的作用，請參見地點列表。
  4. 物品賣出的價格為原價70%。

戰鬥提示：

  1. 戰鬥時，由我方按隊員次序先攻一輪，接著才會由敵方進攻。
  2. 戰鬥時，若主角陣亡便立即視為失敗，此時會損失所有卡片、道具裝備（身上的除外）、以及一半的金錢，並回到主城繼續。
  3. 打敗敵人後，經驗值分配的公式為：
    　每人配得經驗值 = 怪物總經驗值 / 隊伍人數
    　隊員實際獲得經驗值 = 每人配得經驗值 / 隊員等級
    隊員實際獲得經驗值最低為2。
  4. 愈高等級的法術，效果未必比較好，例如［神恩］的治療效果便比［回復］差。請參考法術列表中的傷害計算公式，以及種族相性表。

綜合提示：

  1. 人物若失去所有領地，便算失敗，並從遊戲角色中剔除。所有玩家控制的角色都失敗，則遊戲結束。
  2. 遊戲一開始的首要目標就是提升等級，因為可以召募的傭兵等級、能用的法術、以及財寶事件的金錢數都和等級有關。而且要獨自作戰，以獲取最多的經驗值。能力較差的凱麗和雪莉，可能還是需要找一個或兩個傭兵做輔助，但不要太多，否則得到的經驗值會較少。最好善加利用存取檔大法及步數卡片，儘量去遇到敵人以快速提升等級（若被打敗便讀檔重來），至少要高過別人10級以上。
  3. 劇情模式下，當等級提升一定程度之後，務必先消滅對手的全部領地，剩下自己一人過關比較節省時間。儘量善用轉移卡，方便直接前往某地而不必擲骰子。建議是身上備著一張外賣卡，等轉移卡用完時，便利用存取檔大法使用外賣卡，當同時出現外賣卡和轉移卡時才購買，如此就會有用不完的轉移卡。
  4. 第一章由於沒有好的裝備，最好雇用5級以上的地精或盜賊在戰鬥中施展盜取物品：
    　戰士、盜賊、地精、獸人：可偷得神聖護符
    　僧侶、法師：可偷得精靈靴
    第二章或一般模式時：
    　祭司：可偷得疾走靴
  5. 大地圖上的傭兵召募地點，只能召募到一般能力較差的傭兵，想要能力較強的傭兵，必須使用傭兵卡才能召募到。
  6. 隨機遇敵的怪物最高為50級，因此角色超過50級以後會愈來愈難升級。雖然可先在自己的領地召募最高等級的傭兵，然後用交換卡交換成非我方領地後再獨自去攻打，以提高獲得的經驗值。但花了許多時間艱難的打一場，也只能獲得40～50點經驗值而已，還不如帶滿高等傭兵去踩點遇低等敵輕鬆打，50場還是能升1級。況且能到達高等級，對手大概都被全滅了，若沒有修正bug，根本無法使用交換卡。即便修正了，能交換的中立領地也有限，想買到必要的卡片，本身也很花時間，因此並不太推薦此一練功法。

*** 傷害計算公式

  1. 一般攻擊 (物理攻擊)

    若有倍力、必中、鐵壁、虛幻狀態者，該人員的攻擊力、準確度、防禦力、閃避率加倍
    命中率 = (攻方準確度HIT - 守方閃避率EV + 1) %
    傷害 = 攻方攻擊力AP - 守方防禦力DP
    實際傷害 = 傷害*    相性[守方種族][物理] * 100/(80～119)

    註1：因遊戲程式誤寫，命中率多了1%，可進行程式修正。
    註2：遊戲原本是要寫成增減20%的隨機傷害或醫療，但除數與被除數寫錯，變成-16%～+25%，可進行程式修正。

  2. 法術攻擊 (必定命中、無視防禦)

    傷害 = 法術傷害計算公式
    施法人員若有倍力狀態時，傷害加倍
    實際傷害 = 傷害*相性[守方種族][法術形態] * 100/(80～119)

  3. 醫療

    恢復 = 法術恢復計算公式
    實際恢復 = 恢復 * 100/(80～119)

註：實際傷害，至少1點，即便是免疫或回復的敵人仍會受到1點傷害，可進行程式修正。
