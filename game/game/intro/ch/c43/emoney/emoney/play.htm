選單說明
選單說明：主畫面選單、大地圖選單、戰鬥選單

一、主畫面選單

  1. 劇情模式：由單一玩家與三位電腦控制對手，從第一章挑戰至第四章，每章有各自會出現的怪物與物品。各角色優劣，請參見主角列表。
  2. 一般模式：角色可任選由玩家或電腦控制，並可隨意挑選任一章進行挑戰（第四章無法打最終魔王邪神）。所有怪物都會出現，但物品仍與劇情模式的章節相同。
  3. 載入進度：讀取先前儲存的進度。
  4. 離開遊戲：結束遊戲，回到DOS。

無論那種模式，過關後，若立即再開新遊戲，則前一場由玩家操控且未陣亡的角色，其所有資訊，包括屬性、金錢、裝備、卡片、傭兵等，都會繼承下來繼續使用。

二、大地圖選單

  1. 移動：擲一顆骰子決定前進步數（0～6）。
  2. 使用：使用卡片或道具（無法使用者不會出現）。
  3. 地圖：顯示縮小的地圖，查看各城鎮的現況。

    地圖上倒三角形為各角色目前所在位置，不同顏色的領地為各角色的領地，白色為中立領地。領地上的數字代表其規模（1=村落、2=鄉鎮、3=城市、4=城堡），會影響稅金收入以及初始守軍的強度。右邊為選擇領地的基本資訊，包括規模、基礎稅金、守軍。

  4. 裝備：查看擁有的裝備（不含卡片），或更換身上的裝備。人物最多可擁有16個裝備。

    移動到可更換的裝備時，右下角會顯示其更換後的數值效果，此時按下Enter即進行換裝，換裝後會按照人物等級重算AP、DP、HIT、EV等資訊。在此畫面按下Ins鍵，可切換成主角的詳細資料畫面（再按一次切換回來）：

    左邊為可用的法術，與施展時所需消耗的法力。不過因遊戲沒有將主角法術編號+50（轉成實際法術編號），以致顯示到錯誤的數值，同時也會遺漏部份法術，可進行程式修改。右邊各數值意義如下：

    LV：等級。會影響主角的其他屬性值、能使用的法術、法術的殺傷力、隨機遇敵的敵人等級、招募到的傭兵等級、財寶事件的金錢數量等。
    EX：經驗值，與敵人戰鬥後獲得，每100經驗值升1級。
    HP：生命，降至0即陣亡。
    MP：法力，用來施展法術。
    AP ：攻擊力，影響一般攻擊的傷害力，由等級與裝備決定。
    DP ：防禦力，影響受到一般攻擊時的傷害力，由等級與裝備決定。
    HIT：準確度，影響一般攻擊的命中率，由等級與裝備決定。
    EV ：閃避力，影響受到一般攻擊時的命中率，由等級與裝備決定。
    G：金錢

  5. 查看－整體勢力：查看各角色的基本屬性值。
  6. 查看－個人勢力：查看各角色擁有的卡片、領地與守軍（使用PgUp、PgDn更換角色）。

    左邊為該角色擁有的卡片，最多16張。中間為該角色擁有的領地與稅金收入，右邊則為目前選擇領地的守軍或角色的近身隨從。上邊為目前金錢數與總稅金收入。

  7. 查看－地圖：可以一步一步的查看大地圖（但無法查看該位置的相關資訊）。
  8. 機能選項：
    • 讀取進度
    • 儲存進度
    • 音樂：音樂開關。
    • 音效：音效開關。
    • 動畫：動畫開關，關閉時可加速電腦控制對手的戰鬥、遊戲等動作。
    • 結束遊戲：回到主畫面。

三、戰鬥選單

攻擊 法術 物品 防禦 交談 逃跑

  1. 攻擊：即物理性的一般攻擊，受攻防能力影響，請參見傷害公式。
  2. 法術：施展法術，請參見角色列表與法術列表，青色底的為主角色可施展的法術。
  3. 物品：使用卡片或道具，只有主角色可以使用。

    註1：因程式誤寫，戰鬥時主角可使用的法術會受限於主角進入戰場時的法力值，此值耗至不足就無法施放相關法術，戰鬥中使用卡片或道具恢復法力無法增加實質施法次數（只會增加戰鬥後的法力值）。可進行程式修改。
    註2：使用道具選擇對象時，按Esc並無法正常取消，必須回到主角重新選擇，否則可能導致當機。可進行程式修改。

  4. 防禦：防禦暫時增加50%一回合，但實際上只有第一位敵人有效。

    註：程式原本應該在戰鬥回合結束後重置防禦力數值（以消去暫時增加的50%防禦力），但誤放在每次敵人攻擊之後，於是第一位敵人攻擊之後，防禦力就被重置了，後面敵人的攻擊，防禦暫時增加的50%防禦力便沒有了。而且重置防禦力數值時未考慮主角與傭兵不同的資料格式，造成主角的防禦力被重置為極低，且無裝備加成，於是玩家會覺得被愈打愈痛，實則只有被第一位敵人攻擊時有防禦力，之後就是無防禦力挨打。可進行程式修改。

  5. 交談：每場戰鬥限交談一次，領地守軍無法交談

    • 好言相勸：好感度增加10～20，但對主角厭惡的敵人會直接進入戰鬥，先行攻擊。
    • 武力威脅：若敵人等級不低於主角等級，直接進入戰鬥，由對方先行攻擊，否則：
      　50%機率：好感度增加10，不再厭惡主角
      　30%機率：直接進入戰鬥，由對方先行攻擊
      　20%機率：敵人逃跑並結束戰鬥
    • 贈送禮物：對方可能要求禮物道具或金錢，然後直接給與。若有給與，則：
      1. 禮物道具：好感度增加10～30
      2. 金錢（基礎價格*等級*2）：好感度增加35061760
      原本遊戲設計送禮增加的好感度是肉塊、寶石、魔水晶分別增加5、20、40，贈送金錢增加10～30，但因二者程式碼寫反，加上贈送金錢的好感度未初始化，而導致上述的異常結果。可進行程式修改。
    • 要求加入：
      1. 必須是可加入的怪物類型（人類不會加入凱麗，魔族不會加入希洛、雪莉），否則立即拒絕。
      2. 隊伍若沒有空位，也會立即拒絕。
      3. 當好感度超過願意加入的好感度，便會加入隊伍，戰鬥結束。
      4. 當好感度低於願意加入的好感度，有50%機率會明確拒絕並回到戰鬥選單，無法再交談。否則會默不作聲並回到交談選單（若有進行程式修改，則對方會出現要求展現誠意的對話）。
    • 打探消息：有80%機率提供一個隨機消息，然後直接進入戰鬥，由對方先行攻擊。
    • 取消：直接進入戰鬥，由對方先行攻擊。

    各類敵人厭惡主角的機率為：

    人類類人獸族魔族
    希洛70505030
    明50707050
    凱麗80505020
    雪莉30505080

    當進入戰鬥時，會先行決定是否厭惡主角，以及願意加入的好感度（0～99）。

  6. 逃跑：逃離戰鬥，成功率為50%（但攻擊領地時無效，必定失敗）。
