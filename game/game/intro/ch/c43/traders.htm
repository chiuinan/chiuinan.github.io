名稱：外星大富翁 (Traders，中文版)
簡介：LINEL 1992年出品的大富翁遊戲，由漢堂修改成中文版並代理發行。雖然遊戲的名稱
      是大富翁，不過其本質反倒像是策略遊戲。遊戲中並沒有大富翁遊戲傳統擲骰子移動
      的設計，而是以回合的方式由玩家決定各項策略，至於遊戲的目標則是在比誰先達成
      所設定的金錢目標。遊戲的過程包括搶地、開發、買賣、戰鬥等多種設計，種類繁多
      也有些複雜，不過在土地開發的策略設計上，則略嫌單調，不若一般策略遊戲好玩，
      也不像大富翁遊戲一樣簡易有趣。因此這個遊戲雖然另類，但娛樂性卻不見的很好。

攻略：（作者－青衫）

一、財產統計



每回合一開始都會列出所有角色的資產概況，以供玩家了解各家的排名。按下確認鍵後，即
進行下一步驟：建立房子（第一次）或擴張土地。

在此處按下BackSpace鍵可回到主畫面，進行存檔或環境設定。

二、建立房子



遊戲一開始便是要建立自己的房子，亦即自己的根據地。各角色分佔地圖四個角落，按確定
鍵後開始移動人物到所要選擇的地點，再按確定鍵決定蓋房子。不過只有平原地帶能蓋房子，
所以在按確定鍵開始前，最好先行挑好離角色最近的蓋房地點。要注意的是，房子蓋的地點
牽涉到將來土地的佔用，因此應儘量蓋在中央的地方。在蓋好房子後還可再佔一塊地，因此
切莫連續按兩次確定鍵而略過了土地的佔有。由於可佔領的土地必須緊鄰自己的土地，因此
應儘量以先擋住對手土地的鄰接為要，只要空地不與對手鄰接而與自己土地鄰接，將來便可
慢慢將之佔領。特別是本遊戲的目標在於是否達成所設定的金錢目標，而金錢的計算包括了
所有資產，對於不花成本即可佔領的土地，當然要盡其所能的佔領了。

要選擇土地，只要將人物移往所要佔領的地點後，再按確認鍵即可，不過由於對手也同時在
移動佔領，因此手腳一定要快。佔領了土地以後，記得要再選開發模式，只要在土地上按下
確認鍵，然後選擇開發的模式即可（上方人物的圖像會變更為開發模式的選擇圖案）。初期
大都以探測火箭為主（不能在平原上），至於其他的開發模式，我們後面會再提到。全部都
完成了以後，只要回到自己家中按下確認鍵，即可進行下一步驟。

在建立房子與佔領土地的過程中特別要注意的是，人物每走一步，便會耗損一公升的蝸牛油
（如果有設定此項設定的話），因此可別到處走來走去，浪費資源。

三、探測火箭



探測火箭的目的，是要到其他星球尋找適用的資源，包括蝸牛、石油、甲蟲等，這些物品的
作用我們後面會再提到。當土地的開發模式為探測火箭時，下一回合便是選擇探測的地點，
一共有12個星球可選擇。不過火箭飛行的過程有可能受到星際海盜的攻擊，在星球旁邊有加
骷髏圖案的，表示其附近有星際海盜聚集，因此最好避開這些星球。不過若標示的骷髏顏色
和自己人物相同時，表示該群海盜和己方關係友好，因此反而會更順利。另外，被其他角色
選定的星球便不能再選了，因此選定的動作要快。

選定星球後，按下確認鍵確認。當所有角色都選完了，按下確認鍵，便會開始火箭昇空，只
要火箭順利到達該星球，即算探測成功，接著下個步驟便有報告指出收集回來的資源結果。
要注意的是，平原地形是不允許使用探測火箭的。

四、資源收成



驗收各土地的開發成果（包括探測火箭）。按左右鍵到各土地，以了解其收成結果（會顯示
在上方圖像中，同時也可了解其他角色的收成狀況）。每個土地都驗收後，即進入下個步驟。

五、購買機器人裝備



如果設定中有機器人戰鬥的話，那麼每回合都會出現這個畫面，由於沒有時間的限制，因此
可以好好的挑選。在經費方面，能購買機器人裝備的經費，一律為資產淨值的1/10（最低為
$500），而且經費無法累積至下一回合（只有裝備可累積至下一回合），因此購買時，一定
要買到經費用光為止。

選擇商品，再按上鍵或確認鍵購買，最後按下鍵離開商店。畫面中的商店由右到左依次為：

六、機械人戰鬥



戰鬥一開始，首先要修築防禦工事。先將游標移動到機械人停放處（Ｐ點），按下確認鍵，
此時顯示視窗上的外框會開始閃動，利用左右鍵選擇想佈署的裝備，然後按下確認鍵確認，
接著便可以選擇倉庫四周適當的位置來佈署這項裝備了。其中障礙牆在同一地點可佈署六次，
如此敵人必須攻擊六次才能將它摧毀。要更換裝備，則再移動Ｐ點按確認鍵即可。裝備重複
佈在同一地點，則後者會取代前者。又，我方的地雷與圍牆不會影響我方的機器人。當佈署
完畢後，再選擇裝備顯示欄中的完成即可進入戰鬥。

戰鬥時，按一下確定鍵使機械人出現，但如果機械人沒有任何武器，則它便會躲起來不出現，
因此在購買裝備時記得買些武器給機械人。要移動機械人，上下鍵分別表示前進和後退，但
後退速度為前進的一半，左右鍵則為旋轉。當要發射迫擊砲時，只要按一下確定鍵即可（注
意發射方向）。若要發射反機械人火箭，則一直按住確定鍵，直到機械人變成黑色時再放開，
火箭便會自動射出，不過必須敵人在附近才行。

當有河流擋路時，便需搭橋前進。要搭橋時，一直按住確定鍵，直到機械人變成黑色再變回
原來的樣子時再放開，即可搭建出來。不過搭橋時，只能往某個方向搭，且必須搭在河兩岸
的陸地上。

當摧毀某方的倉庫時，便可掠奪該角色的財物，掠奪的效果依機械人設備而定。至於戰鬥的
結束有三種：

1.將機械人帶回基地上，按下確認鍵，視窗外框閃動後，再按上鍵離開。
2.按Esc離開。
3.機械人彈藥用盡或被擊毀。

因此如果對方快打到自己的倉庫時，不妨按Esc離開，算是一種投機法。

戰鬥結束後，沒有消耗掉的裝備在下一場機械人戰鬥還可再使用，包括已佈下的圍牆和地雷。

七、商品交易



本遊戲的重要資源主要有水、花、香料、蝸牛和石油，其中石油必須到商店街購買，其餘的
四種資源便由四位遊戲主角相互交易，互通有無。水、花和香料為土地開發時的所需與收成
資源，蝸牛則為每回合必須上繳的稅。商品交易時，一共分成四個部份，依次交易的物品為
水、花、香料和蝸牛。在畫面右邊為各角色現有的貨品存量，底下的數字，左邊為現金數，
右邊為貨品數。有時在玻璃槽中還有個黑色的橫線，表示目前土地開發時所需的貨品量，若
存量低於該線，便會導致土地開發的收成不好。而在左下角的數字，表示目前貨品交易時的
最低價格。

交易時，賣方在上，買方在下，可用上下鍵來選擇。由於商品交易有時間限制，因此要迅速
決定，或用Ｐ鍵暫停遊戲來思考。決定交易方式後，再來便是開始正式交易了。



正式交易時，左邊為買方容器，右邊為賣方容器，買賣雙方必須調整售價與買價來進行交易，
可用上下鍵來調整。若已不想賣出，則按確認鍵離開，若已不想買進，則按下鍵低過最低價
即可。如何利用商品交易來圖利自己，以便控制對手的收成，便是值得玩家深思的事了。

八、胖麥克大方送

這是個特殊事件，不一定會發生，通常是在該回合全部收成都不好時。發生時沒有任何等待，
因此必須趕緊做出判斷去找出最佳的路徑收集胖麥克給的物品（只要人物經過即可）。這些
物品大致便是商品交易時的四種資源。

九、繳交蝸牛稅



這是每回合都會發生的事，因此記得常備有蝸牛以支應蝸牛稅。蝸牛的來源除了購自對手外，
最主要的便是使用探測火箭了。繳交時，胖麥克會有三次不同的要求數值，高低並不一定，
玩家必須決定是否要在此次支付。其實如果數額不高，自己又付得起，最好還是付完比較好。
如果三次機會過了又不繳交，則會被收回幾塊土地，因此最好還是能付時先付。如果確定要
付時，只要按下確認鍵即可，不過繳交蝸牛稅是有時限的，因此決定要付時行動要快。此外，
每三回合便有一次優秀納稅人選拔，只要蝸牛數最多的便會當選，可以得到金錢或土地，若
獎品是土地時，可以任選一塊土地，且不必和自己土地鄰接，因此最好儘量維持蝸牛的數目。

十、商店街



在繳完蝸牛稅後，便可以上街購物了，不過購物是有時間限制的，因此思考與操作的速度要
快些。首先按左右鍵移動到商店門口，按上鍵進入商店，然後按下鍵離開商店。以下是畫面
中由右至左的各個商店，而各商店物品的單價則在上方告示牌中：

1.寶石店：將蟲子換成寶石或賣掉寶石
2.火箭店：購買火箭，按確定鍵購買
3.油店  ：購買石油，每人所能持有的最大上限為４０公升，而現有油量在商店前的牌子上。
          由於眾角色可能會搶購，因此行動要快。
4.大家樂：進入後按左右鍵選擇燈號顏色，當開獎時的燈號與自己選的燈號相同便是中獎了。
          沒事的話便來此碰碰運氣吧。

逛完商店街，便是一回合的結束，又回到回合開始的財產統計。

十一、擴張與開發土地

每次出現地圖畫面時，便是擴張土地與土地開發的時候了。記得要先依”建立房子”中提到
的要領先佔土地（每次增加一塊），然後再決定各土地的開發。全部決定後再回到自己房子
按確定鍵即可，同時胖麥克還會發一筆開發獎金。不過如果石油不足時，便連動也不能動了，
特別是走到一半若石油用盡時，會自動送回到自己的房子所在，同時開發獎金也會被沒收，
因此在移動人物時要特別注意石油的剩餘量。

在開發土地時，除了探測火箭外，還可開發水池、免子、花等三種項目，其原料、收成、和
天氣好壞關係如下：

    ┌─────┬──┬──┬──┐
    │          │水池│兔子│ 花 │
    ├─────┼──┼──┼──┤
    │主原料    │ 水 │香料│ 花 │
    │副原料    │香料│ 花 │ 水 │
    │收成      │ 花 │ 水 │香料│
    │陽光／熱浪│ － │ 壞 │ 好 │
    │刮風／颱風│ 好 │ － │ 壞 │
    │下雨／下雪│ 壞 │ 好 │ － │
    └─────┴──┴──┴──┘

要注意的是，有兩塊土地以上開發同一產品時有加乘效果，因此愈多塊土地開發同一產品，
效果愈好。當然地形也會有一些關係。此外，開發該產品的土地佔持有土地的百分比，也會
影響到生產的效果，因此最好全部開發同一產品，不過要注意原料是否足夠，否則收成會不
好。在決定開發時，要看看別的角色在開發什麼，避免開發同一產品太多，否則賣價會不好。

除了開發土地外，你也可以賣地，不過這是最後不得已的手段了。
