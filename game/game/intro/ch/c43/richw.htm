名稱：大富翁環遊世界 (中文版)
簡介：定碁科技1990年出品的大富翁遊戲，由第三波代理發行。整個遊戲其實就等同紙上大
      富翁遊戲一樣，以各國公民權為標的物，環繞一圈通過起點後便可再領2000元。遊戲
      中可選擇1～4個人玩，不過通常都是只選擇一個人玩，此時電腦控制的人數只有一人，
      形成和電腦進行兩人對抗，過程變得有點平淡無聊。此外，遊戲過程中並無法存檔，
      無法用存取大法來改變敗勢，這點讓人玩起來有點不爽。因為即使像這樣陽春的遊戲，
      仍不免有一般智育遊戲中電腦自己做手腳之嫌。



保護：不明。
備註：經查閱遊戲反組譯程式碼，確認寫檔動作均不會被觸發，亦即遊戲不會存檔。
操作：

Enter或Space = 動作（擲骰等）
F1 = 檢視資產
F5 = 聲音切換
F10 = 結束遊戲
Esc = 取消
