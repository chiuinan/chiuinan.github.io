資訊修改
修改：WDATA.DAT

1.道具資訊：位置0000h開始，按編號排列，每個24h=36 byte

0000 NN NN NN NN NN NN ?? TY PR PR AT ?? DS DS DS DS
0010 DS DS DS DS DS DS DS DS DS DS DS DS DS DS DS DS
0020 DS DS DS DS

NN = 道具名稱
TY = 道具形態，00=牌卡，01=短兵類武器，02=長桿類武器，03=弓弩類武器，04=火槍類武器
PR = 價格
AT = 武器攻擊力（道具另有其他意義）
DS = 道具說明

2.君主資訊：位置0D5Ch開始，按編號排列，共14位，每位50h=80 byte

0000 N1 N1 N1 N1 N1 N1 N1 N1 N2 N2 N2 N2 N2 N2 N2 N2
0010 F1 F1 F1 F1 F1 F1 F1 F1 F1 F1 F2 F2 F2 F2 F2 F2
0020 F2 F2 F2 F2 NN NN NN NN NN NN IN OD P1 P2 P3 P4
0030 ?? ?? -- -- -- -- T1 T2 T3 T4 T5 T6 T7 T8 T9 TA
0040 TB TC TD TE TF TG TH TI MN MN MN MN SD SD SD SD

N1～N2 = 相關名稱1～2
F1～F2 = 相關檔名1～2
NN = 君主姓名
IN = 是否參與遊戲，02表由電腦操控，01表由玩家操控，00表未參與
OD = 參與遊戲時的順序編號，由00開始
P1 = 內政思想，00=儒家思想，01=法家思想
P2～P4 = 內政制度：一夫多妻制、強徵民夫制、貴族至上制，00表禁用，01表啟用
T1～TI = 道具／武器編號（參見裝備列表）
MN = 金錢數
SD = 兵員數

3.將領資訊：位置11BCh開始，按編號排列，每位36h=54 byte

0000 NN NN NN NN NN NN F1 F1 F1 F1 F1 F1 F1 F1 AN AN
0010 AN AN AN AN ID -- ST WT WP A1 A5 A4 A2 A3 ?? ??
0020 -- -- N1 N2 -- -- FT ?? RT ?? -- -- -- -- -- --
0030 -- -- -- -- -- --

NN = 將領姓名
F1 = 相關名稱
AN = 特別能力名稱
ID = 將領編號
ST = 狀態，00=健康，01=生病，02=重傷，03=發瘋
WT = 使用的武器形態，1=短兵類，2=長桿類，3=弓弩類，4=火槍類
WP = 手持武器編號（參見裝備列表）
A1～A5 = 忠誠度、貪污度、統御力、智力、武力（含裝備攻擊力）
N1～N2 = 手持武器、特別能力的攻擊次數
FT = 前軍攻擊形態，02=攻擊全體
RT = 後軍攻擊形態，特定編號可攻擊全體（例如02,17h～21h,24h,25h等）

1.城池資訊：按編號排列，每個28h=40 byte

WDATA-1.DAT = 秦朝版圖
WDATA-2.DAT = 唐朝版圖
WDATA-3.DAT = 明朝版圖

0000 NN NN NN NN NN NN LD PT A1 A2 P1 P1 P2 P2 SD SD
0010 G1 G2 G3 G4 G5 G6 G7 G8 GN -- -- ?? ?? ?? ?? ??
0020 ?? ?? ?? ?? ?? ?? ?? ??

NN = 城池姓名
LD = 所屬君主編號+1，0表無人城池
PT = 城池形態，00=驛站，01=城鎮，02=名勝，03=關邑，04=神廟
A1～A2 = 人民忠誠度、建設度
P1～P2 = 目前人口數／最大人口數
SD = 駐軍兵員數
G1～G8 = 城內將領編號
GN = 城內將領數目
