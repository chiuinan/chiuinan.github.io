記憶體修改
修改：人物資料在記憶體中的安排為（每位28h=40 byte）：

-- -- -- XX YY ZZ CL LV WP AR IT IT IT IT IT IT
IT IT H1 H1 H2 H2 MT MT DF DF -- -- AP AP DP DP
EX SS MX TT TT TT TT EV HT MV DX -- -- -- -- --

XX,YY = 所在座標（左上角0,0開始）
ZZ = 出場順序（影響地圖上顯示的肖像）
CL = 職業代碼
LV = 等級
WP = 武器代碼
AR = 護甲代碼
IT = 道具，共4個，每個2 byte=代碼（FF=沒有）＋數量／剩餘次數
H1/H2 = 目前／最大生命值
MT = 力量（影響AP）
DF = 耐力（影響DP）
AP = 攻擊力（換章會依力量值與武器重新計算）
DP = 防禦力（換章會依耐力值與護具重新計算）
EX = 經驗值（每滿100升一級，改成255可連升2-3級）
SS = 動作狀態，00=尚未行動，10=死亡，80=行動完畢
MX = 人物代碼（影響人物圖像、名字、最大成長值範圍）
TT = 四個道具各自最多的使用次數
EV = 閃避率（換章會依速度值重新計算）
HT = 命中率（HIT，換章會依速度值重新計算）
MV = 移動力
DX = 速度（影響HIT和EV）
