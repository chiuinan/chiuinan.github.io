前言

這是一段很久很久的歷史了。在遠古時代，有兩股勢力正互相對戰，他們就是
傳說中的聖族及魔族，當時魔族的領導者是大邪神，其強大無比的力量，一度
使得聖族落於下風。最後，聖族派出了精英的十二戰士，對大邪神展開突襲，
以求結束戰爭。最後大邪神雖然被擊倒了，但十二戰士也傷亡殆盡，為了避免
大邪神再度復活，剩餘的戰士將大邪神的軀體，封印在遙遠空間的一塊小島上，
並守護此地，以防封印失效。這個小島，便是後來所稱的達拉尼亞島。

失去首領的魔族，再也無力和聖族對抗，僅剩的少數魔族，只好利用大邪神的
剩餘魔力，紛紛逃到達拉尼亞島上。他們雖然想要設法解除封印，但在聖族戰
士的守護下，從來沒有成功過。不過魔族的行動，卻對後來遷居到島上的人類，
造成了極大的困擾。為了對抗魔族的不斷侵擾，原本散居的人類組成了軍隊，
終於遏阻了魔族的騷擾行為。擊退魔族之後，人類為了預防魔族捲土重來，便
倡導成立一個有組織的國家，建立常備性的軍隊。而率領軍隊有功的卡亞那將
軍，便被推舉來領導國家，這就是羅特帝亞王國的由來。

在許多年以後，消失已久的魔族，突然大舉侵攻王城，而原本鎮守邊疆的大將
軍哈奇，也趁著這個機會，突然起兵叛亂。皇宮在一夕之間，毀於大火之中，
只有忠心的御前侍衛克隆，受到國王亞美達二世的遺託，將尚在襁褓之中的二
王子雷特，從一片混亂中救了出來。為了避免哈奇的追殺，克隆在南方偏遠的
利達村定居，除了隱姓埋名外，還特地在當地娶妻生子，以避免引人耳目。但
是在妻子佩亞生產時，卻不幸因難產而去世。強忍悲痛的克隆，便一手將雷特、
獨生女米蘭，及後來收養的艾利歐，茹苦含辛地撫養長大，就這樣過了十幾年。

在雷特十八歲的生日當天，克隆將雷特叫到跟前，並告訴他真正的身世。悲憤
不已的雷特，決定替父兄報仇，就在這個時候...
