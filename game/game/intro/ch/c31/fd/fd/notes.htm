新手提示
操作：

1.快捷鍵
    F1    = 顯示全地圖
    Esc   = 切換至下一個未行動的人物
2.游標在地圖上
    Space = 打開系統選單（回合終了、顯示全地圖、過關條件、系統設定、存取檔：選存檔才會問是否結束）
            系統設定中，動畫=敵方攻擊動畫，視窗=地形視窗
3.游標在人物上
    Space = 打開操作選單（移動、攻擊、使用法術或物品、查看狀態、休息：恢復20%生命）
    F2    = 查看狀態

提示：（作者－青衫）

1.本遊戲有兩個結局路線，端視第七章如何通過，請參見遊戲路線圖，以及
  攻略第七章的說明。
2.遊戲中人物能力的強弱，一半取決於升級增加的屬性值，一半取決於裝備。
  故要容易過關，除了購買最強的裝備外，便是努力升級。想要練功升級，可
  以在每章留下一個最高等級的弓兵，圍住讓他每回合休息回血20%，然後慢慢
  攻擊（不打死他），來不斷取得經驗值。不過必須從頭到尾停著不動，且不
  是跟隨著關卡王移動的弓兵，否則他不會休息回血（攻略會告知那隻可練功）。
  如果是弓兵，則可以去打停著不動，且移動力較低的敵人，只要不進入他的
  攻擊距離，敵人便只會休息回血。
3.簡易難度（Easy）攻擊所獲得的經驗值，以及各章結束後所獲得的金錢，都
  會比普通難度（Normal）多，升級比較容易，也能買到所有的最佳裝備，因
  此簡易難度大多不必努力練功升級。不過選簡易難度走路線二時，是無法看
  到最後的結局劇情與圖片的，必須選擇普通難度才行。另外，金錢最多只累
  積到65535，因此簡易難度時，能買的裝備要儘量買，才不會浪費。
4.普通難度的金錢不夠購買所有最強的裝備，因此要妥善規劃各章應買的裝備。
  攻略中商店買賣物品後面的建議，係個人針對各章難度與金錢多寡，所提出
  的裝備購買與移轉清單，玩家可自行參考斟酌。此建議清單的考量如下：
  (1)攻擊就是最好的防禦，因此每個人到最後，除了法師外，應儘量裝備最強
     的武器。而法師則視情況購買需要的法術，不一定購買最強的法術。
  (2)由於金錢總數不足以購買所有最強的護甲，因此二線不入流的人員只配予
     次一級的護甲，主力人員一律使用最強護甲。弓兵因不會直接和敵人接觸，
     只配予轉職前最佳的鎖子甲即可，有閒錢才再往上一級。
  (3)各章的遊戲過程，仍要購買足以過關的裝備，不能過份省錢而完全不買，
     只靠升級增加的屬性撐，使得過關的困難度大增。新買較好的裝備儘量給
     一線人員使用，二線人員就撿別人不要的裝備穿。不過增購的裝備仍嫌不
     足，因此若有建議練等級的地方，一定要練到足夠的等級。
  (4)到最後一章要把金錢花到幾乎全盡，不可浪費。
5.各職業最多20級。當人物從第19級升到第20級時，會自動轉職成更高職業的
  第1級。以低職業第1級就加入的人物而言，總共可升18＋20級。
6.升級時屬性的增長範圍很大，甚至不增長，因此要特別注意升級時增長的值，
  若實在太差，最好讀檔重來（遊戲可隨時存檔），否則到後面可能會打不太
  動敵人。各人物升級的屬性最大增長值，請參見人物職業升級屬性一覽表。
7.法術採道具方式處理，可在商店購買（可重複購買，最多四個），之後在戰
  場上便可使用該法術道具施展法術，但每個法術道具都有數量限制，用完就
  不能再使用。每過一個章節，法術道具的次數會自動恢復（非消耗品）。注
  意補血法術使用後是沒有經驗值的。
8.人物的最大生命力是由裝備的護甲來決定的，並不隨著升級而增加。護甲有
  等級限制，因此在各章之後，儘量要練到可購買穿上該章最強護甲的等級。
  護甲另外控制了其餘物品的載重力，如果購買東西的重量加上人物身上武器
  與物品（包含法術）的重量大於該護甲的可承載重量，那麼就無法購買該物
  品。因此若在購買裝備或道具時，出現帶不動的訊息，就應考慮移走或賣掉
  一些道具、換成更輕的武器，或是更換成可承載更重的護甲。
9.在戰場上選擇休息可恢復20%血，因此基本上並沒有需要買補血的藥。但要注
  意若選擇回合終了是無此一效果的，必須一一對人物做休息的動作才能回血。
10.雷特和莉娜在轉職後都能使用初級法術，因此不要的初級法術可以留下來轉
  給他們，否則到後期便沒得買了。
11.商店物品賣出的價格為原價的75折（3/4）。
12.由於遊戲設計上的疏忽，若萊特每次升級都是最大值時（例如使用最大值成
  長），那麼滿級後，閃避率將會Overflow而歸零。解決方法有：
  (1)如果是手動升級，不要每次都最大值，至少要少掉7，加乘後剛好為255。
  (2)如果是使用最大值成長，可以在轉職後，到20級前讓他穿鎧甲，防/血/移
     動力都略降，但閃避率變成252，只是這樣變成最多只能裝備銀矛，攻擊
     力會降低不少，並不太值得。
  (3)如果不想有前述問題，也可以修正它的升級最大值，例如轉職後速度最大
     值資料從8改成7（也就是最大值變成6），這樣閃避率變成240，基本上還
     是很高，但已不會有滿溢的問題。
13.每個人物幾乎都有特定屬性的加乘，轉職後，加乘的比率與範圍都會再擴大，
  請參見人物職業升級屬性一覽表。
14.遊戲中使用的公式：

   AP = (力量+武器AP) * (1+人物加乘率)
   DP = (耐力+護甲DP) * (1+人物加乘率)
   HIT = (速度+武器命中率) * (1+人物加乘率)
   EV = (速度+護甲速度) * (1+人物加乘率)
   物理攻擊傷害 = (AP-DP) + 10%隨機增減，AP=攻方AP*(1+地形AP修正率)，DP=守方DP*(1+地形DP修正率)
   物理攻擊命中率 = (攻方HIT-守方EV)%
   法術攻擊傷害/恢復 = 法術內定傷害/恢復 + 10%隨機增減
   法術攻擊命中率 = 法術內定命中率
