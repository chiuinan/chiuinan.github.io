名稱：炎龍騎士團外傳－風之紋章 (中文版)
簡介：漢堂1998年出品的角色扮演遊戲，以2D畫面呈現，採團隊回合方式戰鬥。
      劇情描寫蘭迪斯隨著前代主角索爾冒險的事蹟，最後消滅黑暗教派召喚
      的魔王，維護了世界和平。整個劇情其實並不怎麼樣，但所有二代的元
      素，本遊戲都有，因此基本上仍有一定的娛樂性。難度已比二代為高，
      欲過關需多花許多時間。
 
備註：本版已做成OGG音樂版
保護：光碟，用DosBox玩時，應設定：
        mount d [光碟所在磁碟]:\ -t cdrom -usecd 0
      遊戲會讀取光碟上的音樂和動畫檔，故亦可燒錄成檔案模擬光碟來避免
      （由於是資料與音軌混合，不可以轉成iso檔）。破解碼如下：
      FDPS.EXE（破解後無音樂）
        74 26 B8 3C
        EB -- -- --
        F8 00 E8
        -- 01 --
備註：

1.mod1目錄下的FDPS.EXE = 最大值成長、按Esc進入神秘商店、行動後可再行動、調整遊戲速度
2.mod2目錄下：
    FDPS.EXE = 同上，三倍速成長、徽章不消失、風精之羽不消失
    FIELD.VFS = 一次玩三種路線（無轉職限制）
3.mod3目錄下的FDPS.EXE = 同上，限制50級

檔案：fdps-s.rar = 簡中版 (主要差別在FIELD.VFS字形檔)
