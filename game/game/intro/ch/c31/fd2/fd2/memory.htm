記憶體修改
修改：人物資料在記憶體中的安排為（每位50h=80 byte）：

-- -- -- -- -- -- -- -- XX YY Z1 Z2 Z3 AA BB FA
NN NN IT IT IT IT IT IT IT IT IT IT IT IT IT IT
IT IT M1 M2 M3 M4 M5 RA CL LV A+ D+ H+ -H XA XM
-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- MT
MT DF DF MV EX -- DX DX H1 H1 H2 H2 M1 M1 M2 M2
AP AP DP DP HT HT EV EV -- -- -- -- -- -- -- --

XX,YY = 所在座標（左上角0,0開始）
Z1 = 人物圖形
Z2 = 人物圖形方向
Z3 = 人物圖形跑步動作
AA = 動作狀態，00=尚未行動，01=死亡，80=行動完畢
BB = 陣營，00=敵方，01=友方，02=己方
FA = 肖像編號
NN = 人物姓名
IT = 物品，共8個，每個2 byte=狀態+物品編號，狀態值40=裝備中、00=持有中、80=無物品
M1 = 法術一：01=火炎，02=烈炎，04=炎龍，08=天火，10=電擊，20=落雷，40=轟雷，80=神雷
M2 = 法術二：01=聖光彈，02=咒殺，04=碎岩，08=地震，10=裂地，20=治療，40=回復，80=再生
M3 = 法術三：01=神恩，02=魔刃，04=魔鎧，08=風行，10=解毒，20=祛麻，40=封咒，80=傳送
M4 = 法術四：01=破龍擊，02=行動術，04=毒擊，08=麻痺，10=淒煌斬，20=熾炎刀，40=音速刃
M5 = 法術五：01=熾天使，02=風妖精，04=破壞神，08=暗邪鬼
RA = 種族編號
CL = 職業編號
LV = 等級
A+ = AP增強狀態（00=無，02=有）
D+ = DP增強狀態（00=無，02=有）
H+ = HIT/EV增強狀態（00=無，02=有）
-H = 中毒狀態（00=無，02=有）
XA = 麻痺狀態（00=無，02=有）
XM = 封咒狀態（00=無，02=有）
MT = 力量（影響AP）
DF = 耐力（影響DP）
MV = 移動力（不可太大，最好25以內）
EX = 經驗值（每滿100升一級，改成255可連升2-3級）
DX = 速度（影響HIT與EV）
H1/H2 = 目前／最大生命值
M1/M2 = 目前／最大法力值
AP = 目前總攻擊力（MT+武器AP，修改無效）
DP = 目前總防禦力（DF+防具DP，修改無效）
HT = 命中率（HIT，由DX計算，修改無效）
EV = 閃避率（由DX計算，修改無效）
