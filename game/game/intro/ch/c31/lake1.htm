名稱：聖少女戰隊 (Lakers，中文版)
簡介：Apple Pie 1994年出品的１８禁策略角色扮演遊戲，由華義中文化並代理發行。劇情
      描寫卡魯邦帝國邪帝加爾文佔領布蘭德星球後，想要繼續侵略地球，而主角沼田俊係
      布蘭德星球人，集合另五位來自布蘭德星球的少女，藉用雷卡水晶的神力，變身成聖
      少女戰隊，最後打敗邪帝加爾文，保衛了地球的故事。雖然遊戲中的角色可以升級，
      但升級後只提升生命力和法力，其他屬性均未變動，亦無武器防具可購買，使得遊戲
      顯得單調了些。至於在劇情演變過程中所添加的色情部份，顯得多餘且無聊，因此這
      款遊戲只能算是普通而已。



保護：數字密碼，破解碼：
      SAN.EXE
        33 FF E9 E1 02 9A  (不用輸入密碼)
        -- -- -- D1 -- --
      或
        73 03 E9 17 FD  (不用輸入密碼)
        -- -- -- -- 90
      或
        9A 92 05 4E 02  (不用輸入密碼)
        EB 03 90 -- --
      或
        3B 56 F6 75 18  (密碼隨便輸入)
        -- -- -- EB 0F
備註：若安裝時卡在要求1.44M磁片，可進行下列修改：
      ISAN.EXE
        EB 05 8B C6 E9 58 02
        -- 00 -- -- -- -- --
備註:主程式SAN.EXE有壓縮，請使用XOE或UP反解壓，不可使用UNP或EXESHAPE，解壓後大小
      為80288 byte。
密技：安裝後在其目錄裡建一個檔名為MEMORY的檔，即可觀看過關圖片。
操作：進入後按滑鼠左鍵繼續。

滑鼠左鍵＝執行選項
滑鼠右鍵＝回到上層選單
