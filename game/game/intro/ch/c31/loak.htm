備註：安裝須知

1.DOS版硬碟空間可能誤判而無法採用建議安裝，可先採用最小安裝，再將下列檔案拷入：
    OCIV.PRG, OCIV1.MAP
  最後修改OCIV.CFG第一行DEVICE第一個數字為2。
2.Windows版必須將setup.exe設成Win9x相容。
3.安裝的相關資訊會寫在OCIV.CFG檔案中，若環境變更，必須更改第一行DEVICE最後一個字
  元（光碟所在磁碟，大寫字母）。

備註：是否可逆轉職必須執行SETS.EXE或SET.EXE進行設定，內定為否。
備註：遊戲過程換片方式

1.DosBox：按下Ctrl+F4換片，但必須先使用下列方式掛載雙光碟：

imgmount d .\CD\GCD004-1.cue .\CD\GCD004-2.cue -t cdrom

2.Windows：按住Shift不放（直到換片後回到遊戲中），按Alt+Esc切回桌面換片後，再切回
  遊戲中放開Shift鍵，最後按下滑鼠左鍵。由於切回桌面再切回遊戲，可能造成顯示不正常，
  建議需要換片前先存檔，然後改用D3DWindower掛載，方便換片。但用D3DWindower掛載，
  滑鼠會不正常，必須使用鍵盤操控，因此最好只用來處理換片事務。如果都不會使用，請
  使用不必換片的破解版。

備註：玩西方路線（布雷克），第一關完後便跳出表示換錯片，請載入進度並更換正確光碟。

密技：

一、通用密技

戰場上按Esc鍵出現系統選單後，再按Alt+F5，出現PassWord，輸入JACK98，成功會有音效。
以下為密技按鍵：

Alt+V：修改滑鼠指向的人物屬性
Alt+R：滑鼠指向的我方人物可再次活動 
Alt+N：結束任務，立即過關
F10：修改金錢和難易度（要在戰場上）

二、進階密技

戰場上按Esc鍵出現系統選單後，再按Alt+F5，出現PassWord，輸入98U&JOCIV，成功會有音
效。以下為密技按鍵：

Alt+V：人物功能表
   1.生命點數
   2.法力點數
   3.精力點數
   4.魔力值
   5.反應力
   6.攻擊力
   7.防禦力
   8.劍術值
   9.戰技值
  10.經驗點數
  13.各種技巧點數(1~24，7~24輸入數字後會*10)
  14.座標位置
  15.取得物品，輸入物品編號
  16.相性值
  17.性別
  18.離開
  19.關係值
Alt+R：滑鼠指向的我方人物可再次活動 
Alt+T：滑鼠指向的敵方人物可再次活動 
Ctrl+A：滑鼠指向的人物可暫時學到所有的魔法及招式
Alt+N：結束任務，立即過關
Ins：新增人物(新增人物太多會導致當機)
   1.所屬陣營
   2.兵種編碼(1~150，輸入其他可能會當機)
   3.出場等級
   6.出場方向
   7.性別
   8.離開
Del：殺死滑鼠指向的人物
F10：通用功能表
   6.金錢數(調整太大可能會當機)
  10.難易度調整
ALT+F9：系統功能表
   3.得到所有的物品各一份（不含裝備）

操作：

Esc = 系統選單
Alt = 取消/快速略過對話 (Windows版才有)
Alt+A = 顯示版本

在休息站中按一下滑鼠左鍵，可顯示各商店與通道之位置名稱。
