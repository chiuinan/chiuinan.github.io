名稱：古大陸物語5：毀滅天境／大地之絆 (Farland Story: Bonds of Earth，中文版)
簡介：TGL 1995年出品的策略角色扮演遊戲，1998年中文化並委由智冠發行 [待補]。



保護：無（選擇無聲即可免光碟，若要有背景音樂則需選CD音源，並放入光碟）
備註：DosBox音效爆音與戰鬥動畫花屏問題，是因為EMM386的模擬問題，請以下列方式解決：

1.修改dosbox.conf，將EMS、XMS等功能均關閉：
　ems = false
　xms = false
　umb = false
2.在遊戲目錄中建立一個檔名為"EMMXXXX0"的空檔案。
3.下載JEMM，並執行"JEMMEX FRAME=E000 LOAD"來模擬EMM386。
4.執行遊戲。
