修改：存檔save\tks?.sav

位置000088h：金錢，4 byte，負數格式
位置0025D0h：各人物資訊，每位1F8h=504 byte，格式同人物資訊檔
位置0292A8h：戰場中人物資訊，每位70h=112 byte，次序會隨著人物移動而調換

00 -- -- -- -- -- -- -- -- ID ID ID ID ?? ?? ?? ??
10 ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ??

60 ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ?? ??
70 ?? ?? ?? ?? ?? ?? ?? ?? -- -- -- -- -- -- -- --

ID = 對應的人物編號
HH = 攻擊加乘
       04h = 法術攻擊時我方火系技能增加100%
       08h = 法術攻擊時我方水系技能增加100%
       40h = 法術攻擊時對方火系抗性減少50%
       80h = 法術攻擊時對方水系抗性減少50%
AS = 行動狀態
       02=已行動
       01=已決定面對方向
P1 = 抗昏迷指數
P2 = 抗混亂指數
P3 = 抗中毒指數
P4 = 抗癱瘓指數
S1 = 昏迷旗標，01表昏迷
S2 = 混亂旗標，01表混亂
S3 = 中毒旗標，01表中毒
S4 = 癱瘓旗標，01表癱瘓

位置02CA98h：戰場寶箱資訊，每個24 byte

XX XX XX XX YY YY YY YY PP PP PP PP IT IT IT IT
MM MM MM MM AC AC AC AC -- -- -- -- -- -- -- --

XX,YY = 寶箱位置
PP = 寶箱編號
IT = 物品編號，-1表金錢
MM = 金錢數
AC = 是否作用中，1表作用中
