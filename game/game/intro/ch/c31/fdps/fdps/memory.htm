記憶體修改
修改：人物資料在記憶體中的安排為（每位50h=80 byte，一有人行動位置就會變化）：

-- -- -- -- -- -- -- -- XX YY Z1 Z2 Z3 AA BB FA
NN NN IT IT IT IT IT IT IT IT IT IT IT IT IT IT
IT IT M1 M2 M3 M4 M5 RA CL LV A+ D+ H+ -H XA XM
-- -- -- -- -- -- -- -- -- -- -- -- -- -- -- MT
MT DF DF MV EX -- DX DX H1 H1 H2 H2 M1 M1 M2 M2
AP AP DP DP HT HT EV EV -- -- -- -- -- -- -- --

XX,YY = 所在座標（左上角0,0開始）
Z1 = 人物圖形
Z2 = 人物圖形方向，00=下，01=左，02=上，03=右
Z3 = 人物圖形跑步動作
AA = 動作狀態，00=尚未行動，01=死亡，80=行動完畢
BB = 陣營，00=敵方，01=友方，02=我方
FA = 肖像
NN = 人物姓名
IT = 物品，共8個，每個2 byte=狀態+物品代碼，狀態值40=裝備中、00=持有中、80=無物品
M1 = 法術一：01=業火，02=狂暴巨燄，04=烈獄之火，08=光之箭，10=聖光柱，20=落雷術，40=奔雷彈，80=暴雷絕擊
M2 = 法術二：01=冰爆術，02=絕殺冰封，04=裂地術，08=封神裂震，10=震空重力彈，20=鬼死靈動陣，40=恢復之光，80=治癒之風
M3 = 法術三：01=痊癒之泉，02=封魔咒術，04=腐毒術，08=麻痺術，10=神之祝福，20=傳送術，40=神行術，80=咒殺術
M4 = 法術四：01=甦醒術，02=魔龍霸炎，04=強力衝擊，08=金鋼斬，10=獄炎烈破彈，20=轟神砲，40=流星箭，80=靈彈超必殺
M5 = 法術五：01=審判之雷，02=鎮魂之歌，04=超重力黑洞，08=聖龍烈霸斬，10=暴炎狂龍，20=冰魔超速彈，40=極度冰凍，80=萬神降臨
RA = 種族代碼
CL = 職業代碼
LV = 等級
A+ = AP增強狀態（00=無，02=有）
D+ = DP增強狀態（00=無，02=有）
H+ = DX/HIT/EV增強狀態（00=無，02=有）
-H = 中毒狀態（00=無，02=有）
XA = 麻痺狀態（00=無，02=有）
XM = 封咒狀態（00=無，02=有）
MT = 力量（影響AP）
DF = 耐力（影響DP）
MV = 移動力（不可太大，最好25以內）
EX = 經驗值（每滿100升一級，改成255可連升2-3級）
DX = 速度（影響HIT與EV）
H1/H2 = 目前／最大生命值
M1/M2 = 目前／最大法力值
AP = 目前總攻擊力（MT+武器AP，修改無效）
DP = 目前總防禦力（DF+防具DP，修改無效）
HT = 命中率（HIT，由DX計算，修改無效）
EV = 閃避率（由DX計算，修改無效）
