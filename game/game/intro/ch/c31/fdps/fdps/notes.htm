新手提示

操作：

F1 = 縮圖
Esc/Pad-5/Del = 下個可行動人物
Space+己方人員：進行移動，移動後上=攻擊
                                左=施法
                                右=物品（使用、給予、裝備、丟棄）
                                下=防禦（未行動者休息回血20%，中毒與麻痺除外）
F2/Home+己方人員：顯示人物屬性
F2/Home/Space+敵方人員：顯示人物屬性
Space+空地：上=系統選單（條件、存檔、讀檔、退出遊戲）
            左=行軍，右=設定（音樂、音效、動畫、視窗）
            下=結束本回合（未行動者休息回血20%，中毒與麻痺除外）

提示：（作者－青衫）

1.人物等級到達20級以上時，便可到教會轉職（但最好等到最高的40級）。若
  身上有帶轉職的特殊道具（光之徽章、暗之徽章），便可以轉職成光明系或
  黑暗系的職業。若人物本身即是光明系或暗黑系，則不必特殊物品，即可轉
  成該系的職業。
2.若要讓蘭迪斯轉職成英雄，則在未拿到勇者徽章之前（也就是在第18章之前），
  不能把任何一人轉職，否則就無效了。在拿到勇者徽章後，即使把其他人轉
  職，也能把蘭迪斯轉職成英雄。
3.每100經驗值升一級，人物升級所增加的屬性是在特定的範圍內以亂數決定，
  可以利用存取檔來達到較佳的屬性。
4.遊戲有英雄、真炎龍劍、高能量砲三種路線，玩前要先決定一下。如果不知
  如何選擇的話，建議就走英雄路線。
5.遊戲有點難度，故每章最好都練一下等級，否則到後面可能會玩不下去。其
  中尤利安若學到神之祝福，要多利用這個法術升級，會比補血快非常多。
6.每章最多255回合，超過會當機。
7.物品賣出給商店為原價的75折（3/4）。
8.遊戲所使用的公式（小數點以下均去除）：

  物理攻擊傷害計算順序：
      AP = 攻方AP
      DP = 守方DP
      暴擊時，DP = 守方DP/2，取整數
      AP = AP*(1+攻方地形AP修正率)，取整數
      DP = DP*(1+守方地形DP修正率)，取整數
      最大傷害 = AP-DP
      實際傷害 = 最大傷害*0.9 ～ 最大傷害-1
  物理攻擊命中率 = (攻方HIT-守方EV)%
  絕招攻擊傷害計算順序：
      AP = 攻方AP*絕招加乘率，取整數
      DP = 守方DP
      最大傷害 = AP-DP
      實際傷害 = 最大傷害*0.9 ～ 最大傷害-1
  法術攻擊傷害計算順序：
      最大傷害 = 法術最大傷害*(1-守方魔法抗性)
      實際傷害 = 最大傷害*0.9 ～ 最大傷害-1
  法術攻擊命中率 = 法術內定命中率
