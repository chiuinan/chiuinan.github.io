名稱：FinFin海豚鳥／FinFin另一個地球 (TEO: The Other Eearth，中文版)
簡介：富士通1996年出品的動物模擬養成遊戲，由展筠代理發行。原版軟體有附贈一個專門
      用來輸入聲音互動的「TEO天線（TEO Antenna）」，但以現今硬體只能使用鍵盤互動。
      由於遊戲必須長時間互動才會慢慢有成果，中間一成不變的重複過程很容易讓人覺得
      枯燥無聊，沒耐心的玩家基本上會玩不太下去 [待補]。



備註：本版為1999年追加版
保護：光碟
限制：必須在Win9x系統上安裝，但可在XP以上系統執行，必須進行下列Registry註冊（假設
      安裝目錄在C:\finfin，光碟在D:\）：

[HKEY_LOCAL_MACHINE\Software\FUJITSU\fin fin The REAL Computer Creature\9.0]
[HKEY_LOCAL_MACHINE\Software\WOW6432Node\FUJITSU\fin fin The REAL Computer Creature\9.0]

"TeoSysPath"="C:\\finfin"
"DataPath"="D:\\"
"VolumeName"="FinFin6-1"
"ScreenMode"="-fullscreen"
"SupportedInterface"=dword:00000001
"SelectedVxDName"="TEOANTN0.VXD"
"SelectedOEMName"="TEO Antenna"
"RunMode"="install"
"WINPOS_U"="10"
"WINPOS_V"="10"
"WINPOS_W"="320"
"WINPOS_H"="240"
"SCR_Eye"="off"
"SCR_Eye_time"="30"
"SCR_Ear"="off"
"SCR_Ear_time"="30"
"SCR_Ear_sensitivity"="50"
"SCR_Auto_cancel"="off"
"SCR_Auto_cancel_time"="30"
"SCR_Finfin_special_event"="off"

操作：

1～5 = FinFin寶寶叫聲
Space = 餵食「愛樂蒙果實」
W = 使用口笛呼叫FinFin
F = 自動跟隨FinFin
S = 拍照
B = 心情探測器
Esc = 結束通訊
Tab = 切換「心情探測器」狀態／主畫面時切換場景
上鍵或Enter = 主畫面時進入選擇的場景
下鍵 = 場景回到主畫面
左鍵 = 進入「愛樂蒙袋」
右鍵 = 進入TEO天線
O = 在「愛樂蒙袋」可重新開啟一包愛樂蒙果實

*** 「心情探測器」狀態

Happiness = 快樂指數
Fear = 恐懼指數
Hunger = 飢餓指數
Thirst = 口渴指數
Fatigue = 疲倦指數
Sleepiness = 想睡指數
Temprature = 溫度指數
Friendliness = 友善指數

說明：需先建立通訊員後，才能進入遊戲。
