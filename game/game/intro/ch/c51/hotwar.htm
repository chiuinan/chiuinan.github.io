名稱：幻影特攻 (Hot War，中文版)
簡介：昱泉1998年出品的第三人稱／第一人稱射擊遊戲，改編自香港同名電影 [待補]。



保護：光碟
備註：非安裝移機者，必須進行下列Registry註冊（假設光碟機在D）：

[HKEY_CLASSES_ROOT\HotWar]
"SrcDisk"="D:"

備註：VirtualBox無法正常執行，虛擬機請使用VMware。
操作：

上/滑鼠右鍵 = 前進（連按兩下為跑步）
下 = 後退
左右 = 左右轉
PgUp/PgDn = 上下看
Alt = 蹲下
Ctrl/滑鼠左鍵 = 射擊
Space = 動作（調查、撿取、對話）
Shift = 使用道具
Ins = 啟動通訊器
F4 = 使用狙擊鏡
F5 = 使用夜視鏡
Tab = 視角切換
