名稱：鐵甲紀元：戰魂 (Metal Century O.G.R.E.'S Revenge，中文版)
簡介：新意1997年出品的第一人稱射擊遊戲 [待補]。



保護：光碟
備註：光碟上的README.TXT有標示demo版，尚不清楚是否為正式銷售版。
密技：執行時加上參數METAL_TWAYJCRK，則進入戰場時會是無敵狀態。
操作：

左右 = 左右轉向
上下 = 前進／後退
End/PgDn = 左右平移
WX = 軀幹上下傾
AD = 軀幹左右轉
QZ = 上飛（有飛行器時）／下降（蹲下）
12 = 開啟左右手（才能發射子彈）
34 = 開啟左右肩（才能發射飛彈）
Ctrl = 發射子彈
Alt = 發射飛彈
R = 變更雷達範圍
79 = 左右狀態螢幕開關
80 = 變更左右狀態螢幕
V = 啟動／關關自動瞄準功能
L = 鎖定目標
T = 換下個目標
U = 夜視鏡開關
C = 開啟通訊器
Space = 動作
Esc = 系統選單
O = 無敵（只適用於展示版）

說明：

1.必須先到L-Register，填入Call-Sign的名稱後，才能進入M（Mission）進行遊戲。
2.進入M後，要先到Briefing點選地點，看完任務介紹後，才能回來按Action開始任務。
