名稱：格鬥風雲 (中文版)
簡介：長重1992年出品的打鬥遊戲 [待補]。
 
保護：手冊密碼，破解碼：
      PASS.EXE
        8C 5E FA C7 46 F8 94 00 C7 46 F4 03 00 33 C0
        B8 C7 00 BA 00 90 8E C2 26 A2 FF FF E9 EA 02
備註：本版磁片裡的SHA壓縮檔均非正確資料，必須INSTALL.EXE檢測到正版磁片後，才進行
      修正並送入SHARC.COM解壓程式中。目前只有在PCem或86Box系統上掛載磁片檔，才能
      讓INSTALL.EXE檢測到正版磁片，DosBox掛載無效。自行拷貝檔案或解壓檔案，均無法
      正確安裝。
備註：C:\必須有CONFIG.SYS，並有FILES = 20這一行（若沒有，遊戲會自動加入並重開機）。
操作：

左右 = 前進／後退
上 = 跳起
下 = 蹲下
Home/Pgup = 向後／向前跳
Shift+Home = 上身防禦
Shift+End = 下身防禦
Shift+上 = 原地飛拳
Shift+右 = 正拳
Shift+PgUp = 飛拳
Shift+PgDn = 蹲身拳
Ctrl+上 = 原地飛踢
Ctrl+右 = 踢
Ctrl+PgUp = 飛踢
Ctrl+PgDn = 蹲身踢
Shift+Enter = 秘技1
Ctrl+Enter = 秘技2
Shift+Ctrl+Enter = 秘技3
Alt+M = 音樂開關
Alt+S = 音效開關
Esc = 結束遊戲
