名稱：爆笑出擊 (中文版)
簡介：大宇1992年出品的射擊遊戲，採水平捲軸前進方式射擊，全部計有７關。
      遊戲敵我的造型爆笑有趣，並提供６種武器選擇。不過由於己機被敵方
      一擊中便死，要過關並非易事。但整體而言，這是一款值得一玩的射擊
      遊戲。
 
注意：若無法正常執行，必須執行SETUP.EXE重新設定環境。
保護：圖案密碼，破解碼：
      MEO.EXE (先用UNLZEXE解開)
         3B 06 2A 00 75 04
         A1 2A 00 90 90 90
      或
         75 04 (共三組)

密技：

Tab+H = 加99隻
Tab+F3 = 跳關
Tab+F4 = 穿透（敵方子彈與敵機撞擊無效，但撞壁有效，且無法取得武器）

操作：

Shift-Q = 回到DOS
Shift+F9 = 加速
Shift+F8 = 變回原速
Space = 發射子彈，按住可持續發射
Ctrl = 切換武器

說明：

1.(紅)小砲１號：前射單一子彈，很弱，為最基本武器
2.(藍)筆心雷射：前射短雷射
3.(橙)菜刀雷射：前射兩條短雷射，以及上下兩道雷射波
4.(青)追蹤式小砲：前後射單一子彈，以及一枚追蹤飛彈
5.(銀)強力肉砲彈：前射一發強力砲
6.(粉)甜甜圈：前射一個由小變大的圓圈

己機被擊毀時，使用的武器也會跟著損毀（小砲１號除外）。
