名稱：青澀寶貝射擊，又名脫衣射擊雀 (Sentimental Shooting，日文版)
簡介：日本同人社團while (1);於1999年製作的一個十八禁射擊遊戲，遊戲以
      青澀寶貝的12位美少女為主，玩家必須在遊戲過程中，將美少女身上的
      衣服全部射爛脫下。整個遊戲簡單好玩，只是敵彈多而密集，過關相當
      不易。而且關卡捲動次數有限，若在捲動結束前未將衣服全部射掉，仍
      視同未過關。尤其某些地方只會出現一次，玩家若未趕到定點射擊，便
      無法過關，使得玩家難以兼顧閃躲敵彈。如果能夠改成不斷捲動，直到
      玩家射完衣服過關，那就完美了。



備註：本版為12人完整版，無需使用日文系統，且內含能源無限版、無敵版。
保護：無

*** 主畫面說明

上下鍵 = 移動
Z = 確認

1.Game Start = 開始射擊遊戲
2.Option = 遊戲選項
  (1)GAME MODE：困難度，共有三種類型：EASY、NORMAL和HARD。由於這個遊
     戲基本上是比較難的，建議先從EASY玩起。
  (2)PREVIEW：遊戲一開始是否將整個背景移動瀏覽一遍。如果已經玩過很多
     遍，可以關閉此選項，以加快進入射擊遊戲。
  (3)FREE SCROLL：過關後是否可自由滾動背景看圖。
  (4)FRAME SKIP：如果電腦速度不夠快，可開啟本選項略過部份繪圖，來減輕
     CPU的負載。
  (5)MUSIC：背景音樂的開關。
  (6)SOUND EFFECT：音效的開關。
3.Exit = 離開遊戲

*** 射擊畫面說明

Z = 射擊
X = 消彈
C或Esc = 暫停
Q = 結束射擊，返回角色選擇
Alt+F4 = 強制終止程式

1.每個人物有兩關，必須100%射破所有衣服才算過關（衣服射破後會褪去）：

第一關：學生服，敵人為小嘍嘍，玩家必須在背景捲動結束前射破所有衣服。
第二關：內衣，敵人為魔王，玩家必須在打敗魔王前射破所有衣服。

2.射破衣服後，會掉下次武器，然後依照玩家拾取的位置，決定次武器的附加
  位置，計有左中右3個位置：

L = 雷射
M = 飛彈

3.當我方無法閃躲敵彈時，可按下X鍵消彈，但必須消耗能源128點。能源初值
  為792（顯示在左上角）。
4.遊戲暫停後，可按下X鍵截取目前的畫面，或按下Z鍵只截取背景畫面。
5.過關後觀看圖像時，可按下X鍵截取整個背景圖。
