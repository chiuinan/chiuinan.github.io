名稱：拂曉攻擊－決戰中國海之役 (Dawn Raider，中文版)
簡介：大宇1990年出品的射擊遊戲，採垂直捲軸前進方式射擊，全部計有１２
      關。遊戲中己艦的屬性有機槍子彈、飛彈和燃料三種，子彈和飛彈射出
      後即會減少，而當被敵方擊中時，燃料值會先減少，當燃料全部用盡時
      即算結束，並非那種一擊斃命的射擊遊戲。遊戲過程中也有一些補充彈
      藥、燃料與增強火力的物品，不過補充的並不多。特別是燃料用盡時並
      沒有接關的設計，必須由第一關重頭開始，而敵機又特別的多，因此要
      完全過完１２關是不大可能的事。對於早期而言，這款已算是不錯的遊
      戲了。

操作：

左右上下／ADWX = 1P／2P移動
[／G = 1P／2P發射子彈，按久一些可發射雷電
]／H = 1P／2P發射飛彈
T = 暫停遊戲（按任一鍵繼續）
Esc = 結束遊戲

說明：

1.有降落傘者，為空投物資，可加以射擊變更物資形態，最後一種為增強武器
  火力。若武器火力已是最強（主武器兩條紅線），則再增強會重置回最弱的
  火力（初始值）。
2.武器火力提升到一定程度以後，按著子彈發射鍵才能持續發射子彈，否則只
  能按一次發射一次。
