名稱：幻象雷電 (Mirage Thunder，中文版)
簡介：大宇1993年出品的射擊遊戲，共有８關。故事描寫一船艦被不明物體攻擊，所有人被
      俘虜而去，而奉命救援的幻象雷電乃追擊而去，終於攻破對方堡壘。遊戲中的己艦被
      擊中時，會先損失武器火力值，幸好火力值的補充還蠻多的。此外，遊戲中也有提供
      ３種不同的武器，只是到最後幾關敵人又多又強，要破關可非易事。整體而言，這款
      遊戲的娛樂效果還不錯，值得玩家一玩。
 
備註：本版含α版內容。將子目錄alpha檔案拷貝到主目錄，即成α版，難度大幅降低，並
      新增武器強力爆彈。
限制：DosBox必須0.74版以上才能玩

密技：先按Enter暫停，再輸入I LOVE CAT，則武器火力值會立即補滿，但每關只能使用一次。
操作：

F2 = 離開遊戲
F9 = 音樂開關
F10 = 音效開關
Enter = 暫停／繼續遊戲

其餘操作按鍵可在遊戲裡設定，內定鍵為：

Alt = 發射子彈
Ctrl = 旋轉掛載武器
左Shift = 移動速度快(H)／慢(L)切換，顯示在右邊Power條圖右側
          α版為發射爆彈，並暫時無敵數秒

說明：

1.子彈發射鍵按住不放可持續發射。當不發射子彈時，可累積能量（顯示在下方能量棒），
  當按下子彈發射鍵時，即可釋放累積的能量，發出高攻擊力的能量彈。累積的能量愈多，
  攻擊力愈高（滿值為34h=52）。
2.主武器有3種，各有5段火力（0～4，初值為1），必須吃相同顏色的才會增加（由右邊Power
  條圖可得知）：
  (1)紅：電射，前方直射，能量彈=雷射巨砲。
  (2)藍：冰雷，前方散射，能量彈=自動追蹤巨冰雷。
  (3)黃：爆彈，四方散射，能量彈=繞射巨爆彈。
3.副武器有3種：
  (1)M：飛彈，攻擊力高
  (2)H：追蹤飛彈，攻擊力弱
  (3)S：側擊彈，攻擊力中
4.主武器或副武器不立即拿取的話，過一段時間會自動變換武器種類。
5.若覺得敵機難打，可到主畫面Option Mode裡設定難易度為Easy。
6.難易度為Hard時，破關才會有動畫結局。
