名稱：快打至尊 (Super Fighter，中文版)
簡介：全崴1993年出品的打鬥遊戲，為早期一個相當不錯的打鬥遊戲。遊戲中可選擇的人物
      共有８位，每位人物各有其絕技，而玩家必須經過１１場戰役爭取總冠軍。除了挑戰
      電腦控制的選手外，玩家也可以選擇對戰模式和朋友進行對打。整個遊戲的娛樂效果
      不錯，值得一玩。
 
備註：本版為1.1版，並修正計時速度隨CPU速度而變的問題。
保護：數字密碼，破解碼：
      SF.EXE
        CD 10 C3 B8 00 00  (不會出現輸入密碼)
        -- -- -- C3 -- --
        74 0D BE 02 00
        EB -- -- -- --
      或
        75 03 E8 05  (隨便輸入密碼皆正確)
        EB -- -- --
        75 03 E8 27
        EB 0C -- --
備註：1.正常版請執行PLAY.BAT，簡易版請執行EPLAY.BAT。
        簡易版各人物的絕招按法，所有斜移按鍵（↖↗↙↘）都不必再按，便可發出絕招。
      2.將CHEAT目錄下的資料拷到主目錄下，則電腦控制的對手將不會主動攻擊（但近身時
        仍會攻擊）。11MAN目錄則為可選11人的1.1版主程式。

資訊：特殊技按鍵表

SFMAN??.DAT位置0600h開始，每組12 byte：

ID -- PS K1 K2 K3 K4 K5 K6 K7 K8 --

ID = 招式編號，用來處理動畫與傷害
PS = 位置
  80h = 近身
  20h = 空中
K1～K8：按鍵組合，最後一鍵在最前
  80h = 任一位元on即可
  10h = 手
  20h = 腳
  01h = 右
  02h = 左
  04h = 下
  08h = 上

說明：

1.單打：與電腦對戰，打敗全部8位選手。若困難度等級為〔強〕，則再跟三大魔頭對打。
2.雙打：雙人對戰。
3.設定：設定操控方式與困難度等級（單打）。
4.離開：結束遊戲。

*** 特殊技

1.標示〔〕者，表示括弧中任選一種；標示…者，為持續按著一小段時間（或連按三次）。
  蹲=〔↙↓↘〕，跳=〔↖↑↗〕，退=〔←↖↙〕，進=〔↘↗→〕，移=〔進退〕，任移=
  任一方向鍵。
2.各按鍵組合均以我方在左方的情況為準，若在右方，則左右按鍵必須對調。
3.某些特殊技是由多個動作所組成，若敵方位置剛好未被某個動作擊中，則該細項傷害將無
  作用（各細項傷害詳列在括號中）。
4.防禦可大幅抵消傷害，一般攻擊基本上都無法造成傷害，但摔技傷害是無法抵消的。
5.雙方生命初值7Ch=124，短時間累積傷害超過40，會進入暈眩狀態，暫時無法行動。
6.生命會隨著時間緩慢恢復（約數秒1點）。
