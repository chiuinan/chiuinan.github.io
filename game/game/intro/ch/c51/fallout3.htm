      推出「年度版（Game of the Year Edition）」 [待補]。



備註：本版為年度版
保護：光碟SecuROM 7.38，可使用SRLoader啟動
備註：5個DLC分別為：

1.安克雷奇行動（The Operation: Anchorage）
2.匹茲堡（The Pitt）
3.斷鋼（Broken Steel）
4.眺望角（Point Lookout）
5.母艦澤塔（Mothership Zeta）

備註：VirtualBox無法執行，虛擬機請使用VMware。
備註：遊戲需要.NET FrameWork 3.5、Microsoft Game for Windows Live、Mircrosoft Games
      for Windows Marketplace、Windows Live ID Sign-in Assistant等多個套件。
備註：年度版安裝完DLC後，必須更新至較新版的xlive，否則會出現5360錯誤。
備註：執行過一次FalloutLauncher.exe後，即可直接執行Fallout3.exe（無保護、免光碟）。
備註：簡中漢化過程如下：

1.安裝時選擇Custom，修改安裝目錄為Fallout 3 goty（重要！簡中漢化時以此目錄名為準）。
2.安裝主程式光碟與DLC光碟。
3.安裝新版xlive（Microsoft Game for Windows Live）。
4.更新至1.7.0.3版。
5.先備份FalloutLauncher.exe，再安裝1.7.0.3簡中漢化檔，目錄選擇Bethesda Softworks。
  如果選擇的目錄底下沒有"Fallout 3 goty"這個子目錄的話，是無法進行安裝的。
6.將新的FalloutLauncher.exe更名為FalloutLauncherCN.exe。
7.還原備份的FalloutLauncher.exe（或拷貝免光碟檔），執行後選擇DATA FILES，勾選所有
  的.esm檔後離開。
8.修改"我的文件\My Games\Fallout3\FALLOUT.INI"，將[Fonts]以下改成：

[Fonts]
sFontFile_1=Textures\Fonts\GB_Glow_Monofonto_Medium_Common.fnt
sFontFile_2=Textures\Fonts\GB_Glow_Monofonto_Medium_Common.fnt
sFontFile_3=Textures\Fonts\GB_Glow_Monofonto_Medium_Common.fnt
sFontFile_4=Textures\Fonts\GB_Glow_Monofonto_Medium_Common.fnt
sFontFile_5=Textures\Fonts\GB_FixedsysOrigion_Hei18_low.fnt
sFontFile_6=Textures\Fonts\GB_Glow_Monofonto_Medium_Common.fnt
sFontFile_7=Textures\Fonts\GB_Glow_Monofonto_Medium_Common.fnt
sFontFile_8=Textures\Fonts\GB_Glow_Monofonto_Medium_Common.fnt

9.此後一律以FalloutLauncherCN.exe啟動遊戲（不可直接執行Fallout3.exe）。
10.新遊戲若會崩潰當機，請使用修正檔。
