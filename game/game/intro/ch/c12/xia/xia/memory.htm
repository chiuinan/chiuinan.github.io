-- -- -- -- XX YY -- -- -- -- AA AA BB BB -- --

   AA = 目前體力，滿值200（C8h）
   BB = 目前內力，滿值200（C8h）
   XX,YY = 所在地圖位置（改變後可瞬移）

   EE = 招式總攻擊力（即一招可發出的所有攻擊波／道的攻擊力總和）
   FF = 招式防禦力
