名稱：地城領主／龍與地下城：地下城主 (Dungeon Lords，中文／英文版)
簡介：Heuristic Park 2005年開發的角色扮演遊戲，由英特衛代理發行，採3D畫面呈現，
      以單人即時方式戰鬥。由於開發商為了趕上產品發佈日期，造成原始版本錯誤百出，
      使得本遊戲收到眾多差評。直到2006年，才重新發佈較為完整的「收藏版（Collector's
      Edition）」。2012年由Nordic推出重製版「MMXII」，2015年經由Steam推出Steam版
      [待補]。



備註：本版為原始版繁中版，已含1.4版更新檔內容。
保護：無
備註：英文收藏版版本為C1.5版（沒有中文版）。
操作：

W/上鍵 = 前進
S/下鍵 = 後退
AD/左右鍵 = 向左/向右
Space = 跳躍
Shift = 動作(如交談、開箱、撿取等)
滑鼠左鍵 = 攻擊/施法
  前進+滑鼠左鍵 = 刺擊 (傷害低但攻距長)
  後退+滑鼠左鍵 = 過頭砍擊 (常用來對付飛行怪物)
  左右+滑鼠左鍵 = 揮砍
  跳躍+滑鼠左鍵 = 跳躍砍擊 (提高命中率與傷害)
滑鼠右鍵 = 用盾牌格擋(如果有裝備的話)
F = 快速法術/物品選單
1～8 = 使用快速法術/物品
Q = 切換近戰/遠距武器(須先裝備好)
C = 狀態欄
E = 物品欄，再按一次離開
L = 任務欄
M = 小地圖開關
V = 設定滑鼠靈敏度
Z = 鎖定攝影機Y軸開關
PgUp = 向上看
PgDn = 向下看
Ctrl+上下左右鍵 = 轉動攝影機
滑鼠滾軸/Del = 攝影機拉近
滑鼠滾軸/End = 攝影機拉遠
R = 復活死亡的角色 (若沒有復活物品，人物會失去部份屬性)
Esc = 系統選單
/ = 幫助，再按一次離開
