A1～A7：目前屬性值（含裝備與魔法作用）
B1～B7：實際屬性值，依次為力量、力量18的比例值、智能、學問、敏捷、體
        質、魅力
C1～C2：目前生命／最大生命，最大不要超過900，否則休息時可能不會治療
DD：AC值，變更裝備時會重算
E1：種族，偶數為男性，奇數為女性
  00～01=人類
  02～03=精靈
  04～05=半精靈
  06～07=矮人
  08～09=侏儒
  0A～0B=半身人
E2：職業
  00 = 戰士
  01 = 遊俠
  02 = 武士
  03 = 巫師
  04 = 牧師
  05 = 小偷
  06 = 戰士／牧師
  07 = 戰士／小偷
  08 = 戰士／巫師
  09 = 戰士／巫師／小偷
  0A = 小偷／巫師
  0B = 牧師／小偷
  0C = 戰士／牧師／巫師
  0D = 遊俠／牧師
  0E = 牧師／巫師
E3：陣營
E4：相片
FF：食物量，最大64h
G1～G3：各職業等級，最大0Dh
H1～H3：各職業經驗值
KK：已記憶或祈求的法術編號，每個1 byte，共160個
LL：巫師已抄寫的法術，每個位元表示一個法術，填4個FFh即可有全部法術
M1～M2：主手、副手持有的物品索引值
MM：物品欄的物品索引值，共14組，每組2 byte，指向後面的物品資料
NN：身上的物品索引值，共11組，依次為箭囊第一枝箭、護甲、護腕帶、頭盔、
    項鍊、鞋子、腰帶x3、戒指x2
SS：狀態值
  01 = 固定設定
  02 = 身染劇毒
  04 = 全身麻痺
  08 = 變成化石
ST：不良狀態恢復正常尚需的時間
R1：個人法術作用旗標
  00000001h = 魔冑術
  00000008h = 魔盾術
  00000010h = 朦朧化身
  00000040h = 隱形術
  00000100h = 強力隱形
  00004000h = 魔法之袍
  00008000h = 百邪不侵
  00010000h = 迅捷術
R2：全隊法術作用旗標
  00000002h = 偵測魔法
  00000200h = 透視術
  00000400h = 祝福術
  00008000h = 祈禱術
  00020000h = 龍氣護罩
GF = 全隊旗標
  8000H = 隱月記號

物品資料區（索引值最大為599=257h）：

記錄遊戲中所有物品的資料，修改前記得先備份，否則萬一不小心破壞了過關
的必要物品，最後會造成遊戲卡關。

開始位置 = 894h（對應索引值=1，索引值=0表無物品）
物品索引值對應的物品資料位置 = 索引值*14 + 2182
物品資料格式（每個14 byte）：

  A1 A2 BB CC DD EE FF FF NN NN PP PP LL GG

  A1 = 名稱編號（不影響）
  A2 = 辨識後名稱編號，01表標準處理
  BB = 物品狀態
         80h = 1表示魔法物品（偵測魔法會藍色顯示）
         40h = 1表示已辨識（名稱後會顯示魔法力）
         20h = 1表示詛咒物品（無法從手上拿下）
         1Fh = 魔法剩餘次數（最多31次）
  CC = 圖片編號（不影響）
  DD = 物品型態（重要）
         00 = 斧
         01 = 長劍
         02 = 短劍
         05 = 匕首
         06 = 頭骨
         07 = 弓
         08 = 大腿骨
         09 = 矛
         0A = 戟
         0B = 釘頭鎚
         0C = 鏈枷
         0D = 杖
         0E = 投石索
         0F = 鏢
         10 = 箭
         11 = 號角
         12 = 石塊
         13 = 板條甲
         14 = 鎖子甲
         15 = 頭盔（無魔法防護力）
         16 = 皮甲
         17 = 三叉戟
         18 = 鎧甲
         19 = 鱗甲
         1A = 特殊施法物品，GG：法術編號（0D=火燄球，1C=魔粉）
         1B = 盾
         1C = 開鎖工具
         1D = 魔法書
         1E = 牧師法器
         1F = 食物，GG：食物量
         20 = 鞋子（無魔法防護力）
         21 = 遺骨，GG：人物編號，00或63=通用骨骸
         22 = 巫師卷軸，GG：法術編號
         23 = 牧師卷軸，GG：法術編號
         24 = 記事文件，GG：訊息編號
         25 = 石器，GG：石器編號
         26 = 鑰匙，GG：鑰匙編號
         27 = 藥水，GG：藥水編號
         28 = 寶石，GG：寶石編號
         29 = 長袍
         2A = 保護戒指
         2B = 護腕帶
         2C = 裝飾用項鍊
         2D = 雙手劍
         2E = 一般物品，GG=物品編號
         2F = 法術戒指，GG：01=巫師，02=止飢，03=緩降，0A=紅色的環
         30 = 魔杖（只限巫師使用），GG：魔杖法術編號
         39 = 梅杜莎護盾
         3B = 鍾納斯之袍
         3C = 水晶鎚
         3D = 特殊護符，GG：00=生命護符，01=死亡護符
         3E = 魔杖（所有職業均可使用），GG：魔杖法術編號
         3F = 貼紙
  EE = 物品在格內的位置
  FF = 物品所在位置座標，FFFFh=物品已不存在（消耗掉），FFFEh=箭囊
  NN = 同一格內，下個物品的索引值
  PP = 同一格內，上個物品的索引值
  LL = 物品所在層，00=人物身上，FF=箭囊
  GG = 作用值（重要）
         武器 = 魔法傷害力
         保護防具 = 魔法防護力
         其餘參見物品型態中說明，無標示者固定為00

簡易的消耗性物品複製法（包括鑰匙）：

  消耗前先存檔備份起來，消耗後存檔，找出物品資料不同處（可利用前述索
  引值對應位置公式計算出實際位置），將變成FF FF的地方改回00 00，並修
  改回身上應有的物品索引值

注意事項：

不可填入非已撿取的物品索引值來獲得物品，最好是直接修改已撿取且沒有用
的物品資料，成為自己想要的物品。
