資訊修改
一. 物品形態定義檔（ITEMTYPE.DAT）

檔案格式：
  物品形態數目：2 byte
  物品形態定義：16 byte

物品形態定義格式：

  AA AA BB BB CC DD EE F1 F2 F3 G1 G2 G3 00 HH 00

  AA = 允許置放的位置
    100h = 戒指
    080h = 腰帶
    040h = 項鍊，後面的職業限制與AC修正不作用
    020h = 頭盔，後面的職業限制與AC修正不作用
    010h = 鞋子，後面的職業限制與AC修正不作用
    008h = 手
    004h = 護腕
    002h = 護甲
    001h = 箭囊
  BB = 物品作用的位置，格式同上
  CC = AC修正值，有正負
  DD = 允許使用的職業
    20h = 遊俠
    10h = 武士
    08h = 盜賊
    04h = 牧師
    02h = 巫師
    01h = 戰士
  EE = 允許使用的手，00=副手，01=主手，02=雙手
  F1～F3 = 對中小型怪的傷害力，擲骰格式：F1*(1..F2)+F3
  G1～G3 = 對大型怪的傷害力，擲骰格式：G1*(1..G2)+G3
  HH = 物品類型
    00h = 防禦裝備
    01h = 近戰武器
    02h = 投擲武器
    03h = 長程武器
    04h = 一般物品
    05h = 魔法書
    06h = 牧師法器
    07h = 食物
    08h = 遺骨
    09h = 巫師法術施法物品
    0Ah = 牧師法術施法物品
    0Bh = 記事文件
    0Ch = 石器
    0Dh = 鑰匙
    0Eh = 藥水
    0Fh = 寶石
    10h = 法術戒指
    12h = 魔杖
    13h = 號角
    14h = 特殊護符
    80h = 是否可被怪物破壞

怪物資料格式：

  AC TH LV HP HP HP AN A1 A1 A1 A2 A2 A2 A3 A3 A3 ?? ?? AB AB ?? ?? EX EX SZ S1 S2 ?? FA ... TU ?? ?? ?? ??

  AC = 防禦力
  TH = 命中率（THAC0）
  LV = 等級
  HP = 生命力，擲骰格式，亦即A*(1..B)+C
  AN = 攻擊次數
  A1～A3 = 各攻擊次數的傷害力，擲骰格式
  AB = 能力旗標
         0010h = 中毒
         0020h = 麻痺
         2000h = 爆炸
         4000h = 全體攻擊
         8000h = 石化
  EX = 經驗值
  SZ = 大小，0=中（人型），1=小，2=大
  S1～S2 = 攻擊／移動時用的聲音
  FA = 是否有遠程攻擊能力，00=有，FF=無，當有時才有下列資料：
       不明：1 byte
       遠程攻擊數目：1 byte
       遠程攻擊方式：每個2 byte
         FFFC = 投擲匕首
         0000 = 魔法飛彈
         0002 = 火球術（強）
         0003 = 定身術
         0004 = 雷電術
         0007 = 火球術（中）
         0008 = 火球術（超強）
         0009 = 火球術（弱）
         0014 = 心靈攻擊
  TU = 是否可超渡，FF=不可
