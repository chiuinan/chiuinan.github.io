名稱：殺氣沖天 (中文版)
簡介：北京八爪魚2000年出品的角色扮演遊戲，台灣由大宇代理發行，以3D斜角畫面呈現，
      採單人即時方式戰鬥 [待補]。



保護：光碟
檔案：patch.rar = 雙人連線問題更新檔
密技：在遊戲中按[Enter]鍵，然後輸入下列字串可得到相應的效果：

mylife = 加生命
mypower = 加功力
friend = 降低敵意至0
getgold = 得到金錢1000
getgoods = 得到物品，遊戲會要求繼續輸入物品編號1-98
           （1-30武器；31-53藥品；57-64秘籍；65-77鎧甲；78-82面具；83-98暗器）
gotoscene = 顯示總地圖，點中哪裡去哪裡（切勿點中未開發好的地方）

使用秘技不當可能會使情節發展或數字方面出現錯誤！

操作：

Shift = 加快遊戲進行速度
M = 地圖
F3 = 抓圖
