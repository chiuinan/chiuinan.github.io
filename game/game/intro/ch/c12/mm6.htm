名稱：魔法門６奉天承運／天堂之令
      (Might and Magic 6 - The Mandate of Heaven，繁中／簡中／英文版)
簡介：New World Computing 1998年出品的角色扮演遊戲，以3D畫面呈現，採
      即時方式戰鬥（可設定為回合制）。魔法門系列從第三代開始大受玩家
      喜愛，從本代開始，採用自由度更高的仿3D畫面，技能也更加予以細分，
      風格雖變，但仍極受歡迎。角色畫像改以真人代替，不同情況有不同表
      情，相當有趣，只是人物稍嫌醜了些，若是俊男美女，應該更受歡迎。
      劇情跟前幾代一樣，都是四處冒險，完成任務，因此比較難以從中明瞭，
      不過這並不影響本遊戲的娛樂性。相較於前幾代而言，本代難度變高了
      許多，但還不至於過份變態，頗值得角色扮演遊戲的喜好者一玩。



保護：光碟，在安裝目錄下建立一個mm6.ini檔，並輸入下列內容，即可免光碟
      （但會無背景音樂）：

[settings]
use_cd=0

限制：中文版必須使用相同語系之視窗系統，否則會無法輸入正確的中文字
      （但不影響過關與所有任務的完成）。
備註：本版為繁中版，已更新patch檔，並修正人物狀態欄裡的字詞錯誤、替換
      成中文語音。
檔案：mm6-s.rar = 簡中版（已修正人物狀態欄裡的字詞錯誤，中文密碼均改回英文）
      tpatch.rar = 繁中版Patch檔（修正阿拉莫斯城密碼無法輸入等錯誤）
      MM6patch.rar = GrayFace MM6 Patch v2.0
      MMArchive.rar = LOD檔編輯程式
      LodCompare.rar = LOD檔差異比對抽取器
      LodMerge.rar = LOD檔合併程式
      MM8LevelEditor.rar = MM6-8資料通用編輯器3.47版（需java環境）
      MM6EDT36.ace = 人物屬性／物品編輯器（安裝檔）
      MM6Editor.7z = 人物屬性／物品編輯器（執行檔）
      mm6che_1.20_en.rar = 人物屬性／物品編輯器
      bestnew.7z = 技能稍微增強之初始檔（簡中版）
      bestchar.7z = 全部最強之初始檔（簡中版）
      bestall.7z = 帶有超強裝備與雜項技能之初始檔（簡中版）
      t-Icons.rar/s-Icons.rar = 繁／簡中版裝備增強檔，覆蓋原檔即可
        （性能提升3～6倍，特別是神器與遠程武器）
      t-Icons2.rar/s-Icons2.rar = 繁／簡中版裝備超強檔
        （性能提升10～20倍，魔法屬性值可到255，榴彈槍最強）
修改：利用MM7View或MMArchive修改Icons.lod裡的ITEMS.TXT，可造出超強之
      武器。其他如MONSTER.txt可調整怪物屬性，MapStats.txt可調整各地
      圖的重置週期，STDITEMS.txt可調整魔法物品增加屬性的範圍。

攻略：請參閱mm6\index.htm（簡中／繁中／英文版）。
