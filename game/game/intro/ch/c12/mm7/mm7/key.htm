操作按鍵
Space = 對目標物執行動作，如進入、交談、撿取等 (尤其滑鼠點擊無效時)
Shift+上／左右／下鍵 = 跑步／快速旋轉／快速後退
Ctrl+左右 = 側移
Ctrl+滑鼠左鍵或空白鍵 = 偷竊
X = 跳躍
Enter = 即時／回合制戰鬥切換
PgDn = 往上看
Del = 往下看
End = 平視
PgUp = 往上飛升
Home = 往下掉落
Tab = 切換人物
A = 近距攻擊／射箭（自動選擇目標）
S = 施展已設定法術／射箭（自動選擇目標）
滑鼠左鍵 = 選擇目標進行近距攻擊／射箭
Ctrl+滑鼠左鍵 = 選擇目標進行施法攻擊
C = 施法
Q = 任務
T = 時間
M = 地圖
Z = 狀態
N = 記事
R = 休息
-/+ = 縮放簡易地圖
F4 = 視窗模式/全螢幕模式切換（螢幕顯示必須設定為16 bit高彩模式）
B = 回合制戰鬥中略過該人物
Y = 喊叫
Esc = 主選單
