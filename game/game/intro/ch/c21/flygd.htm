名稱：飛翔傳說：武者之旅 (中文版)
簡介：琪薇1993年出品的劇情冒險遊戲，同時結合了格鬥遊戲的元素，由精訊代理發行 [待補]。



備註：本版為更新版（舊版名字為春麗，新版名字為麗子）
保護：圖案密碼，破解碼：
      GTITO.PBC
        3B 06 AA 0E 75 03
        A3 AA 0E 90 90 90
      或
        75 03 E9 52
        -- 00 -- --
備註：安裝時若更改音效卡設定，會直接寫入到磁片中，可使用預設值安裝後，再到安裝目
      錄更改設定。
備註：某些硬碟版因缺少MPIC\M1.DAT等多個檔案，會導致新遊戲開頭動畫結束後即跳出。
檔案：flygd-old.rar = 原始版
操作：

Esc = 移動／取消（標題畫面按Esc鍵可進入點唱機）
Space = 確定
上 = 查看
左 = 物品
右 = 攻擊
下 = 更換服裝
F1 = 儲存遊戲
F2 = 讀取遊戲
F3 = 音樂開關
F4/Q = 離開遊戲
