名稱：恐龍危機2 (Dino Crisis 2，中文／英文版)
簡介：CAPCOM 2000年出品的動作冒險解謎遊戲，由大新資訊中文化並委由第三波代理發行 [待補]。



備註：本版繁中光碟有壞軌，不確定是否造成壞檔（待查）
保護：英文版為光碟，繁中版為光碟SafeDisc 2.51，破解碼（必須完全安裝）：
      Dino2.exe
        56 57 6A 01 68  (英文版)
        -- -- -- 00 --
備註：VirtualBox需搭配WineD3D，或虛擬機改用VMware。
檔案：nocd_cht.rar = 繁中免光碟檔
      dino2_chs.rar = 簡中版
操作：

上下鍵 = 前進後退
左右鍵 = 左右轉
D = 180度轉身
X = 瞄準（射擊前要先按）
C或Space = 決定／調查／主武器攻擊
V = 取消／副武器攻擊
W = 地圖
Z = 道具選單
A = 暫停／選項選單
Alt+F4 = 結束遊戲
