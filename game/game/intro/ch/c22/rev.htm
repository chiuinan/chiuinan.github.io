名稱：第六感之戀 (Reverse，中文版)
簡介：Gloria 1997年出品的１８禁故事閱讀遊戲，由歡樂盒中文化並代理發
      行。劇情描寫牧原拓馬車禍死亡，靈魂共存在姐姐卓美身上，之後卓美
      進入教職，利用獲得的入夢力解決一眾學生的問題。整個劇情編得還算
      不錯，採用多結局方式，由遊戲中的選項來決定最終的結局，只是結局
      數目嫌少了些。圖片部份也還算不錯，因此整體而言，算是可以玩的一
      個遊戲。

資訊：REV?.GTR檔案格式

1.檔頭

4E 53 58 00 NN NN PP PP PP PP -- -- -- -- -- --

NN = 指示資訊數目
PP = 指示資訊位置

2.指示資訊：每個26=1Ah byte

20 -- -- -- -- SS SS SS SS NN NN NN NN NN NN NN
NN NN NN NN NN 00 PP PP PP PP

SS = 資料大小
NN = 資料名稱
PP = 資料位置，內容依資料形態而異，有可能是壓縮資料

3.壓縮資料

解壓後大小（4 byte）
壓縮資料（SS-4 byte）

解壓演算法如下：

buffer = 1000h byte空間，並清為0
index = 0x0FEE
while 還有壓縮資料 do
  a = 讀取1 byte
  for i=1～8 do
    if 已經沒有壓縮資料 then exit
    if a的位元i為1 then // 由低位元開始檢驗起
       b = 讀取1 byte
       輸出b
       buffer[index] = b
       index = (index + 1) & 0x0FFF
    else
       b1,b2 = 讀取2 byte
       pos = ((b2 & 0xF0) << 4) | b1
       no = (b2 & 0x0F) + 3
       for j=0～no-1 do
         b = buffer[(j+pos)&0x0FFF]
         輸出b
         buffer[index] = b
         index = (index + 1) & 0x0FFF
       end for j
    end if
  end for i
end while

備註：CG圖有露點。
操作：

CapsLock = 文字段落完是否等待按鍵的開關（關閉後，可按著滑鼠左鍵加快換段）
Esc = 系統選單（等待按鍵時）
Space = 暫時關閉文字顯示框（看全圖）
Enter/滑鼠左鍵 = 換下段文字

遊戲對於CapsLock作用與否，是使用記憶體40h:17h的40h位元（DOS的CapsLock
狀態），基本上不會有按了無效的問題。如果發現在DosBox按了無效，可能是
鍵盤輸入的作用視窗不是DosBox，用滑鼠點擊DosBox視窗內部後再按一次試試。
