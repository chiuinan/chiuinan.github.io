新手提示
平時按鍵：

滑鼠左鍵、上下左右鍵 = 行走（雙擊為奔跑）
滑鼠右鍵 = 跳躍（僅限特定地點）
滑鼠滾輪、A/D = 旋轉視角（僅限外景）
Tab = 更換控制人物

戰鬥按鍵：

F5 = 關掉戰鬥AI
F = 本回合全體人物攻擊
R = 重覆上一次操作
P = 暫停戰鬥

圖像操作：



點選右上頭像，可進入遊戲功能界面
上方紅色為目前體力（精），下方紫色為目前法力（神）




點選左下圖案中心，可出現小地圖
