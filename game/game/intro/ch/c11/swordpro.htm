名稱：靈劍傳奇 (中文版)
簡介：泰騰1998年出品的角色扮演遊戲，由華義代理發行，以3D斜角畫面呈現，團隊回合制
      方式戰鬥 [待補]。
 
備註：本版為DOS/Windows二合一版，DOS/Win31或Win95/XP未放光碟片者請執行PLAY.BAT，
      Win95/XP有放光碟片者請執行SWORD.BAT。
保護：光碟。第一次成功執行CDPLAY.EXE產生CDPLAY.DAT，之後不再執行CDPLAY.EXE，即可
      免光碟。CDPLAY.DAT亦可執行其他遊戲的CDINFO.EXE，再將CDINFO.DAT更名過來亦可。
備註：必須在Win9x系統上安裝，但其實只要拷貝光碟根目錄所有檔案，再執行SETSOUND.EXE
      設定音效卡即可。
備註：本遊戲主要分有兩個程式：

1.CDPLAY.EXE：32位元音軌播放程式，必須在Win95以上系統執行。沒有它便沒有背景音樂。
2.SWORD.EXE：16位元遊戲程式，必須在XP以下系統執行。必須先有正確的CDPLAY.DAT。

因此本遊戲若在DOS/Win31上執行，因為沒有CDPLAY.EXE，所以沒有背景音樂。必須在Win9x/XP
上執行，才能有背景音樂。不過原版SWORD.BAT在XP上無法正確執行，因為CDPLAY.EXE在多重
執行緒的XP會在批次檔裡卡住，無法常駐，必須改以下列命令啟動：

START CDPLAY.EXE

或是分別點擊CDPLAY.EXE與SWORD.EXE來啟動遊戲。主程式SWORD.EXE在結束時，會自動關閉
常駐的CDPLAY.EXE程式。
