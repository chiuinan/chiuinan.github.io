資訊修改
以下為軒轅劍4各遊戲資訊檔的作用：

ObjData.GLL    名稱資訊檔
LEVEL.GLL      升級資訊檔


*** 名稱資訊檔 ObjData.GLL

檔頭標記：16 byte，固定"GLL022"，其餘補00h
名稱資訊位置：每個名稱4 byte，按名稱代碼排列。位置=名稱資訊位置[名稱代碼-1]+16
名稱資訊：多個{特徵碼+特徵值}

名稱代碼範圍：
  敵人：0001h～048Dh
  物品：0BB9h～103Eh
  奇術：1389h～1768h
  天書：26ADh～26C9h


*** 升級資訊檔 LEVEL.GLL

檔頭標記：16 byte，固定"GLL022"，其餘補00h
升級資訊位置：每個主角4 byte（次序：水鏡、姬良、疾鵬），位置=升級資訊位置+16
升級資訊：1～60級升級資訊，每組24 byte

　LV LV -- -- -- -- -- -- -- -- -- -- -- -- -- --
  -- -- SS SS EX EX EX EX

LV = 等級
SS = 學到奇術代碼
EX = 下一次升級所需總經驗值


*** 特徵碼一覽表

03F2h + 速度（1 byte整數）
03E9h + 法寶所需經驗（4 byte整數）
03FCh + 內部代碼（2 byte整數）
0406h + 奇術代碼（5x2 byte整數），0000h為無
0410h + 奇術型態（5x1 byte整數）
041Ah + 可學奇術代碼／法寶類型（2 byte整數），0000h為無
044Ch + 掉落可偷物品（1 byte掉落機率+2 byte掉落物品代碼+1 byte偷竊機率+2 byte偷竊物品代碼）
044Dh + 獲得資源（5x2 byte整數：金錢、金屬、木材、煤炭、經驗值）
04B0h + 2 byte整數
050Bh + 2 byte整數
0514h + 1 byte
05DCh + 天書能力（5 byte：冶金、整備、製藥、神技、00h）
07D1h + 價格（5 byte，不明1 byte+2 byte價格+不明2 byte固定0）
07D2h + 3 byte
07D3h + 裝備屬性增減（5x2 byte整數：攻擊、防禦、速度、仙攻、仙防）
07D4h + 消耗體靈（不明2 byte+體力2 byte+靈2 byte+絕技1 byte）
07D5h + 作用旗標（4x1 byte整數）
        旗標1：作用對象，0=敵己雙方，1=己方，2=敵方
        旗標2：作用數目，0=單人，1=全體，2=自己
        旗標3：不明
        旗標4：不明
07D6h + 傷害（2 byte整數）
07D7h + 狀態變更（1 byte）
        01h = 中毒
        02h = 麻痺
        04h = 昏睡
        08h = 衰弱
07D8h + 狀態變更（1 byte）
        01h = 死亡
        02h = 封魔
07D9h + 屬性增減（5x2 byte整數：命、體、攻、防、0），超過10000表百分比，例如10030為30%
07DAh + 屬性增減（6x2 byte整數：靈、仙攻、仙防、0、速、不明），超過10000表百分比，例如10030為30%
07DBh + 抗性／攻擊屬性（2x2 byte整數：雷水、火風）
07DCh + 1 byte
07DDh + 祭風雲符（2x1 byte：形態、等級）
        00 = 天地符
        01 = 祭雲符
        02 = 祭風符
07DFh + 1 byte
07E3h + 特殊狀態（狀態碼1 byte+持續時間2 byte整數）
        01 = 興奮
        03 = 暴怒
        04 = 哀傷
07E4H + 仙術所需經驗（2 byte正整數）
0803h + 可裝備者（1 byte）
        01h = 水鏡
        02h = 姬良
        08h = 疾鵬
0834h + 生產所需資源（4x2 byte整數：金、木、煤、錢）
0835h + 生產所需物品代碼（5x2 byte整數，0000h表無）
0837h + 生產所需技能（4x1 byte整數：冶金、整備、製藥、神技）
0898h + 1 byte
09C4h + 2 byte
26B1h + FFFFh：資訊結束
26B2h + 名稱字串（以%Q做為結束）
26B3h + 等級（2 byte整數），用來計算怪物屬性值
26B4h + 8x2 byte
26B5h + 類型ID（2 byte整數）
        101 = 主角
        110 = 人類
        111 = 動物
        112 = 神獸
        113 = 鬼
        114 = 精怪
        115 = 妖靈
        116 = 魔獸
        117 = 神祉
        118 = 機關
        130 = 魔王
        201 = 一般物品
        202 = 武器
        203 = 護具
        204 = 裝飾
        205 = 劇情物品
        206 = 法寶
        300 = 武術
        303 = 陰陽術
        304 = 機關術
        305 = 絕技
        400 = 一般建築
        401 = 生產建築
        402 = 仙術道場
        403 = 法寶建築
        404 = 鬼神陣
26B6h + 說明字串（以%Q做為結束）
26B7h + 功能字串（以%Q做為結束）
26B8h + 資料類型（2 byte整數）
        1 = 敵人
        2 = 物品
        3 = 奇術
        4 = 天書
