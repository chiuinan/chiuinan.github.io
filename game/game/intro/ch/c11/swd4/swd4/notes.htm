提示：

1.本遊戲的商店都買不到好東西，必須多藉重天書來造出，大抵需要四種資源：
  金錢、金屬、木材、煤炭。木材和煤炭未到快破關前是買不到的，但因某些
  怪物會掉，而且四處可撿得（閃光處），因此大多不會匱乏。金屬是四種資
  源裡最常缺乏的，雖然買得到，但都很貴，賺錢也不易，因此儘可能不要太
  浪費。
2.打敗敵人後，每個隊員都獲得與一個人打時相同的經驗值，不以隊員數進行
  平均。隊員身上若有裝備法寶時，升級經驗值會撥出10%以做為法寶的經驗值
  （升級經驗值變成原來的90%）。姬良若有學習法術，則以原升級經驗值10%
  做為學習法術的經驗值（不從升級經驗值中扣除）。
3.人物最多為60級，所有超過60級以上的怪都能收化（魔王除外）。收化的機
  率視怪物與姬良的等級差，以及怪物的損血率而定。低於姬良10級以上的怪
  不用損血百分之百可成功收化。怪物等級愈高，收化機率愈低，損血率愈高，
  收化機率愈高。
4.屬性作用如下：
  攻擊力：決定水鏡、疾鵬的攻擊傷害（含一般攻擊、奇術與絕技）
  防禦力：減少被敵人一般攻擊時的傷害
  速度：決定每次行動的間隔時間
  仙攻：決定姬良的攻擊傷害（含一般攻擊與奇術）
  仙防：減少被敵人奇術攻擊時的傷害
  仙命：無作用
5.增加能力的物品，加仙攻與靈力的都給姬良，其餘的都給水鏡。但由於升級
  後所有屬性值都會重置，因此增加能力的物品最好都保留到60級以後再使用
  （升級用的雪肌冰鮑除外），否則一升級，所增加的能力值便會消失不見。
6.防禦時會回復命靈體3～5點，且受到的傷害減半。
7.賣出物品的價格為原價的50%。
8.情緒對角色的影響
  興奮／亢奮：無
  暴怒：攻+60%，防-80%，速+60%
  瘋狂：攻+100%，防+100%，速+100%，自動一般攻擊
  哀傷：仙攻-50%，仙防-50%
  衰弱：攻-80%，防-80%，仙攻-80%，仙防-80%
9.值得使用的法寶：
  水鏡、疾鵬：
    寄鬼蟲：防禦增加絕技值。
    奇魄古玉：聚勁增加攻擊力（用高攻擊力裝備，再配合健身咒、天仙酒）。
    如意鬼手：用來偷魔王身上的好東西，普通怪沒好東西。
    某水：奇術攻擊兩次。
    消費符：增加攻擊力。
    鬼雞爪子：攻擊敵人全體。
  姬良：
    紙蟑螂：增加仙攻。
    帶火風輪：增加閃避率。
10.值得姬良學習的奇術：
  (1)蝶舞飛仙：消耗靈力少，傷害力不低，是初期很好用的奇術。
  (2)乾坤八劍：正常玩時傷害最高的奇術，需耗費大量體力，專門用來打魔王。
  (3)萬貫家財驅窮民：傷害比［蝶舞飛仙］高，需耗費不少體力，打魔王前清
     小兵用（沒體力後改用蝶舞飛仙）。
  (4)化魔符紙：小兵命-90%（機關無效），可用來對付後期生命很高的小兵。
  (5)機關絕壁陣：全體加防，魔王戰好用。
  (6)御劍燎原：遊戲後期才學得到，傷害比［萬貫家財驅窮民］高，需耗費大
     量體力，取代［萬貫家財驅窮民］在打魔王前清小兵。
  (7)陰陽黑火：消耗靈力少、傷害很高的群攻奇術，清小兵很好用，只是必須
     進玄冥宮收化機關黑火魔才能學到。
  (8)黑火陰雷、黑火陽炎：遊戲末期才學得到，消耗靈力少、傷害高的水火奇
     術，通常是用來打玄冥宮裡的多毛民與水魚民。
  (9)呱呱呱、唧唧唧：消耗大量靈力、傷害最高的水火奇術，必須進玄冥宮收
     化多毛民或水魚民才能學到，很不容易。
11.後期魔王建議打法：
  水鏡：奇魄古玉、某水
    水鏡用奇魄古玉聚勁、姬良幫施展健身咒、疾鵬幫使用天仙酒後（1.01版
    以前要先聚勁，1.02版以後次序任意），再用某水的雙重施展驚天動地拳。
  姬良：紙蟑螂、帶火風輪
    抽空施展呱呱呱、唧唧唧或乾坤八劍攻擊。
  疾鵬：如意鬼手、消費符
    偷完東西後，抽空施展乾天太紅（還沒學會的話便隨便打）。

傷害計算公式：

1.一般攻擊基本傷害
  水鏡、疾鵬 = 角色攻擊力 - 敵人防禦力
  姬良 = 角色仙攻 - 敵人仙防
2.奇術攻擊基本傷害
  水鏡、疾鵬 = 奇術傷害 + 角色攻擊力 - 裝備攻擊力 - 敵人防禦力
  姬良物理傷害奇術 = 奇術傷害 + 角色仙攻 - 敵人仙防
  姬良水火傷害奇術 = (奇術傷害 + 角色仙攻 - 敵人仙防) * (奇術效率-敵人抗性)%
  敵人雷水、火風抗性均為0時，奇術效率=100%
3.絕技攻擊基本傷害 = 絕技傷害 + 角色攻擊力 - 敵人防禦力

實際傷害 = 基本傷害 + f(角色等級) + 1～角色等級

1.基本傷害最低為0。
2.f(角色等級)的值目前找不出規則，大約在3～9之間。
3.憤怒一擊時，傷害加倍。但若基本傷害<0時不作用。
4.實際傷害最高9999。
