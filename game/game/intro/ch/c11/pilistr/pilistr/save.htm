存檔修改
修改：存檔SAVE\?.sav，0.sav為自動存檔，1.sav為快速存檔，2.sav為前次存檔，3.sav以
      下才是實際存檔。

1.屬性格式

X1 X1 X1 X1 X2 X2 X2 X2 S1 S1 H1 H1 M1 M1 S2 S2
H2 H2 M2 M2 DF DF IT IT PT PT SP SP -- -- -- --
HR HR QQ QQ -- -- LV LV -- -- -- -- -- -- D1 D2
D3 D4 D5 D6 D7 D8 D9 D0 -- -- -- -- -- -- -- --

X1 = 升級需要的經驗值
X2 = 目前經驗值
S1 = 力道
S2 = 氣勁
H1/H2 = 目前／最大生命
M1/M2 = 目前／最大內力
DF = 防禦
IT = 智慧
PT = 資質
SP = 身法
HR = 準頭
QQ = 靈巧
LV = 等級
D1～D0 = 抗性，依次為風雷冰火土光暗力毒亂，其值為256-抗性（9Ch為100%）

2.物品

利用UltraEdit開檔後，直接搜尋物品代碼，即為物品記錄的相關資訊：

TT TT NO NO

TT = 物品代碼
NO = 數量

如果想新增物品，建議是修改已裝備的武器與修煉的秘笈，如此移走後，便可自動做正確的
分類，其格式為：

修煉秘笈資訊，每個佔6 byte
裝備的武器代碼

修煉秘笈資訊：

TT TT NO NO EX EX EX EX

TT = 秘笈代碼
NO = 數量，固定為01 00
EX = 已修煉經驗值
