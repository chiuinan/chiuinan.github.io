提示：

1.各屬性意義如下（各系屬性最大值為99%，其餘最大值為999）：
  (1)體力：戰鬥時降為0即陣亡。
  (2)內力：使用招式或武器時需要。
  (3)力道：影響普通攻擊的傷害。
  (4)氣勁：影響招式或武器的傷害。
  (5)防禦：影響被敵人普通攻擊時所受的傷害。
  (6)智慧：無作用。
  (7)資質：無作用。
  (8)身法：影響戰鬥時出手的先後順序，以及普通攻擊時，擊中與被擊中的機率。
  (9)準頭：影響暗器傷害與擊中機率。
  (10)靈巧：無作用。
  (11)各系屬性：影響該系掌招或武器的傷害。
2.第七章前秦假仙別練得太高等級，因有一段需要靠黑白郎君的一氣化九百打破巨石，如果
  秦假仙等級練得太高，氣勁太強，一發掌就打死黑白郎君，那麼就會卡在那裡了。建議是
  修練掌心雷射箭，拿到霹靂丹時暫時不使用，等要打巨石時，便拿玄鐵劍施展掌心雷射箭，
  只要秦假山的氣勁低於730便可以成功通過。由於雷系屬性與該時期之前的主攻招式無關，
  比較沒有影響。或是留下大天星的3次攻擊專門對付巨石，如此練多高等級都無所謂。
3.武器部份最好以劍為主，因其秘笈較實用。但裝備劍會影響到掌招傷害，最好依後述傷害
  計算公式（或在戰鬥中實驗）決定是否應裝備劍。等到後期以玄黃一氣、五絕神功為主攻
  技時，應改拿刀而不再拿劍（最佳為巫月神刀，其次為少爺刀）。值得修練使用的武器與
  秘笈如下（依拿到次序列表）：
  (1)攻擊：水藏劍法、風雨襲蘭花、掠地旋風、四方綜合拳、九五至尊劍、玄黃一氣、
           五絕神功
  (2)恢復：銀蝶飛舞、還魂咒、淨衣咒、鎮血斷、冰心訣、贖魂、四象靈光、五氣朝元
  (3)輔助：飛龍探雲手、凌空逝影
  (4)增強：移山轉岳（土+10%）、純陰真體（冰+10%）、昊陽金剛體（火+10%）、
           紫金腿（身法+1）
  (5)飛索：十字黑鋼刀、浪子的鎖鏈、九截鞭
  增強屬性的秘笈以移山轉岳、昊陽金剛體為優先，因為冰屬性將來到天雪尖時可不斷偷取
  極凍果來強化。
4.萬寶洞羅網乾坤崎路人以原價2/3收購物品。
5.戰鬥後，自動恢復25%體力與內力。陣亡隊員亦會在戰鬥後自動復活，但體力只恢復為1。
6.人物等級最高72級，此時再使用金蠶王提升等級，已不會再增加屬性值。
7.幻形神拳秘笈在遊戲中是拿不到的，只能透過修改。因此五絕神功換不到，必須在第八章
  初用買的。但即便拿到，也是必須等第八章後期拿到太陽指後才能去換，用買的還比較快。

*** 傷害計算公式

備註：

1.進入戰鬥時，己方人物有低機率會變成弱化狀態，所有傷害均減少15%，直到該場戰鬥結束。
2.若進入戰鬥時未變成衰弱狀態，則每回合攻擊時，有低機率變成強化狀態，該回合傷害會
  增加15%，但僅限於該回合。
3.每個戰場都有其強化與弱化的掌系，例如九重天山道對力系的傷害會增強10%等，此部份的
  資訊目前暫未分析出來。

計算公式：

1.己方普攻

if 己方力道 < 20 實際力道 = 己方力道 * 0.8
else 實際力道 = 己方力道 * (0.9～1.25)
基礎傷害 = 實際力道 - 敵方防禦*2/3，最小為0
實際傷害 = 基礎傷害 + (己方等級/2～己方等級*4/3)

2.敵方普攻

基礎傷害 = 敵方力道/2 - 己方防禦/3，最小為0
實際傷害 = 基礎傷害*(1.0～1.1) + (2～4)

己方防禦時，傷害減半

3.己方招式／武器

基礎傷害 = 己方氣勁 * (100+己方該系屬性)% * (100-敵方該系抗性)%
實際傷害 = 招式傷害 + 基礎傷害

(1)武器招式均視為力系屬性攻擊。
(2)招式若標示為他系傷害者，依所列各系計算其基礎傷害後，併入實際傷害。
(3)持劍或持雙刃發掌招時，己方氣勁會減少20%，但持刀不會。若持劍或雙刃在戰鬥中要發
   掌招時，最好換下裝備後再攻擊。
(4)標示固定傷害者，其實際傷害=固定傷害，不以本公式計算。

4.敵方招式／武器

為固定傷害，不因己方防禦動作或防禦屬性而減少。

5.己方毒／蠱

(1)未標示傷害者，一律固定傷害=等級x(1～2)。
(2)有標示傷害者，依其屬性按招式計算，若持劍或雙刃，一樣會減少傷害。

6.己方暗器／法寶／迴力／飛索

增強率 = 準頭-6，最高29%（準頭>=35即達最高）
實際傷害 = 暗器傷害 * (100+增強率)%

(1)某些場景會變成固定傷害或無法傷害。
