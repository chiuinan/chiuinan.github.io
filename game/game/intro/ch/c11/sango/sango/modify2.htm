修改：武將資訊檔DAT.CMP

檔案有壓縮，所以改了某些資訊可能會影響其他資訊（特別是速度值）。解壓
方法如下：

準備4096 byte緩衝區做循環寫入，初始寫入位置=0FEEh
while 還有資料 do
  讀取控制碼
  由低位元開始檢驗控制碼的8個位元
  if 位元=1 then
    讀取資料，輸出並寫入緩衝區
  else
    讀取2 byte資料，假設依次為X和Y
    重複位置 = ((Y & 0xF0) << 4) + X
    重複長度 = (Y & 0x0F) + 3
    for i = 0 to 重複長度-1 do
      讀取緩衝區裡，重複位置+i的資料，輸出並寫入緩衝區
    end for
  endif
end while

註：假設緩衝區寫入指標為index，則寫入後，下個寫入指標為(index+1) & 0FFFh

解壓後的武將資訊，每位26 byte，格式為：

AT IT PN PN -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- -- AB JI SP --

AT = 武力
IT = 智力
PN = 最大帶兵量（隨等級增加者，多半為0）
AB = 武將類型，值域0～5，>=3才能當己方軍師
       00 = 無法招降的敵方君主與宗親
       01 = 無法招降的敵方武將
       02 = 可以招降的一般武將
       03～05 = 可以招降的軍師級武將，各有其特別策略（例如05為暗殺之計）
JI = 可招降時機（無法招降者多為C8h）
SP = 速度
