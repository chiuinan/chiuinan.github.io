記憶體修改
存檔有加密，故只能搜尋並修改記憶體。全隊資訊：

-- -- -- -- -- -- -- -- -- -- -- -- -- A1 A2 A3
BB BB BB BB CC CC CC CC DD DD DD DD -- -- -- --

A1/A2 = 剩餘／最大謀略值
A3 = 戰鬥中剩餘謀略值
BB = 金
CC = 兵糧
DD = 經驗值，勿改太大升級過頭，填入00 00 10 00升到50級便差不多了

隊員資訊（16 byte一組，共7組連續）：

-- -- HH E1 E2 E3 E4 E5 E6 E7 E8 F1 F2 F3 GG GG

HH = 武將編號，00表沒有
E1～E8 = 身上物品代碼，00表沒有（必須在後面）
F1～F3 = 裝備物品代碼，依次為手持、身穿、頭戴，00表沒有
GG = 目前帶兵量，0表示死亡

1.最強裝備為13正氣戟、1A正龍盔甲、22火神頭盔。
2.赤兔馬全隊只能有一個，超過的話便會全部無作用。
3.隊伍的武將可以重複，且重複武將都共用第一位重複武將身上的道具（包括
  赤兔馬）。

其他資訊：

1.可以買守護之煙，使用後數值為64，每走一步數值減一，找到位址鎖定後，
  平時便不會再遇敵。
2.隊伍座標格式為XX XX YY YY，找到後可以直接改變該值來移動隊伍。
