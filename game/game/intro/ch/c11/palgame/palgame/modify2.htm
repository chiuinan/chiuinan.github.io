資訊修改
以下為新仙劍奇俠傳各遊戲資訊檔的作用：

GeIndex.dat    各類資訊位置索引檔
ghero.dat      初登場人物資訊檔
gmon.dat       怪物資訊檔
glevel.dat     升級資訊檔
gitem.dat      物品資訊檔
gmagic.dat     仙術資訊檔
gmap.dat       場景資訊檔
maps.dat       場景物件檔

*** 各類資訊位置索引檔 GeIndex.dat

檔頭資訊數：4 byte整數
檔頭資訊：每組8 byte，格式作用不明
資訊類別數：4 byte整數
各類資訊：
  該類資訊數：4 byte整數
  該類資訊：每組20 byte
    資訊長度：4 byte整數，固定10h=16
    資訊特徵碼：4 byte整數
    資訊特徵值：4 byte整數
    資訊所在位置：4 byte整數，實際位置須+200h
    資訊所在檔案：
      1=GFight1.dat
      2=GFight2.dat
      3=GFight3.dat
      4=GFight4.dat
      4=初登場人物資訊：ghero.dat
      5=怪物資訊：gmon.dat
      6=升級資訊：glevel.dat
      7=物品資訊：gitem.dat
      8=仙術資訊：gmagic.dat
      9=場景資訊：gmap.dat

以下為各類資料的特徵碼：

1.物品／仙術
  特徵碼：7F100000h
  特徵值：物品編號或仙術編號
2.怪物／初登場人物／升級
  特徵碼：7F101000h
  特徵值：內部編號
3.執行代碼
  特徵碼：7F102000h
  特徵值：執行代碼
4.場景
  特徵碼：7F103000h
  特徵值：場景編號

資訊所在位置的內容均以特徵碼表示，類似腳本Script的執行，最後以特徵碼
7F00FFFFh做為結束。

*** 特徵碼一覽表

7F00FFFFh = 結束
7F100000h + 物品或仙術編號
7F101000h + 怪物／初登場人物／升級內部編號
7F102000h + 執行代碼
7F103000h + 場景編號
7F200001h + 物品仙術屬性編號 + 屬性值
7F201001h + 人物怪物屬性編號 + 屬性值
7F202041h + 物品編號 + 獲得數量
            7EFFE002h = 金錢
7F102046h + 事件代碼
7F30200Bh + 裝備效果編號 + 裝備屬性編號 + 屬性值
            7EFF1000h = 屬性
            7EFF1001h = 抗性
            7EFF1015h = 屬性（全體作用）
7FF2F000h + 名稱，BIG5或GB碼，以%%做為字串結束
7FF2F001h + 說明文字，BIG5或GB碼，以%%做為字串結束
