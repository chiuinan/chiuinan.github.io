﻿名稱：仙劍奇俠傳 (中文版)
簡介：大宇1995年出品的角色扮演遊戲，採3D斜角畫面呈現，以團隊回合方式
      戰鬥。此遊戲一出品即大受歡迎，即使歷經數年仍為玩家所津津樂道。
      劇情以苗族及女媧為主要題材，故事則主要描寫主角李逍遙與趙靈兒、
      李月如等人的生死戀情。除了美工細膩、音樂動聽外，最吸引人的還是
      遊戲中淒美又感人的劇情，而結局更留給人很大的想像空間。整個遊戲
      的困難度相當容易，也因為如此，玩家更可細細品味本遊戲在劇情上的
      安排。無論對於新手或老手，若沒玩過此款國內算是極品的遊戲，實在
      是很可惜的一件事。

備註：各種版本的差異：

1.DOS磁片／光碟版：1995年出品，遊戲內容大抵相同，前者因磁片空間關係有
  刪減部份內容，後者多了8首CD音軌，可於遊戲中播放。
2.95視窗版／98柔情版：1997年出品，遊戲內容均相同，前者為繁中版，後者
  為簡中版。增加了AVI動畫、4首CD音軌、背景音樂由DOS的FM音源（RIX格式）
  改成MIDI格式、所有道具與法術加入文字說明、簡化部份迷宮、更動少量對
  白、降低我方法術習得等級、降低怪物生命與屬性值等。
3.夢幻版：網友依照仙劍奇俠傳原版，自行修改出的MOD版。最初由外塞之霧製
  作，直到版本V2.11。因為困難度過高，後由劍仙一夢再加以修改，調降其難
  度，版本改為V2.2。
4.新仙劍奇俠傳：2001年重製復刻版，主要劇情相同，但其他如支線、畫面、
  迷宮、怪物、甚至結局等都有加以重新修訂。

備註：95版光碟必須掛在第一個光碟機。此版在XP下無法播放動畫，可將官網
      提供的98柔情簡中版裡，*.avi與musics子目錄替換過來，再將pal.dll
      更名為palold.dll（名字不可錯誤），並將官網版98柔情版裡的pal.dll
      拷貝過來，即可正常玩（同時亦破解掉光碟保護）。
備註：95/98版必須移走MUS.MKF這個檔案，才會使用MUSICS子目錄下的MIDI音樂。
檔案：Pal95.rar = 95繁中版，已修改成XP可執行，並更新音樂檔
       （開頭音樂若要變回原版，請將musics\004-old.mid改回004.mid）
       （要使用DOS版音樂可將DOS版的MUS.MKF檔案拷貝進來）
      Pal98rqp.rar = 98柔情簡中版（官網免費下載版，附繁中化檔案）
      MapEditor.rar = 地圖編輯器（含原始碼）
