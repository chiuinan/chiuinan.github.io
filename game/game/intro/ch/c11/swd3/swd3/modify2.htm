*** 怪物／物品／奇術資訊檔 MON.DAT

檔頭：200h=512 byte
資訊位置指標：各4 byte（實際位置=位置指標+200h），0表沒有定義
  0001h = 奇術傷害定義資訊
  0002h～01F5h = 活物指標編號 = 活物代碼+1
  01F6h～05DDh = 物品指標編號 = 物品代碼+1
  05DEh～06B9h = 奇術指標編號 = 奇術代碼+1

*** 資訊代碼與參數

各參數未特別標示時，內定為2 byte
機率單位值為5%
抗性均為(-值)*10%

  (攻擊奇術+機率)x3 + (不明攻擊代碼+機率)x2 + (恢復奇術+機率)x2 +
  掉落物品+機率 + 可偷物品+數量 + 掉落金錢 + 不明6 byte + 抗性 + 不明
  活物代碼 = 0表非敵人
  屬性 = 等級,攻擊,防禦,敏捷,力量,耐力,智慧,經驗值,生命

*** 場景／雜項資訊檔 MAPS.DAT

檔頭：200h=512 byte
資訊位置指標：各4 byte（實際位置=位置指標+200h），0表沒有定義
  0001h～0012h = 不明
  0013h = 功能字串資訊
  0014h = 地點名稱資訊

*** 功能字串資訊

字串位置表：按代碼排列（例如烈火術=05DDh），指向字串內容位置（實際位置=位置指標+200h）
字串內容：以%Q為結束

*** 升級資訊檔 LEVEL.DAT

檔頭：200h=512 byte
資訊位置指標：各4 byte（實際位置=位置指標+200h），0表沒有定義
  每個人物100個，各代表0～99級升級資訊
  人物依次為賽特、妮可、卡瑪、李靖

*** 升級資訊代碼與參數

0000 + 屬性x7 + 不明x4 + 4 byte所需經驗
  屬性 = 生命 + 靈力 + 體力 + 力量 + 耐力 + 智慧 + 敏捷
0001 + 學到的奇術代碼
0005 = 資訊結束
