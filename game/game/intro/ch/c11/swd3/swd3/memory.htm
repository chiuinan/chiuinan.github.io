修改：（修改可能造成無法完成Domo任務，請自行斟酌）

無法直接使用GameMaster 8尋找，必須搭配GameMaster Plus III，方法是進入
遊戲後，輸入INT中斷遊戲，等GM的命令列顯現後，輸入SCAN和想尋找的數值，
再輸入INT恢復遊戲。等找到只剩一個時，再輸入STP命令鎖定位址與數值（必須
從右邊九宮鍵輸入）。命令使用方法可參見GMP3執行後畫面上的說明。鎖定後，
即可切出遊戲，到GM8裡修改記憶體或其他處理（每個人物佔38h byte）：

XX XX XX XX A1 A1 B1 B1 C1 C1 A2 A2 B2 B2 C2 C2 DD DD EE EE FF FF GG GG

XX = 目前經驗值
A1/A2 = 目前命／最大命
B1/B2 = 目前靈／最大靈
C1/C2 = 目前體／最大體
DD = 力量
EE = 耐力
FF = 智慧
GG = 敏捷

裝備可購買可消耗物品（如藥草），然後在新品的狀態下消耗來搜尋數量（不可
離開物品頁面，否則原記憶體資訊會消失），接著找數量之前的代碼修改。各類
