2.各屬性意義如下：

    (1)生命：降為0即瀕死，無法再戰鬥。
    (2)靈力：施展奇術時消耗。
    (3)體力：施展物品、法寶、與某些奇術時消耗。
    (4)力量：影響攻擊力。
    (5)耐力：影響防禦力。
    (6)智慧：影響奇術傷害力。
    (7)敏捷：影響攻擊的次序與頻率。
    (8)抗性：影響相對應系統攻擊的傷害程度，包括奇術與普通攻擊。
    (9)攻擊：影響對敵方普通攻擊時，所造成的傷害。
    (10)防禦：影響敵方普通攻擊時，己方所受的傷害。

  所有屬性，包括力量、耐力、智慧、命靈體等，皆不可超過9999（含升級後增
  加的值），否則必當。其中敏捷不可點太高，否則進入戰鬥會當，一般加到不
  含裝備200出頭即可。如果進入戰鬥會當，可試著到設定裡調低戰鬥的速度看看。
3.任何支線需要交出的物品，只要裝備在身上便不會交出，但仍可完成支線。
4.己方角色人物等級最高為60級。其中李靖的升級表資訊有誤，當升到60級時，
  力量會從200降至148，因此若未修正此問題時，請勿將李靖升到60級。
5.賣出價格為原價60%。
6.武器熟練後的額外攻擊力是直接作用在攻擊傷害上，並不會顯示在攻擊力的總
  數據裡，因此在更換武器時，必須自行計算總傷害，才能得知是否划算。
7.值得練的法寶有：
  (1)神偷面具（偷竊）：專偷怪物身上的好物品，必練。
  (2)薩登荊環（攻擊+50%）、星君眼（攻擊+50%）、吸血鬼指環（吸血）、
     吸魔壺（吸靈）、靈虛扇（抵抗不良狀態）：這類無指令的法寶均值得練。
  (3)暗黑魔樽（暗抗80%）、雷公鎚（電抗80%）：打最後幾個魔王時需要。
8.賽特的絕技有兩個路線，主要看劇情走到向慧彥學習［御劍燎原］前，是否已
  學會［蛟龍翻身］：
    已學會［蛟龍翻身］：最終技為［霸王崩山勁］（敵全1800物理傷害）
    未學會［蛟龍翻身］：最終技為［天狼巽閃］（敵單6000物理傷害）
9.隨行人員的攻擊力與速度，是以賽特的攻擊力與敏捷屬性為基準的。

*** 傷害計算公式

己方攻擊 = 人物力量 + 裝備附加攻擊力
己方防禦 = 人物耐力 + 裝備附加防禦力

1.己方普通攻擊

  攻防傷害 = 己方攻擊 - 敵方防禦，最低為0
  額外傷害 = 附帶奇術傷害 * (100-敵方該系抗性)%
  基礎傷害 = 攻防傷害 + 武器熟練傷害 + 額外傷害 + 1～(3+己方等級/3)
  實際傷害 = 基礎傷害 * (100-敵方物理抗性)%

2.敵方普通攻擊

  攻防傷害 = 敵方攻擊 - 己方防禦，最低為0
  基礎傷害 = 攻防傷害 + 1～(3+敵方等級/3)
  實際傷害 = 基礎傷害 * (100-己方物理抗性)%

  己方採取防禦動作時，實際傷害隨機減少10%～40%

3.己方奇術攻擊

  基礎傷害 = 奇術傷害 + 1～(己方等級/2+己方智慧/6)
  實際傷害 = 基礎傷害 * (100-敵方該系抗性)%

  若奇術為特殊傷害，敵方抗性為0。

4.敵方奇術攻擊

  基礎傷害 = 奇術傷害 + 1～f(敵方等級)
  實際傷害 = 基礎傷害 * (100-己方該系抗性)%

  己方採取防禦動作時，實際傷害隨機減少10%～40%

5.投擲與其他武器攻擊（非法寶投擲指令）

  實際傷害 = 武器傷害 * (100-敵方該系抗性)%
