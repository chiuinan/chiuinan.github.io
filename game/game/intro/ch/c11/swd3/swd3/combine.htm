煉化表

在還沒拿到東方的封神壇前，還是可以煉東方妖怪。在煉妖的狀態下，先預覽西
方的妖怪，之後按End鍵或右邊的Shift鍵，西方的妖怪會變暗，也就是切換到東
方祭壇，之後按ENTER就可以煉出東方妖怪了。能直接煉出的物品為主角等級+9，
但利用前述方式煉出的東方妖怪不受此限。以下先列出各適當的等級，可藉由此
點煉出的好用物品（拿到東方的封神壇後便無法使用此密技，故李靖所需的裝備
必須在他加入之前便先煉好）。

1.６級

基礎煉化：

  草藥 + 草藥 = 侏儒(15)／水仙(39)

水仙（在右）可煉物品：

  藤鞋(7) = 虎皮衣(妮可，26)
  頭巾(9)／皮靴(10) = 思潮之袍(賽特，27)
  布帽(11)／胸甲(12) = 虎爪鞋(賽特，28)
  皮帽(13)／鋼靴(13)／手鐲(13) = 金鋼斗笠(李靖，29)

2.１８級

水仙（在右）可煉物品（西方為溫蒂妮）：

  甲冑(15)／麟靴(16) = 癒傷之鎧(塞特，李靖，補5%命，30)
  皮手套(17)／銀髮結(17)／鎖子甲(18) = 蛇皮短靴(妮可，卡瑪，31)
  輕鋼盔(19)／鐵鎖手套(19) = 蛟鱗衣(妮可，32)
  鐵手套(21)／片葉鎧(21)／安碧特麗之冠(21)／鷹爪靴(22)／棉布衣(22) = 烏蠶衣(賽特，妮可，卡瑪，李靖，33)
  羽盔(23)／白金之腕(23)／貂皮衣(24)／鎖鎧(24)／鋼手套(25)／虎皮衣(26)／思潮之袍(27)／虎爪鞋(28) = 銀絲戰袍(賽特，妮可，卡瑪，李靖，36)
  金鋼斗笠(29)／蛇鱗衣(30)／癒傷之鎧(30) = 龍鱗盔(賽特，李靖，補2%命，37)
  蛇皮短靴(31)／蛟鱗衣(32) = 天蠶絲衣(妮可，卡瑪，38)

其他：

  飛斧 + 飛斧 = 雪球怪(22)／阿加希歐(27)
  阿加希歐 + 阿加希歐 = 銀笛(27)／玉戒指(28)

3.１９級

若已到血腥地獄買到赤螫，則基礎煉化：

  赤螫 + 赤螫 = 黏怪(12)／石獅精(48)

然後按[防具煉化總表]，利用石獅精可以煉出大部份的防具。35級之後便不必考
慮此一煉化。
