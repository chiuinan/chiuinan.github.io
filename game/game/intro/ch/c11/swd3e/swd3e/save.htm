存檔修改
修改：神魔異事錄Fame.dat

每2 byte一組，表示該怪物的辨識率，14h=20為100%。
由位置0002h到036Eh全填，即可全部100%辨識。

修改：存檔Save\?.sav

修改存檔容易導致當機跳出，且無法完成DOMO任務，最好還是儘量避免。

1.屬性修改：找命靈體數據

  HP HP MP MP TP TP -- -- -- -- A1 A1 A2 A2 A3 A3 A4 A4

  HP,MP,TP = 目前的生命、靈力、體力
  A1～A4 = 力量、耐力、智慧、敏捷

2.法術修改法：

  假如主角習得新的奇術或絕技，則直接按後面的奇術代碼找到後，替換成想得
  到的奇術。

修改：符鬼存檔swdghost.dat

swdghost.dat的檔案內容沒有保護，因此可以直接修改裡面符鬼各屬性和奇術。
奇術建議十邪散魂一個就好（F9 17）。

金符鬼資料開始位置：0000h
木符鬼資料開始位置：0158h
水符鬼資料開始位置：02B0h
火符鬼資料開始位置：0408h
土符鬼資料開始位置：0570h
