新手提示
操作：

滑鼠右鍵 = 移動
滑鼠左鍵 = 交談、撿取物品
R = 走路／跑步切換
A = 自動攻擊
P = 抓圖（放在ScrnShot目錄下）
其餘按鍵可參考設定選單裡的按鍵設定

提示：

1.本遊戲為了避免玩家過度練功，若同一種怪物組合打超過５次以上，則第６次
  殺掉該怪物組合所得的經驗值就會減半。必須結束遊戲重新執行程式，才會回
  復。
2.遊戲有兩個結局，端看對阿雪與玉兒的好感度而定。若想和誰有結局，就儘量
  先跟她談話，或者依她喜好的方式進行。
3.己方角色人物等級最高為60級。
4.賣出價格為原價60%。
5.武器熟練後的額外攻擊力是直接作用在攻擊傷害上，並不會顯示在攻擊力的總
  數據裡，因此在更換武器時，必須自行計算總傷害，才能得知是否划算。
6.值得練的法寶有：
  (1)藍格怪衣（龍飄）
  (2)暗殺寶錄／御夫錄（投擲）
  (3)乾坤袋（偷竊）
  其他還有星君眼（攻擊+50%）、玄能臂環（吸血）、靈虛扇（抵抗不良狀態）、
  七仙神針（免死一次）。初期的鼠鎚也可練練。
7.金塊除了可以拿來賣錢之外，還可以作為投擲敵人的道具，殺傷力頗高。可利
  用煉化方式煉出一堆來用。
8.使用奇術攻擊時，要先弄清楚奇術屬性與敵人抗性，否則有可能反而替敵人補
  血。以下為五行相生相剋表：
    金抗：怕火象  抗木象  吸土象
    木抗：怕金象  抗土象  吸水象
    水抗：怕土象  抗火象  吸金象
    火抗：怕水象  抗金象  吸木象
    土抗：怕木象  抗水象  吸火象
  或
    金象：剋木抗  懼火抗  生水抗
    木象：剋土抗  懼金抗  生火抗
    水象：剋火抗  懼土抗  生木抗
    火象：剋金抗  懼水抗  生土抗
    土象：剋水抗  懼木抗  生金抗
9.己方各人物均有其特有的抗性屬性：
    陳靖仇：木5%
    于小雪：土15%
    拓跋玉兒：火15%
    張烈：水20%
  當裝備加抗性的裝備時，只能增強人物所特有的抗性屬性。例如陳靖仇若裝備
  泥蚣袍，雖然五抗+40%，實際上只有木抗+40%，而且遇到剋木的金象奇術，傷
  害也增加了40%，並非抗性愈高愈好。
10.若要方便煉化，最好遊戲不久便利用密技將金錢變成極多，以便買大量的煉化
  材料。建議是在黑山鎮買羅勒草到99個，再到隋軍營區撿一個便可以達成。

*** 傷害計算公式

己方攻擊 = 人物力量 + 裝備附加攻擊力
己方防禦 = 人物耐力 + 裝備附加防禦力

1.己方普通攻擊

  基礎傷害 = 己方攻擊 - 敵方防禦，最低為0
  實際傷害 = 基礎傷害 + 武器熟練傷害 + 1～(3+己方等級/3)

  若武器附帶奇術傷害，則按後述［己方奇術攻擊］相生相剋原則計算額外傷害
  （基礎傷害=附帶奇術傷害）。但此處系統在計算上有bug：

  (1)吸：實際抗性 = 100-敵方抗性（應為+才對）

  結果應該補血的數量變成了傷害的數量。

2.敵方普通攻擊

  基礎傷害 = 敵方攻擊 - 己方防禦，最低為0
  實際傷害 = 基礎傷害 + 1～(3+敵方等級/3)

  己方採取防禦動作時，實際傷害隨機減少10%～40%

3.己方奇術攻擊

  基礎傷害 = 奇術傷害 + 1～(己方等級/2+(己方智慧-5)/4)

  依敵方抗性（可能多個）與己方奇術傷害屬性，由五行相生相剋表依次決定：

  (1)吸：實際抗性 = 100+敵方抗性
     若有其他抗、怕抗性，則依比例增減敵方抗性。
  (2)抗：實際抗性 = 敵方抗性
     若有其他怕抗性，則依比例減少敵方抗性。
  (3)怕：實際抗性 = -敵方抗性
  (4)無：實際抗性 = 0

  實際傷害 = 基礎傷害 * (100-實際抗性)%

4.敵方奇術攻擊

  基礎傷害 = 奇術傷害 + 1～f(敵方智慧)
  實際傷害依前述［己方奇術攻擊］的相生相剋原則計算

  己方採取防禦動作時，實際傷害隨機減少10%～40%

5.投擲武器攻擊（非法寶投擲指令）

  基礎傷害 = 武器傷害
  實際傷害依前述［己方奇術攻擊］的相生相剋原則計算

*** 神魔異事錄

1.必須存檔後，新的辨識度才會跟著存檔。
2.怪物每打敗一次，或煉化出來一次，便得到5%辨識度。
3.一般的怪物可利用煉化方式提高辨識度，即先存檔，煉化後另存新檔，然後取
  舊檔再煉化一次，如此反覆20次即達到100%辨識度。
4.關卡魔王可於打前存檔，打完後另存新檔，然後取舊檔再打一次，反覆20次即
  達到100%辨識度。
5.魔性師父必須玉兒結局才能存檔增加辨識度，阿雪結局因直接進入結局，無法
  存檔，故無效。
6.某些不會出現的怪物，如九天玄女、上元夫人，或是打不敗的魔王，如刑天等，
  只能藉由煉化的方式增加辨識度。
7.神魔異事錄有兩格是無法辨識到的，一是太白金星，等級63，無法煉化出來。
  一是斛律安，神魔異事錄裡記錄的是長沙無敵的斛律安，而非巴蜀古王墓裡的
  斛律安，因為不敗，所以無法辨識到。
