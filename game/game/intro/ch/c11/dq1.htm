名稱：勇者鬥惡龍1 (Dragon Fighter，中文版)
簡介：ENIX 1989年出品的角色扮演遊戲，由精訊中文化並代理發行。整個遊戲以2D畫面呈現，
      採單人（對單人）回合方式戰鬥。劇情描寫龍王偷去了以前勇士洛特用以制伏妖魔的
      光珠，使人民生活於黑暗之中，國王乃囑咐主角，前往打倒龍王並取回光珠。勇者鬥
      惡龍系列為早期頗受玩家喜好的遊戲，其變態的練功方式也相當著名，因此不喜花大
      量時間在練功上的玩家，實際上可以不必考慮玩此系列的遊戲。本遊戲的劇情不長，
      謎題也很簡單，除了敵人遭遇頻繁、道具太滿再拿取物品時會當機等的缺點外，大致
      還算是一個不錯的遊戲，沒玩過的玩家可以試試這款以前的經典遊戲。





保護：磁片
檔案：crack.rar = 破解檔
操作：

Space/Enter = 遊戲選單
Q = 離開遊戲
S = 音樂開關
註：與國王交談後可存檔

攻略：（作者－青衫）

01.在與國王談完話後，在附近３個寶箱拿取１５０元、火把和鑰匙，與附近
   的人聊聊，得知蘿拉公主被怪物抓走了。用鑰匙開門下樓，出拉達多姆城，
   先到東邊的拉特達姆鎮買些武器防具，然後開始出鎮練功。等到等級差不
   多，武器防具都買到最好的時，便往西北走。
02.先到最西北角的加萊鎮買更好的武器後，再進入其東南的洞窟。在洞窟內
   地下第２層拿到洛特的石板，由上面的刻字得知，要到魔島必須找到三位
   賢者拿取裝備。
03.出洞窟往東過２座橋，再折往北到瑪依拉村稍作休息，並購買一些藥草。
04.出瑪依拉村往南，經過沼澤進入洞窟，沿著西邊牆穿過洞窟到南方大陸去
   （以免遇到洞中的惡龍）。
05.出洞窟後往南不遠，在山與水圍繞的中央，進入利姆達爾市。先買些武器
   防具，尤其是魔法盔甲，可邊走邊補體力。接著由入口的橋前，沿著市區
   外圍走到西北角，進入買些鑰匙（最多６把）。
06.回到北邊大陸的瑪依拉村，在村內的妖精之泉下方４步處調查，得到妖精
   之笛。
07.回到最初的拉達多姆城，用鑰匙開啟東北的門，在東北角落的商店也有賣
   鑰匙。由賣鑰匙的商店往東走，沿著外牆到東南角，進入地下室拿取太陽
   之石。
08.到西北的加萊鎮，用鑰匙開門進入，再到各處往北嘗試出去，然後進入東
   北角的迷宮（若在迷宮前遇人擋道，便要等他出來為止，否則一進城便趕
   緊開門走到迷宮前）。由迷宮第一層東南角用鑰匙開門下到第二層，然後
   由最西南角的樓梯下到第三層，再走往東北到中央的樓梯下到第四層，然
   後回到第三層拿取銀之豎琴。
09.出加萊鎮先到瑪依拉村南邊的洞窟，在洞窟東南角處殺掉惡龍（等級要夠），
   救出蘿拉公主，將她帶回拉達多姆城交給國王，得到蘿拉之愛（可隨時查
   詢所在位置及升級經驗數）。
10.回到加萊鎮，往南過橋再往南，在沼澤東邊的洞窟內有戰士戒指，但使用
   會中詛咒，必須回拉特達姆鎮解詛咒，否則無法和國王交談存檔，還是不
   要拿比較好。繼續往南過橋，再往南進入廢棄的達姆杜拉鎮，在鎮內東南
   被沼澤圍住的樹下調查，得到洛特盔甲（經過沼澤不受傷害）。
11.出達姆杜拉鎮往南到底過橋，再到最東北角過橋，然後往南走過沼澤來到
   美爾吉特城，在城門前被石頭人哥倫擋住，使用妖精之笛催眠它，再將之
   打敗。進入美爾吉特城，買些武器防具後，並得知距拉達多姆城西４０里、
   北７０里處有物品。
12.出美爾吉特城往北過橋，往南走到中央再往東過橋，進沼澤，在最窄的地
   方向右五步處調查（可利用蘿拉之愛確認位置），可得洛特徽章。
13.回到瑪依拉村，到其西北方的祠堂，用銀之豎琴換取雲雨神杖。
14.到南方大陸的利姆達爾市，往南走到祠堂，用洛特徽章、雲雨神杖、太陽
   之石造得彩虹水滴。
15.回到利姆達爾市往西北走到盡頭，使用彩虹水滴造出彩虹橋。繼續往前進
   入龍王城，往西到北邊的椅子後方調查，可發現樓梯，進入後一直走到有
   叉路處（附近有另一個上樓樓梯），往西走下樓梯後，到東北方有個樓梯，
   不要下去，繞到東南上樓去拿洛特之劍。回到原先上樓處，改由西北下樓
   梯，到達龍王住所。北邊用鑰匙開門，裡面有賢者之石（可在戰鬥中無限
   次恢復體力）和魔法皮帶（不要使用，會被詛咒，無法和龍王交談）。最
   後和龍王交談，開始與龍王戰鬥，第一次打敗它後，它會化為龍形再次戰
   鬥，再次打敗它，回到拉達多姆城與國王交談，然後便結局了。
