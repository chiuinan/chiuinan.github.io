名稱：軒轅劍 (中文版)
簡介：大宇1990年出品的角色扮演遊戲，以2D人物排排走方式呈現，以團隊回
      合方式戰鬥。大宇軒轅劍系列是從第二集開始大受國內玩家的喜愛，但
      第一集（就是本遊戲）其實並不算好。主要因為原作者當兵，大宇便將
      只完成第一個章節的遊戲推出銷售，使得整個遊戲的完成度嚴重不足，
      劇情太短、謎題太少，讓人有騙錢的感覺，而且練功部份亦嫌太多了些。
      整體而言，這款遊戲是稍差了一些，想玩的人或許可以趁早改變主意吧。



保護：顏色密碼，破解法：

AA = 薰草不遇敵的剩餘步數
B1～B3 = 人物身上的物品編號（每人最多16個）
人物屬性（每人佔32 byte）：
  CC = 等級（最高25級）
  D1～D4 = 力量／體質／速度／運氣
  E1～E2 = 目前經驗／升級經驗（最大60000=EA60h）
  FF = 金錢（不是主角者，值固定FFFFh）
  G1～G3 = 攻擊力／防禦力／反應力（修改無效，會自動重算）
  H1/H2 = 最大／目前體力
  H3/H4 = 最大／目前法力

升級後，屬性會重置，因此若要修改屬性，應先升到最高的25級（先改成24級，
再改經驗後打怪升級即可，法術自動學會）。

-- -- PP PP XX XX -- -- -- -- -- -- -- -- -- --

PP = 座標，最左上為0，先水平後垂直進行編號，可改變移動人物位置
XX = 地圖水平寬度

從師父家出來後，座標為1078h=4216，大地圖水平寬度為0140h=320。

*** 屬性說明

力量：基礎攻擊力，升級時變更
體質：基礎防禦力，升級時變更
速度：基礎反應力，並決定敵我攻擊次序，升級時變更
運氣：決定逃跑成功的機率，升級時變更
攻擊力：角色的物理攻擊力=基礎攻擊力+裝備攻擊力
防禦力：角色的物理防禦力=基礎防禦力+裝備防禦力
反應力：決定閃躲攻擊的成功機率=基礎反應力+裝備反應力
