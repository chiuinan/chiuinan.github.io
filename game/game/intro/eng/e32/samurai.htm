操作：

上下左右 = 移動
Enter = 選擇／攻擊（配合移動鍵）／撿拾
BackSpace = 防禦（配合移動鍵）
Space = 暫停
R = 徹退
F1 = 各領主狀態欄
F2 = 大地圖
F3 = 各領主總結欄
Esc = 回到上一層
Alt-S = 存檔
Alt-R = 讀檔
Alt-V = 音樂音效開關
Alt-Q = 離開遊戲
