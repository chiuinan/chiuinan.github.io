HT = 健康，最多200=C8h
AM = 護甲，最多200=C8h
AT = 護甲形態，00=無，01=綠護甲，02=藍護甲
P1～P6 = 是否有強化裝置，非0=有/剩餘作用時間（單位tics=1/35秒）
         依次為無敵、狂暴、隱形、幅射防護衣、地圖、夜視鏡
K1～K6 = 是否有鑰匙，1=有
         依次為藍卡、黃卡、紅卡、藍骷髏、黃骷髏、紅骷髏
BP = 是否有背包，1=有
CW = 目前使用的武器，0 = 拳擊，1 = 手槍，2 = 霰彈槍，3 = 機槍
     4 = 火箭炮，5 = 電漿槍，6 = 雷射炮，7 = 電鋸
NW = 目前武器彈藥用盡準備替換的武器，0Ah表不更換（1.2版為09h）
W1～W9 = 是否有該武器，1=有，依次為拳擊、手槍、霰彈槍、機槍、火箭炮、
         電漿槍、雷射炮、電鋸、超級霰彈槍（Doom 2才能使用）
A1～A4 = 現有彈藥量，依次為子彈、霰彈、電量、火箭
M1～M4 = 最大彈藥量，依次為子彈、霰彈、電量、火箭
