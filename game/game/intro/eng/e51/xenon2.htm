名稱：氙星異形２ (Xenon 2 : Megablast，英文版)
簡介：Bitmap Brothers 1989年出品的射擊遊戲，畫面精美，當時不知風靡了多少玩家。各
      關都有頭目把守，不過途中可以購買更強的火力及裝備，還算是蠻不錯的射擊遊戲。
      只是這款遊戲一擊中便死，而愈到後面，迷宮愈複雜，敵人也愈多，要打到後面實在
      不怎麼容易。關卡一共有５關，且過關後不會從第１關繼續，有點稍嫌少了些。
 
保護：磁片
備註：正常版請執行PLAY.BAT，不死版請執行UPLAY.BAT
修改：XENON2.EXE

AF = 自動射擊數目，最大03，再大如08幾乎是連珠炮，但再吃到武器會當
SC = 分數
MM = 金錢
SP = 加速器數目，最大02
NT = 超級毀滅武器剩餘時間
AR = 防護力，最大27h
NO = 剩餘架次

密技：在選擇圖形界面時，按下F7，然後在玩時，再按下i鍵，便會成為無敵。
操作：

空白鍵 = 發射／開始
Enter = 潛航
P = 暫停
Esc = 離開遊戲
F10 = 結束遊戲

說明：

畫面最下方為防禦值，減到0即被擊毀，該關重來。消滅敵人後，會出現的東西：

1.膠囊：打掉特殊敵人會出現，裡面有可增加座機威力的裝備。
2.泡泡：打掉一組敵人或關卡魔王會出現，內含金錢，可用來買賣物品，泡泡愈大，金錢愈
  多。不過金錢過關後會重置，故在關卡最後的商店要儘量用完。
