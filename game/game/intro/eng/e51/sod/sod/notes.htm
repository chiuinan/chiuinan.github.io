*** 操作

一、選單

New Game：開新遊戲，須選難度，共有四種難度：
  1.Can I Play Daddy：毫無經驗的初學者（等級1）
  2.Don't Hurt Me：稍有經驗的新手（等級2）
  3.Bring'Em On：一般玩家（等級3）
  4.I Am Death Incarnate：想挑戰更高難度的老手（等級4）
Sound：聲音設定
Control：操作設定
Load Game：讀檔
Save Game：存檔
Change View：變更視野視窗
Read This!：顯示說明
View Scores：查看排行榜
End Game：結束遊戲
Back to Demo/Game：回到遊戲
Quit：離開遊戲

二、按鍵

Esc = 顯示選單／離開遊戲
F1 = 老闆來了（顯示假的Dos畫面，再按Esc回到遊戲）
F2 = 存檔
F3 = 讀檔
F4 = 聲音設定
F5 = 變更視野視窗
F6 = 操作設定
F7 = 結束遊戲
F8 = 快速存檔
F9 = 快速讀檔
F10 = 離開遊戲
Pause = 暫停
1 = 武器換成小刀（Knife）
2 = 武器換成手槍（Pistol）
3 = 武器換成機槍（Machine Gun）
4 = 武器換成鏈槍（Chain Gun）

以下為內定操作設定（可變更）：

上下 = 前進後退
左右 = 左右轉身
右Shift = 快速移動（搭配方向鍵）
Alt = 平移（搭配左右鍵）
Space = 開啟／搜尋
Ctrl = 射擊（機槍或鏈槍按住可連發）

*** 遊戲說明

1.初始武器=刀、槍，子彈（Ammo）=8，健康（Health）=100%，生命（Lives）=3，分數
  （Score）=0，所在關卡（Floor）=1。
2.被敵人擊中時健康度下降，下降至0%時即死亡，減少一命並從本關重新開始。
3.殺敵或撿拾寶物可獲得分數，每40000分增加一命，最多9命。
4.經過物品時會自動撿拾。除了地上可見物品外，殺死警衛經過屍體時也可撿拾子彈，最多
  99發（SS衝鋒隊有時會掉機槍）。
5.某些門需要鑰匙才能開啟，撿到鑰匙會顯示在子彈的右邊2小格內。
6.每過一關（無法回頭），依照各項指標給予額外分數：
    (1)殺死敵人100%：10000分。
    (2)拿取寶物100%：10000分。
    (3)推動暗格100%：10000分。
    (4)低於標準時間：每秒500分。
  秘密關或魔王關不以上述指標為標準，過關直接加15000分。
7.敵人不會站著不動，聽到槍聲甚至會四處搜尋或開門出來攻擊。

*** 物品說明

狗食（Dog Food）：恢復4%健康
餐點（Dinner）：恢復10%健康
急救包（First Aid）：恢復25%健康
子彈（Ammo）：獲得8顆顆子彈（若是敵人陣亡掉落則為4顆子彈）
子彈箱（Ammo Box）：獲得25顆子彈
機槍（Machine Gun）子彈可連射
鏈槍（Chaingun）子彈可連射，射速更快
十字徽章（Cross）／收音機（Radio）：加100分
聖杯（Chalice）／鈽（Plutonium）：加500分
寶箱（Chest）／控制板（Control Panel）：加1000分
皇冠（Crown）／炸彈殼（Bomb Casing）：加5000分
鑰匙（Keys）
加命（One Ups）：健康加滿、子彈加25、增加一命
