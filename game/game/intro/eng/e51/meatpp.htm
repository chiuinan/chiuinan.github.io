名稱：辣妹大進擊 (Meat Puppet，英文版)
簡介：Kronos 1997年出品的第三人稱射擊遊戲，由Playmates代理發行 [待補]。



備註：本版為GOG版
保護：光碟
備註：非安裝移機者，必須設定下列Registry：

[HKEY_CURRENT_USER\Software\Kronos\Meat Puppet\1.0]
"Directory"="."

備註：原始光碟版只能在PCem+Win9x系統上執行，否則播放音樂時會找不到光碟而跳出。建
      議玩GOG版（音軌已做成檔案，無需光碟），Win64需搭配dgVoodoo2。
備註：若嫌遊戲畫面太暗，記得到選項中將之調亮。
密技：遊戲進行中，輸入下列字串即可啟動密技：

~ingesth = 生命全滿
~ingesta = 彈藥全滿
~ingestw = 彈藥全滿
~ingetste = 能源全滿
~boom = 引發小型爆炸
~dansmartbomb = 螢幕上的敵人全滅
