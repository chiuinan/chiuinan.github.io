名稱：德軍總部3D (Wolfenstein 3D，英文版)
簡介：Id Software 1992年出品的第一人稱射擊遊戲，其概念源自於1981年由Muse Software
      所出品的德軍總部，描寫主角逃離德軍總部監獄城堡的故事，並買下德軍總部的商標，
      成為德軍總部系列的延續作品。遊戲最初採共享軟體方式發行，只有第一章十個關卡，
      玩家必須付費註冊以啟用其他章節的關卡，全部共有六章60個關卡。本遊戲一發行即
      熱銷，並獲獎無數，成為第一人稱射擊類型遊戲的鼻祖。遊戲公司同時也公開分享了
      遊戲原始碼，於是各種不同平台都有人移植本遊戲，同時也衍生許多MOD遊戲模組，及
      新的同類型遊戲，可說影響甚遠，實為經典中的經典遊戲。之後還發行超級升級包，
      提供地圖編輯器，可讓玩家自製關卡；以及關卡隨機生成器，可自動產生關卡，更提
      高了遊戲的可玩性，娛樂性十足，非常推薦的一款遊戲。
 
備註：本版為1.4g青衫修正版，內含地圖編輯器、關卡隨機生成器
保護：無
備註：正常版請執行PLAY.BAT，不死版請執行UPLAY.BAT
備註：推動暗格時，系統原本為2格移動，但在較快的CPU上會變成3格移動，導致有幾關的
      通道被卡死、或寶物被壓住無法撿取。可以在推動時，等暗格快移到2格處，再按下
      Pause鍵暫停，再慢慢按Pause鍵看看暗格是否確實停在2格處。不過不是每次都成功，
      最好按照攻略裡的修改碼修正。
