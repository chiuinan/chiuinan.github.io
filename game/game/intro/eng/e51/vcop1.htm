密技：

1.先將操作設定成鍵盤，啟用密技後可再設回滑鼠。以下任一密技啟動後，下次進入均不用
  再啟動（會記錄在VCop.dat中）。
2.在標題畫面出現SEGA時，按住裝填鍵不放（內定為H），輸入上下左右（內定為WSAD），會
  有一聲槍聲，OPTION裡可以設定接關（CONTINUE）可無限（FREE PLAY）。
3.輸入以上密技後，再等到標題畫面出現AM2時，按住裝填鍵不放（內定為H），輸入下上右
  左上上左右（內定為SWDAWWAD），會有一聲槍聲，OPTION裡右下角箭頭按下有增加選項，
  將GUN SELECT設成ON，則在遊戲中暫停時按裝填鍵，就可以選擇想用的槍械。

操作：

F3 = 暫停
F5 = 變更操作方式與按鍵（後面所列為內定操作鍵）
F6 = 變更螢幕顯示方式
F9 = 中止遊戲，並進行連線遊戲
Alt+F4 = 結束遊戲
滑鼠左鍵 = 射擊
滑鼠右鍵 = 裝載（連按兩次，隨時可裝填）
WS = 上下
AD = 左右
V = 射擊
G = 加速
H = 裝填（連按兩次）
Space = 開始／暫停
