名稱：雷電３ (Raiden 3，英文版/日文版)
簡介：Seibu授權MOSS於2005年開發的射擊遊戲，原為大型電玩，之後移植做PC
      版。整個遊戲以直向捲動畫面進行射擊，共有７關，攻擊火力分為主副
      兩種，只是前版可以自動鎖敵的雷射到本版變不行，實用性大為降低，
      受歡迎度也比不上雷電２。不過這款遊戲的聲光效果做得相當地好，因
      此還是具有相當的娛樂性，喜歡射擊遊戲的玩家仍可以玩玩看。

SC = 目前分數
LF = 剩餘機體數（含目前在玩的那隻）
F1 = 主武器火力形態，0=火神機炮，1=雷射，2=質子雷射
S1 = 主武器強度（0～5）
F2 = 副武器火力形態，3=核子飛彈，4=導向飛彈，5=雷達飛彈
S2 = 副武器強度（1～3），0表沒有
BM = 剩餘炸彈數

操作：（可執行Config.exe來變更）

Z = 射擊
X = 炸彈
Enter = 確認
Esc = 暫停／離開

說明：

1.選單
  Game Start：闖關模式
  Score Attack：針對某一關進行得分競賽
  Boss Rush：直接打各關魔王（必須先破關）
2.主武器：分6階段威力提升，吃到即提升
  紅色：火神機炮，廣範圍射擊
  藍色：雷射，前方一直線連續射擊
  綠色：質子雷射，可左右移動的雷射
3.副武器：分3階段威力提升，吃到即提升
  m：核子飛彈，擊中產生爆炸
  H：導向飛彈，自動追蹤敵人
  R：雷達飛彈，只會追蹤前方敵人

*** 關卡

第一關（鄉野）

魔王為兩台陸戰型火炮坦克，當損害到一定程度時，會舉起炮台開始轉動射擊。
由於後面還有第二隻，故要儘快消滅第一隻。

第二關（都市）

中間會有兩隻很硬的敵機。魔王為大型攻擊機，有兩面，會不斷放出自殺飛機，
當損害到一定程度時，則開始發射霰彈。

第三關（海戰）

中間會有兩隻很硬的大型敵機。魔王為大型戰艦，會發射多種彈體，相當密集，
要善用炸彈。

第四關（小行星帶）

中間會有一隻非常硬的四腳坦克，旁邊還有砲台，不太好對付。魔王為太空梭，
一共三截，必須全打掉，且炮火相當密集，需要花費不少時間。

第五關（太空站通道）

魔王為圓型敵機，共有三截，必須全打掉。

第六關（前哨）

中間有兩架大型敵機，和一台坦克，都頗硬。魔王為巨大敵艦，不很耐打，打
掉後會變形成飛機再打一次。

第七關（太空站）

中間有兩架大型敵機，和一個地面圓型砲台，都頗硬。魔王為巨大太空戰艦，
最初為左中右三個，打完後剩中間那個再打一次，最後變成紅色水晶生命體，
炮火很密集，又極難擊毀，非常難打。
