名稱：邊緣戰區 (SideLine，英文版)
簡介：光畫科技 1996年出品的橫向捲軸射擊遊戲，由精訊代理發行 [待補]。
 
保護：光碟
備註：本遊戲除了開頭LOGO有中文之外，其餘均為英文。
備註：光碟版本：

1.1996年台版：精訊代理版，只有DOS版。
2.1997年陸版：

為DOS、Win95二合一版，亦為官網免費提供下載的版本。不過DOS版執行上較麻煩，建議使用
Win95版的MAIN95W.EXE即可（直接在光碟上執行，XP以上系統需設成Win9x相容）。若要執行
DOS版，安裝後需進行下列設定：

(1)進入DRIVER子目錄，執行SETSOUND.EXE設定音效卡。
(2)遊戲採用Jemm管理記憶體，但光碟裡並沒有附上，必須自行取得。然後將DosBox的EMS、
   XMS都關掉，在執行遊戲前先以下列方式載入Jemm：

JEMMEX NOEMS LOAD

檔案：JEMM578B.rar = JEMM 5.78B版
操作：

Ctrl = 射擊
Shift = 炸彈
Alt = 變更副武器射擊方向
