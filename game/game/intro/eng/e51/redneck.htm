名稱：火爆鄉巴佬1／農夫也瘋狂1 (Redneck Rampage，英文版)
簡介：Xatrix 1997年出品的第一人稱射擊遊戲，由Interplay代理發行。同年推出「粗話包
      （Cuss Pack）」與「66號公路（Suckin' Grits on Route 66）」資料片。1999年將
      一代所有內容連同二代收錄在「家族集合（Family Reunion）」中 [待補]。



備註：本版含資料片，要玩主程式，請執行PLAY.BAT；要玩資料片，請執行PLAY2.BAT。
保護：無，放入光碟才有背景音樂
備註：遊戲雖可在Windows上安裝，但本身仍為DOS程式，在Windows系統執行容易有相容性的
      問題，建議使用DosBox執行。
備註：家族集合版的資料片有做部份修改，但修改不完整，會造成無法安裝，或遊戲不正常，
      不建議使用。
備註：想使用或停用粗話包，請執行SETUP.EXE，進Sound Setup，選擇RTS File Selection，
      再選Change RTS file。正常版為REDNECK.RTS，粗話版為CUSSPACK.RTS。
