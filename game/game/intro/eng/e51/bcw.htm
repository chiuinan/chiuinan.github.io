名稱：德軍總部２超越德軍總部 (Beyond Castle Wolfenstein，英文版)
簡介：Muse Software 1984年於Apple II出品的射擊冒險遊戲，在1985年時移植到PC上，為
      德軍總部系列的第二部作品。此次玩家的任務，是找到藏在櫥櫃的定時炸彈，將它放
      到希特勒的秘密會議室，再返回入口處逃離。本質上和第一代相差不多，但開箱已不
      用等待，改成猜數字密碼解鎖，比較有趣一些。不過本代已經無法無限接關，且失敗
      會自動刪除存檔，使得過關變得頗為困難。此外還有警報器連通整個城堡，喜歡殺敵
      反而更難過關。德軍制服的偽裝，也改成偽造通行證，每進一個房間都會被盤查一次，
      於是遊戲大部份的時間都花在警衛盤查與遞交通行證上，有點降低了遊戲的娛樂性。
 
保護：無
備註：CPU不能太快，否則一下子就陣亡了。DoxBox的cycles建議在200左右。

修改：存檔與房間定義GAME.DAT

位置014000h開始為主角與房間定義，每1000h byte後，會間隔1400h byte，共4個區間，合
計為4000h=16KB：

    14400-153FF
    16800-177FF
    18C00-19BFF
    1B000-1BFFF

主角狀態：

       00 01 02 03 04 05 06 07 08 09 0A 0B 0C 0D 0E 0F
014400 -- T1 T2 T3 T4 T5 -- -- -- -- -- -- -- -- -- --
014410 -- -- -- -- -- -- -- -- -- DA -- -- -- -- -- --
014420 TT TT RM LD RD -- -- YY XX -- BU -- MN -- -- --
014430 -- -- -- AL -- -- -- -- -- PA LV -- -- -- -- --

T1～T4 = 匕首、開鎖工具、急救醫療箱、定時炸彈、鑰匙的數量
DA = FFh表示手上為匕首
TT = 定時炸彈倒數時間
RM = 所在房間編號
LD = 行進方向
RD = 瞄準方向
XX,YY = 我方所在位置
BU = 子彈數
MN = 金錢數（馬克）
AL = 01表警報被啟動
PA = 通行證，最低5個位元分別表示通行證#1～#5
LV = 玩家等級，0～4

位置0144FCh以下為房間定義，由#1開始（亦為初始房間），每個房間256 byte，最多63個
房間，亦即合法的房間編號為1～63。各房間定義如下：

00h～47h = 房間形狀定義，每8 byte一行，共9行（即8x9）
48h = 不明
49h～4Dh = 分別為左邊、右邊、上邊、下邊、電梯所連接的房間編號
4Eh～7Fh = 不明
80h～FFh = 房間物品定義，16 byte一組

房間形狀定義各byte定義所在格的狀態，各位元定義為：

  01h = 左邊有牆
  02h = 右邊有牆
  04h = 上邊有牆
  08h = 下邊有牆
  10h = 所在格為電梯

房間物品定義：

  TP -- IT NO LK -- -- -- -- -- DT -- -- -- -- --

TP = 物品形態，20h表示櫥櫃，60h表示會議室
IT = 櫥櫃物品編號（00h～0Fh）
NO = 物品數量
LK = 是否上鎖
DT = 秘密通道通往的房間編號

說明：（作者－青衫）

1.地圖每次重新產生後都不一樣，必須自行摸索闖關。
2.初始子彈數=10（最多10發）、金錢=100（最多100馬克）、不定張數通行證。玩家的位置
  是在第一層入口處，目標是找到定時炸彈，然後將它放到第三層希特勒的秘密會議室前，
  最後再返回原入口處。由於定時炸彈爆炸後，警衛會全數出動搜捕，所以炸彈放置後最好
  重設時間，然後迅速返回。
3.遇到警衛盤問時，須立即上前出示通行證，有2次錯誤的機會。通行證共有5種，我方持有
  的通行證會顯示在左方，每一層的通行證都會使用同一種。也可以使用金錢賄賂警衛，若
  出現Heil!字串，便表示成功通過。若遲遲不上前出示通行證、兩次均錯、或持用手槍，
  均會引起警衛起疑。
4.起疑的警衛要立即打死，否則他會去啟動警報器，並開始攻擊。若有辦公桌的警衛，便會
  立即啟動警報器。警報器一旦啟動，便會有源源不絕的警衛開始搜捕，除非找到警報系統
  控制器予以停止，否則幾乎寸步難行。因此儘量以不讓警衛起疑為原則，比較容易過關。
5.殺死敵人後，要將之拖離主通道，否則被其他警衛看到，一樣會起疑。同時可搜尋其身上
  的物品。搜尋可能得到的物品有：

    Nothing = 無物品
    Bullets = 子彈（交換，非填滿）
    Passes = 不定張數通行證
    Marks = 金錢（馬克）

6.櫥櫃開啟時，若標示Locked表示有上鎖。要開啟有下列方法：

  (1)使用開鎖工具開啟，失敗可再試，但有可能卡死。
  (2)按下3位數字密碼開啟，按對每個數字時會有聲音，按錯時須重頭輸入。
  (3)透過射擊打開，但會使房間的警衛起疑，若櫥櫃裡有子彈、火藥或炸彈，會爆炸導致
     玩家死亡。

  櫥櫃可以得到的物品有：

    00 empty = 無物品
    01 Bullets = 子彈
    02 Money = 金錢
    03 Passes = 通行證
    04 Powder Kegs = 火藥桶（無用）
    05 A coat check = 衣帽間（無用）
    06 Art Objects = 藝術品（無用）
    07 Keys = 鑰匙
    08 Keys = 鑰匙
    09 Hitler's Diary = 希特勒的日記（無用）
    0A The Alarm Controls = 警報系統控制器
    0B A dagger = 匕首
    0C A first aid kit = 急救醫療箱
    0D A tool kit = 開鎖工具
    0E A secret passage = 秘密通道，可前往另一房間
    0F The BOMB = 定時炸彈

  除了子彈、金錢、通行證外，最多只能帶3樣物品。
7.被抓住或被敵人子彈射死時，原存檔會自動刪除，只能重頭來過，無法不斷嘗試（和一代
  不同）。
8.任務完成後，會提升玩家等級，並建立新的城堡。玩家等級共有五級，決定新城堡困難度：
    等級1 = Resistance fighter（地下鬥士）
    等級2 = Allied intelligence agent（智諜）
    等級3 = Top secret operative（密碟）
    等級4 = Master saboteur（怪諜）
    等級5 = Elite strike force spy（謎諜）
