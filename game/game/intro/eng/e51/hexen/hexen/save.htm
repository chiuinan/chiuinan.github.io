CL = 職業，0=戰士，1=牧師，2=法師
HT = 生命，最多100=64h
     直接修改可能會和顯示的數據不同步，必須切換關卡後才能正常
A1～A4 = 護甲AC值*50000h，依次為戰甲、盾牌、頭盔、項鍊
IT,IN = 寶物形態與數量，共34個
ST = 目前選擇的寶物形態
TN = 寶物總數
SN = 物品欄總數
P1～P8 = 是否有強化裝置，非0=有/剩餘作用時間（單位tics=1/35秒）
         依次為無敵、地圖、火把、飛行、不明、不明、速度、召喚
KY = 已撿到的鑰匙（各位元代表有無）
CW = 目前使用的武器
NW = 目前武器法力用盡準備替換的武器，05h表不更換
W1～W4 = 是否有該武器，1=有，依次為初始、一階、二階、終極
M1～M2 = 目前法力，依次為藍魔法、綠魔法
