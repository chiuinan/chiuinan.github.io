名稱：氙星異形 (Xenon，英文版)
簡介：Bitmap Brothers 1988年出品的射擊遊戲，一共有４關，分別為陸戰、空戰、陸戰、
      空戰，之後再循環重複。敵機被擊落時會留下一些特別物品，可加強火力或增加武器。
      而己機被擊中時，會先損失防禦值，並非一擊斃命的形態。不過這個遊戲困難度頗高，
      並不是很容易過關，而且用鍵盤非常的難操控，因此若沒有搖桿，建議還是不要玩比
      較好。

操作：

O = 向左
P = 向右
Q = 向上
A = 向下
Z = 變成坦克/飛機
Space = 發射
F3 = 暫停
F10 = 重玩
Esc = 結束遊戲
S = 音效開關

說明：

遊戲畫面右側為狀態顯示：



1.得分

2.剩餘機數

3.所在區域

4.剩餘油量（初值24）

5.坦克速度

6.子彈射程

7.武器管制表

被射中或撞到時，會減少油量。當剩餘油量為0時，機身便會爆炸結束。摧毀異形後，有時
會出現威力膠囊：

W = 機身側砲，在機身左右各加裝一門砲（去除武裝衛星）
* = 武裝衛星，最多可裝備三枚武裝衛星跟隨機身協助射機（去除機身側砲）
L = 雷射砲，在飛機上裝備雷射砲（去除機砲）
G = 機砲，在飛機上裝備機砲（去除雷射砲）
S = 水平雷射，在飛機上裝備左右水平方向的雷射砲
H = 導向飛彈，9秒內可同時發射8枚導向飛彈
A = 超級護甲，9秒內可用機身撞擊敵人
F = 油料補給，油箱加滿
P = 加強火力，增加子彈射距
R = 加速器，提升坦克引擎性能
Z = 爆破，異形和砲塔全滅

前述獲得的武器會顯示在武器管制表中：

W L S H A
B X X 0 0

BXX為武裝衛星的指示（最多３個），H（導向飛彈）和A（超級護甲）底下為其剩餘有效時間。

整個遊戲共分成四個戰區，每個戰區分成四段戰場，每兩段戰場會出現一隻守關的巨大異形
（其弱點在下方的發射孔）。失敗時均從所在段戰場起點開始，所有武器威力回到最初狀態。
每過一關剩餘隻數增加一隻，同時所有武器與威力亦回到最初狀態。

*** 異形資料

1.瓢蟲型（Ladybirds）：防禦用小型地面戰機，機動性弱，前方裝備飛彈武器。
2.豆莢型（Pods）：特攻型防衛隊，全力攔截入侵者。
3.眼球型（Eyes）：可從很遠的地方以極快速度追蹤衝撞過來。
4.蜘蛛型（Spinners）：在特定地點待命，收到攻擊訊號後才衝出攻擊。
5.發射台（Launchers）：座落通道兩側，防禦火力強大，定時發射一枚旋轉彈。
6.單擊型（Middleshooters）：裝備單管雷射，常以硬拼方式作戰。
7.旋轉彈（Rotators）：由發射台發射出來，採衝撞式攻擊。
8.側翼型（Sidewinders）：大型異形戰艦，停在通道兩側，偵測到敵人才緩慢移到中央。
9.側擊型（Sideshooters）：大型異形戰艦，裝甲比側翼型弱，機動性較高。
10.防衛砲塔（Side Guns）：座落在通道兩側，有自動追蹤目標射擊、或固定射擊範圍等多種類型。
11.新蜘蛛型（Spinners）：裝備飛彈，可飛行。
12.組織型（Cells）：一組四艘在戰場上徘徊，發現敵人便發射飛彈。
13.風箏型（Kits）：由高處高速往下衝擊。
14.唧筒（Pumps）：通道障礙物，週期性地伸縮。
15.眩目型（Blinkers）：類似防衛砲塔，但裝甲較強，一次發射三枚飛彈。
16.心型（Hearts）：行動模式變化多，有不同威力與外貌。
17.環繞型（Circle Moulds）：配備飛彈的快速戰機。
18.腦型（Brain）：配備高速飛彈的重裝甲戰機。
19.莖柱（Stalks）：反覆發射大量飛彈數次後，會由頂端放出氣泡。
20.氣泡（Bubbles）：採撞擊攻勢。
21.巨砲（Lip Guns）：擁有最強裝甲，並配備兩門雷射砲和飛彈。
