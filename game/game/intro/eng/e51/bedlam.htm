名稱：末日大戰 (Bedlam，英文版)
簡介：Mirage 1996年出品的動作射擊遊戲，由GT代理發行 [待補]。



保護：無
備註：本遊戲光碟會自動判斷系統（Dos/Windows）執行對應的程式，但原則上以DOS程式為
      主，故Windows系統只能在Win9x上執行，否則會有很多相容性問題，特別是音效卡。
備註：安裝目錄會固定在C:\MIRAGE\BEDLAM裡，包括設定檔（CONFIG.BDL和OPTIONS.BDL）。
備註：進入遊戲時，若發現滑鼠幾乎不動，可在遊戲執行前先執行下列設定：

config -set "cpu cycles=auto 7800 100%% limit 26800"

操作：

滑鼠左鍵 = 移動
滑鼠右鍵 = 攻擊

說明：遊戲一開始要打掉有閃電的建物，以關閉阻礙前進的力場。
