名稱：彈子檯／彈珠台（Solid State Pinball: Tristan，英文版）
簡介：Littlewing 1991年出品的彈珠檯遊戲，由Amtex代理發行，台灣由第三波代理進口。
      整個彈珠檯都在一個畫面中，不需要隨彈珠的位置移來移去，比較不會頭昏眼花，但
      缺點便是文字不易看清。而整個遊戲只有一個彈珠檯，變化性亦略嫌單調了些。

保護：序號+密碼，破解碼如下（任意序號皆可通過）：
      TRIST256.EXE, TRIST16.EXE
        55 8B EC 81 EC 52  (不用再輸入密碼)
        CB -- -- -- -- --
        38 07 75 09
        -- -- 90 90
      或
        38 07 75 09  (密碼隨便輸入)
        88 -- 90 90
        F9 F3 A6 74 06
        -- -- A4 EB --     
操作：

Esc = 離開遊戲
F2 = 暫停遊戲
Enter = 彈射彈珠
左右Shift = 控制彈擊棒
Space = 搖檯，搖太多次檯面會傾斜

說明：

一. 遊戲功能表，由左到右為：

1.離開遊戲
2.將最高分記錄寫到檔案中
3.打開／關關喇叭
4.遊戲人數
5.開始遊戲

二. 彈子檯特性

1.額外獲得彈子數：打下左上角第一組３個牌，右上Extra Ball燈會亮起，再
  通過該燈即可多加一個彈子數。
2.多彈子：
  (1)打下左上角第二組３個牌，上方Lock燈會亮起，再將彈子打進Lock洞，即
     可鎖住該彈子，並重新獲得一個彈子。
  (2)當彈子通過左上通道，即可釋放原鎖住的彈子。
  (3)當兩個彈子同時進行遊戲時，計分欄會加重計分，同時Jackpot燈會亮起，
     通過該燈即可加一百萬分。
3.加倍計分：最上方３個通道都通過時，所有分數均加倍計分。
4.特別計分：打下左邊４個牌，Tristan Bonus燈會亮起，再打進該洞，可獲得
  特別計分。若兩個彈子同時進行遊戲時，此計分係以百萬計。
5.額外計分：打下右邊４個牌，Tristan Bonus和Collect Bonus燈會亮起，打
  入任一洞，均可獲得額外計分。
6.完成魔法：將左邊４個牌全打倒５次，打開Magic所有燈，此時Jackpot燈會
  亮起，通過該燈即可看到一段魔法。
