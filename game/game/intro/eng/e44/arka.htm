名稱：快打磚塊１ (Arkanoid，英文版)
簡介：TATIO 1986年出的打磚塊遊戲，共出了兩代，不過好像賣得不怎麼樣。但其實是一款
      相當有趣的打磚塊遊戲，每個掉下的英文字母分別有不同的功效，同時也支援滑鼠，
      方便操控。老玩家可以回味以前打磚塊遊戲的樂趣，不過速度有點慢，不太容易死，
      玩時也需有些耐心。
 
保護：不明。
備註：undead目錄下為不死版檔案，拷貝至主目錄即成不死版。
備註：啟動參數（例如ARKANOID EMS），沒有參數會直接詢問：

E = EGA/VGA顯示卡
C = CGA顯示卡
T = Tandy顯示卡
M = 滑鼠操控
J = 搖桿操控
K = 鍵盤操控
F = 遊戲速度快
S = 遊戲速度慢

修改：ARKANOID.DAT

1.無限多隻
  FE 0E A9 02 74 0B
  C6 06 -- -- 03 90

操作：

ZXCV或左鍵 = 打擊板左移
M<>?或右鍵 = 打擊板右移
空白鍵 = 發球／發射
Esc = 暫停，之後再按：
        C = 繼續遊戲
        Q = 結束遊戲

說明：

1.每擊落一塊能源板，依顏色可得70～130分。
2.每收集一個膠囊可得1000分。
3.每擊毀一個外星幽浮可得100分。
4.得分達到20000分時，可增加一命。

*** 膠囊一覽表

B = 跳關，直接進入下一關
C = 磁石，可將球吸停在打擊板
D = 多元球，畫面多出3個球
E = 擴充打擊板，增大一倍
L = 雷射，打擊板可發射打掉磚塊
P = 加台，增加一次陣亡機會
S = 減速，球的速度變慢
