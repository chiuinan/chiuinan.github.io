名稱：連環泡 (Pick'n Pile，英文版)
簡介：Ubisoft 1990出品的益智遊戲，台灣由智冠代理進口。遊戲目標是消除畫面上的所有
      圓球，規則為垂直若都為相同顏色圓球即可消去，玩家可藉由物體交換方式進行排列
      以消除圓球 [待補]。
 
保護：無
操作：

1.主選單畫面

F1 = 切換玩家人數
F2 = 切換困難度，Easy困難度從第1關開始，Hard困難度從第50關開始
F3 = 切換遊戲控制方式
F4 = 遊戲中時間音效的開關

2.遊戲畫面

Space/滑鼠左鍵 = 選擇要交換的物品
Ins/滑鼠右鍵 = 再掉落物品
P = 暫停
Ctrl+Alt+F10 = 結束遊戲

說明：

1.圓球：必須清除的物品。
2.加分磚塊：可視為任一顏色的圓球，以便清除圓球並獲得額外加分。
3.牆壁：用來擋住圓球滾動。
4.沙漏：清除後可增加20秒。
5.鑽石：獲得的鑽石會列在螢幕下方。清除圓球得分超過1000，即會出現鑽石。
6.花瓶：無法移動的物品。
7.炸彈：爆炸後，周圍八個物品會消失。
8.火燄：無法清除，且會衍生。
9.骷髏頭：當碰到地面時，會加快時間消逝。
