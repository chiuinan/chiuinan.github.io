      智冠代理進口 [待補]。

備註：執行檔BL.EXE後面可接參數：

EH = 高解析度EGA顯示模式
EL = 低解析度EGA顯示模式
C = CGA顯示模式
H = 單色顯示模式
T = Tandy顯示模式

操作：

WEQ = 逆時針各方向旋轉
SDA = 順時針各方向旋轉
空白鍵 = 直接掉落
P = 暫停／繼續遊戲
O = 音效開關
Esc = 跳回遊戲選單
