名稱：立體西洋棋5 (Sargon 5 - World Class Chess，英文版)
簡介：Activision 1991年出品的西洋棋遊戲，下棋引擎由Spracklen撰寫。該
      下棋引擎初代於1978年推出，之後因應不同公司的商業需求，以1983年
      的三代為基礎，分別修改出了四代與五代。雖然以現今的眼光來看，其
      棋力並不算高，但它曾獲得第1～4屆電腦西洋棋大賽冠軍，為當時最強
      的電腦西洋棋程式。
 
保護：文字密碼，破解碼不明（必須輸入任一文字）
參數：

/V = 使用VGA顯示
/E = 使用EGA顯示
/T = 使用Tandy顯示
/H = 使用Tandy 640顯示
/J = 使用搖桿
/M = 使用滑鼠
/R = 使用Roland MT-32音效卡
/S = 使用Sound Blaster音效卡
/A = 使用AdLib音效卡
/Y = 使用Tandy音效卡
/P = 使用PS/1音效卡
/I = 使用PC喇叭
/N = 不使用音效卡

操作：

Esc/F1/F2/滑鼠右鍵 = 系統選單
方向鍵 = 移動游標
+/滑鼠左鍵 = 選擇棋子、選擇要移往的目標格

說明：西洋棋規則。
說明：

一、Sargon（騎士符號）

1.About Sargon（Ctrl-Z）：顯示遊戲相關訊息。
2.Game Results：顯示目前遊戲的勝負結果。

二、File

1.New Game（Ctrl-G）：使用目前的設定值重新遊戲。
2.Load Game...（Ctrl-L）：載入遊戲存檔。
3.Load Set...：棋盤設定，可選擇2D/3D模式，以及棋盤、棋子樣式。
4.Save（Ctrl-S）：儲存遊戲。
5.Save as...（Ctrl-A）：另存新檔。
6.Save moves（Ctrl-M）：以文字方式儲存棋步。
7.Print moves...（Ctrl-P）：以文字方式列印棋步。
8.Print borad：以文字方式列印棋盤佈局。
9.Adjourn：暫時結束遊戲離開。下次啟動時，會回復到原下棋的狀態。
10.Quit Sargon（Ctrl-Q）：結束遊戲離開。

三、Edit

1.Undo（Ctrl-U）：恢復被刪除的棋譜記錄。
2.Delete（Del）：刪除指定棋子之後的棋譜記錄。
3.Board setup...：殘局設定。
4.Preferences...：遊戲設定。
  (1)Mouse Rate：滑鼠移動速度。
  (2)Cursor Rate：游標閃爍速度。
  (3)3D Raising：遊戲一開始是否使用3D棋盤。
  (4)Human Hand：是否出現人類的手拿子。
  (5)Sargon Hand：是否出現Sargon的手拿子。
  (6)Auto-Start：遊戲結束是否重新開始遊戲。
  (7)Use EMS Memory：是否使用EMS記憶體。
  (8)Squirrel：
  (9)Check alert：將軍時是否提示。
  (10)Opening alert：開局時是否提示。
5.Player names...：輸入玩家姓名。
6.Sounds...：音效設定。
  (1)About Tune：選擇About Sargon後的音樂。
  (2)Title Music：啟動遊戲後的片頭音樂。
  (3)End Game：遊戲結束時的音樂。
7.Save settings...（Ctrl-D）：儲存目前的設定值。

四、Play

1.Sargon vs Human（Ctrl-N）：電腦與人類下棋。
2.Sargon vs Sargon：電腦與電腦自下。
3.Human vs Human（Ctrl-V）：人類與人類下棋。
4.Blindfold：盲下，平時棋子不顯現，只在移動時才出現。
5.Tournament（Ctrl-T）：比賽模式，不能使用悔棋、提示等各種作弊手法。
6.Odds...：讓子。
7.Mate finder...：由電腦協助尋找制勝敵人的棋步（受限於電腦思考設定）。

五、Level

1.Easy play（Ctrl-E）：停止電腦利用對手時間思考。
2.Cancel openings（Ctrl-X）：取消使用開局庫。
3.Level 1～8（Ctrl-1～8）：等級愈高，棋力愈高，但思考時間也愈久。
4.Infinite（Ctrl-9）：不限電腦思考時間。
5.Equal time：限制電腦思考時間與玩家相同。
6.By time：限制電腦固定時間思考。
7.By depth：限制電腦固定層數思考。
8.Set time...：設定電腦思考的固定時間。
9.Set depth...：設定電腦思考的固定層數。

六、Tutorial

1.Beginner mode：提示棋子可移動的位置。
2.Combination practice...：致勝棋步練習，BG開頭為初學、IM為中等、EX為專家。
  (1)無提示第1次就下對=4分。
  (2)無提示第2次才下對=3分。
  (3)無提示第3次才下對=2分。
  (4)第1次提示第1次下對=1分。
  (5)正確下對=1分。
3.End Game practice...：殘局練習。
4.Illustrative games...：對局練習，OPNGS開頭為開局、MG為中局。
5.Annotated games...：對局註解，同對局練習，下錯時會告知為何不好。

七、Windows

1.Search Window：搜尋視窗，顯示電腦思考的過程。
2.Move List：顯示棋譜記錄。
3.Clocks：顯示時間。

八、Special

1.Invert board（Ctrl-I）：棋盤翻轉，以使黑白棋子位置上下顛倒。
2.Change sides（Ctrl-C）：雙方互換執子。
3.Force move（Ctrl-F）：強迫電腦停止思考，直接下子。
4.Retract move（Ctrl-R）：悔棋（會從棋譜消除）。
5.Move back（Ctrl-B）：倒退一步。
6.Move forward（Enter）：前進一步。
7.Hint（Ctrl-H）：請電腦提示下一步棋的下法。
8.Offer draw...：提出和局要求。
9.Resign enable：允許電腦投降。
10.Resign game!...：棄子投降。
11.Notation...：對棋譜進行註解。
