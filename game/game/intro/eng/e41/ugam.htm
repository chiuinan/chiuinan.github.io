名稱：Ultimate Backgammon (英文版)
簡介：Sentient Software 1994年出品的西洋雙陸棋遊戲。雙陸棋遊戲在亞洲並不太流行，
      因此玩的人很少。遊戲要獲勝有很大的運氣成分，但遊戲的策略仍然十分重要。因此
      每次擲骰子時，玩家都要從多種選擇中選出最佳的走法。說起來簡單，但做起來不易。
      由於屬非主流遊戲，是否好玩其實因人而異。

保護：無
規則：請參見中國象棋之西洋雙陸棋篇
說明：

一開始會決定玩家的名字，右邊兩個圖案分別為新增玩家與刪除玩家。左邊兩個圖案為確認
玩家（Accept Request），以及不使用玩家（Cancel Request）。若為不使用玩家，則下棋
不會做任何記錄。

進行遊戲後，按右下開始擲骰，右邊的圖示選單為：

 Game Option（遊戲選項），按圖示選單外面回到上一層選單
　 Load/Save Game（存取檔）
　 Modem/Net Play（連網下棋）
　 Score History（顯示各玩家的分數記錄）
　 New Board（更換棋盤種類）
　 Set Defaults（設定遊戲的各項內定值）
　 Turnament（進入比賽模式，先達到分數者獲勝）
　 Log Play（儲存下棋過程）
　 Edit Game（編輯棋盤/棋子狀態）

 Switch View（2D/3D顯示切換，按滑鼠右鍵可出現圖示選單）

 Offer Double（加倍）

 Play Style（電腦棋路）
  1.Conservative Play（保守型）
  2.Aggressive Play（攻擊型）
  3.Out For Blood（血戰型）

 Backup Move（悔棋）

 Forfeit Game（投降）

 Quick To DOS（結束遊戲，回到DOS）

 Help（說明）
