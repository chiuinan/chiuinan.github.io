名稱：決戰西洋棋 (Battle Chess，英文版)
簡介：Interplay 1988年出品的西洋棋遊戲，遊戲中每顆棋子都有立體的造型
      和各自的動畫，交戰過程也有爆笑的動畫出現。由於本遊戲在商業上的
      成功，由此造成一股模仿風潮，延伸出多種以動畫為主的棋類遊戲。只
      是這些動畫不免減低了下棋的速度，而且動畫差不多就是那些，一開始
      會覺得很新穎，但下久了反而會覺得動畫很厭煩。幸好遊戲可以設定以
      傳統2D棋盤顯示，來取消此一煩人的動畫。本遊戲的另一缺點就是棋力
      非常低，在眾多電腦西洋棋遊戲裡，可說是敬陪末座，因此比較沒有什
      麼挑戰性。

參數：/COMP = 使用合成視訊顯示器
      /COM1 = 使用COM1連接Modem
      /COM2 = 使用COM1連接Modem
      /KEYB = 使用鍵盤(否則使用滑鼠)
檔案：bchess1e = 加強版(1992)，多了背景音樂
操作：

F1 = 顯示額外訊息(片頭畫面)/系統選單(遊戲畫面)
滑鼠右鍵 = 系統選單
Enter/滑鼠左鍵 = 選擇棋子、選擇要移往的目標格(合法為藍色、不合法為綠色)
F3 = 選擇要擺放的棋種（設定殘局棋盤時）
BackSpace = 刪除已擺放的棋子
F2 = 輸入Modem連線命令
Alt-J = 搖桿歸中

說明：西洋棋規則。
說明：

一、Disk

1.Load Game：載入遊戲
2.Save Game：儲存遊戲
3.New Game：重新遊戲
4.Set Up Board：設定殘局棋盤
  (1)Clear Board：清除所有的棋子
  (2)Restore Board：回到設定前的棋盤狀態
  (3)Done：完成返回
5.Quit：結束遊戲離開

二、Move

1.Force Move（Alt-F）：強迫電腦停止思考，直接下子
2.Take Back（Alt-T）：悔一手棋
3.Replay（Alt-R）：取消悔棋
4.Suggest Move（Alt-M）：請電腦建議下一步走法

三、Settings

1.Sound On/Off（Alt-S）：聲音開關
2.3D Board：使用3D棋盤與棋子
  2D Board：使用2D棋盤與棋子
3.Human Plays Red：人類下先手紅方
  IBM Plays Red：電腦下先手紅方
  Modem  Plays Red：連線者下先手紅方
4.Human Plays Blue：人類下後手藍方
  IBM Plays Blue：電腦下後手藍方
  Modem  Plays Blue：連線者下後手藍方

四、Level

1.Novice：電腦等級為新手
2.Level 1～9：電腦等級為1～9
3.Set Time：設定電腦思考時間，1～10000分鐘
