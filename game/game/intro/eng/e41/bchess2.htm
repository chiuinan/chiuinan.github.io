名稱：決戰中國象棋 (Battle Chess 2 - Chinese Chess，英文版)
簡介：Interplay 1990年出的象棋遊戲，一代則為西洋棋。遊戲中每顆棋子都有立體的造型
      和各自的動畫，交戰過程也有爆笑的動畫出現，只是這些動畫不免減低了下棋的速度。
      而且動畫差不多就是那些，一開始會覺得很新穎，但下久了反而會覺得動畫很厭煩，
      況且這個遊戲的棋力並不如將族，若要真的下象棋，還不如玩將族。

操作：

滑鼠右鍵／F1 = 系統選單
滑鼠左鍵 = 選擇棋子、選擇要移往的目標格
F1 = 顯示額外訊息(在片頭畫面時)

說明：

一、Disk

1.Load Game：載入遊戲
2.Save Game：儲存遊戲
3.New Game：重新遊戲
4.Set Up Board：設定殘局棋盤
  (1)Clear Board：清除所有的棋子
  (2)Restore Board：回到設定前的棋盤狀態
  (3)Next move blue：下一手由藍方開始
  (4)Next move red：下一手由紅方開始
  (5)Done：完成返回
5.Quit：結束遊戲離開

二、Move

1.Force Move：強迫電腦停止思考，直接下子
2.Take Back：悔一手棋
3.Replay：取消悔棋
4.Suggest Move：請電腦建議下一步走法
5.Show Layout：顯示棋盤各子狀態
6.Help Move：顯示選定子可移動位置

三、Settings

1.Sound On/Off：聲音開關
2.3D Board：使用3D棋盤與棋子
  2D Board Chinese：使用2D棋盤與中文棋子
  2D Board Roman：使用2D棋盤與羅馬字母標示棋子
3.Human Plays Blue：人類下後手藍方
  IBM Plays Blue：電腦下後手藍方
  Modem  Plays Blue：連線者下後手藍方
4.Human Plays Red：人類下先手紅方
  IBM Plays Red：電腦下先手紅方
  Modem  Plays Red：連線者下先手紅方

四、Level

1.Novice：電腦等級為新手
2.Level 1～8：電腦等級為1～8
3.Set Time：設定電腦思考時間，單位為分鐘
