名稱：美女樸克2 (Strip Poker 2，英文版)
簡介：Artworx 1988年出品的18禁撲克牌梭哈遊戲，由終結者資訊代理發行。
      遊戲規則為每人先發５張，然後下注後再決定要換那幾張牌，以更換後
      的５張為準與對手相比決定輸贏。遊戲裡可從２位美女中挑選一位挑戰，
      當任一美女的錢輸光時，便會顯示一張限制級美女圖以換取100元籌碼。
      遊戲的操作不是很方便，但由於美女圖頗養眼，因此算是可以玩玩看的
      一款遊戲。
 
保護：磁片，破解碼：
      POKER.EXE
        CD 13 72 09
        -- -- 90 90
備註：正常版請執行PLAY.BAT，必勝版請執行UPLAY.BAT
修改：DosBox在CGA或Hercules模式執行時，顯示會不正常，主要是DosBox對於
      位址40h:87h設定的值不正確，因此可在主程式執行前，先執行下列修正
      程式即可（共9 byte，做成COM執行檔）：

31 C0     XOR AX,AX
8E D8     MOV DS,AX
A3 87 04  MOV [0487],AX
CD 20     INT 20

修改：POKER.EXE

1.我方金錢不減
  83 2E A0 02 05
  90 90 90 90 90
  A1 A0 02 03 06 9A 02 03 06 AC 02 2B 06 AA 02
  -- -- -- -- -- -- -- -- -- -- -- 90 90 90 90
  A1 A0 02 2B 06 B2 02 (共2處)
  -- -- -- 90 90 90 90
  A1 A0 02 2B 06 B0 02 2B 06 B2 02
  -- -- -- 90 90 90 90 90 90 90 90
2.對方金錢不增
  A1 6D 02 03 06 9A 02 03 06 AA 02

修改：記憶體

N1 N2 N3 N4 N5 00 C1 C2 C3 C4 C5 00 N1 N2 N3 N4
N5 00 C1 C2 C3 C4 C5 00 -- -- -- -- -- -- -- --

N1～N5 = 雙方牌數字，第一組為我方，第二組為對方
C1～C5 = 雙方牌花色，第一組為我方，第二組為對方

牌數字：01～09=2～10，0Ah=J，0Bh=Q，0Ch=K，0Dh=A
牌花色：01=♠，02=♦，03=♥，04=♣

備註：CG圖有露點。
操作：

Esc鍵 = 離開遊戲

說明：



遊戲進入後可選擇兩位美女（輸入下方的英文姓名）：

1.Suzi
2.Melissa

一開始雙方各有賭資100元，某一方輸到0元時若要繼續玩便要脫去一件衣物，
以換得100元作籌碼，直到脫光為止（共4件）。

1.按空白鍵開牌，初始賭注5元。
2.第一次發完牌，按左右鍵選擇功能，然後按下Enter確定：
  (1)我方押注時
     Bet：下賭注，按左右鍵選擇05、10、15、20、25（對方可能加賭注）
     Stay：掀牌，沒有換牌機會，直接比大小（對方可能加賭注）
     Drop：放棄，直接認輸，回到步驟1
  (2)對方押注時
     Raise：加賭注，按左右鍵選擇05、10、15、20、25（對方可能再加賭注）
     Call：跟牌，繼續下一步（換牌或掀牌）
     Drop：放棄，直接認輸，回到步驟1
3.換牌時，用左右鍵選擇牌，再使用空白鍵切換是否換牌，牌蓋掉表示要更換。
  當按左右鍵顯示出FINISHED時，再按下空白鍵表示選擇結束。
4.換牌完後，雙方再次相互加賭注，最後掀牌。
5.雙方互加賭注，最多4次。
