名稱：超級國際象棋2 (Complete Chess System II，英文版)
簡介：Oxford 1994年出品的西洋棋遊戲，下棋核心由Chris Whittington撰寫。
      與前作相比，棋力有高了一些，其他功能則大同小異，因此算是一款還
      可以的西洋棋遊戲。



保護：不明
操作：

滑鼠移到畫面上方 = 系統選單
滑鼠左鍵 = 選擇棋子、選擇要移往的目標格

說明：西洋棋規則。
說明：

一、File

1.About：顯示本遊戲的相關資訊
2.New Game（Alt-N）：重新遊戲
3.Strict Game：
4.Load Game（Alt-L）：載入遊戲
5.Save Game：儲存遊戲
6.Who are you：設定下棋雙方的名字
7.Preferences：存取遊戲的設定
8.Export（Alt-E）：以文字格式儲存遊戲
9.Import：以文字格式載入遊戲
10.Quit（Alt-Q）：結束遊戲離開

二、View

1.Layout：畫面的顯示方式
2.Pieces：變更棋子圖像
3.Backgrops：顯示背景圖
4.Co-ordinates：顯示座標
5.Orientation：決定棋盤方向
6.Graphical Clock：顯示時鐘
7.Notation：決定棋譜記號方式
8.Option：顯示選項
  (1)Offer Draws：
  (2)Slide Pieces：
  (3)Must Move Touched Pieces：是否一碰到棋子就必須移動
  (4)Sound：聲音開關
  (5)Capture Animation：吃子動畫開關

三、Control

1.Swap sides（Alt-S）：雙方切換執子
2.White players：指定白棋者
3.Black players：指定黑棋者
4.Communications：網路連線
5.Move now（Alt-M）：強迫電腦停止思考，直接下子
6.Forwards（Alt-F）：取消悔棋
7.Replay（Alt-R）：回到第一手
8.Automatic（Alt-A）：
9.Take Back（Alt-B）：悔一手棋
10.Delete：
11.Learn Line：
12.Thinking：
13.Hash Tables：

四、Special

1.Hint（Alt-H）：請電腦建議下一步走法
2.Test Positions：檢測棋力
  (1)My grade：以題目測試目前設定的電腦棋力之ELO分數
  (2)Your grade：以題目測試玩家的ELO分數
  (3)Goto Next（Alt-N）：
  (4)Terminate：
  (5)Load Position：
  (6)Load Suite：
3.Set time（Alt-T）：設定電腦棋力
  (1)Blitz：快棋
  (2)Infinite：無限
  (3)Average：平均
  (4)Tournament：比賽
  (5)Matching time：單場
  (6)Exact time
  (7)Weak：等級1～10
4.Style：
5.Set clocks：設定雙方用時（重新遊戲時）
6.Offer draw：
7.Set draw：設定和局
8.Pause：暫停遊戲
9.Setup Board：設定殘局棋盤

四、Database

1.Load Database：載入棋譜資料庫
2.Close Database：關閉棋譜資料庫
3.Save：儲存棋譜資料庫
4.Save As：另存棋譜資料庫
5.Split Database：分割棋譜資料庫
6.Export Database：匯出棋譜資料庫
7.Import Database：匯入棋譜資料庫
8.Database Info：修改棋譜資料庫資訊
9.Verify database：檢驗棋譜資料庫
10.Show Hidden Games
11.Clear Filter
12.Invert Filter
13.All Games->Class：將本遊戲存入棋譜資料庫

1.Book control：設定開局庫
