名稱：創世紀前傳：阿卡拉貝／阿卡拉貝：末日世界 (Ultima 0: Akalabeth／Akalabeth: World of Doom，英文版)
簡介：Richard Garriott 1979年出品的角色扮演遊戲，為知名RPG「創世紀系列」的前身。
      遊戲以2D圖形顯示，地下迷宮則為簡易直角3D畫面顯示，採單人回合方式戰鬥。1998
      年移植至DOS，並收錄在「創世紀合輯（Ultima Collection）」光碟裡 [待補]。
  
保護：無
備註：遊戲使用的Dos Extender（CWSDPMI.EXE）無法在DosBox正常執行，會出現SIGSEGV的
      錯誤，可使用WDOSX的STUBIT.EXE改寫遊戲執行檔AK.EXE，即可正常執行。
操作：

左 = 往西走／左轉
右 = 往東走／右轉
上 = 往北走／往前走
下 = 往南走／轉身
X = 進入（城鎮等）
A = 攻擊
Q = 存檔
Ctrl-R = 讀檔
Ctrl-Q或Ctrl-E = 離開遊戲
