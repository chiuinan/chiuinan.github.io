名稱：魔戒少年1／指環王1：魔戒之王 (J.R.R. Tolkien's The Lord of the Rings, Vol. I，英文版)
簡介：Interplay 1990年出品的角色扮演遊戲，劇情改編自J.R.R. Tolkien著名的魔戒小說，
      以2D向下視角畫面呈現，採團隊回合方式戰鬥。台灣由智冠代理進口。1993年再推出
      光碟增強版（Enhanced CD-ROM），增加了影片與背音音樂。1997年將磁片版與其他遊
      戲一同收錄在「劍與鋼的世界（World of Sword and Steel）」光碟中 [待補]。
 
備註：本版為光碟增強版
保護：磁片版無保護，光碟版為光碟
檔案：lord-disk.rar = 磁片版1.3版（智冠版為1.0版，無VGA模式）
操作：

Enter = 加快劇情訊息／結束訊息
Esc = 略過劇情訊息
Space/滑鼠右鍵 = 圖形指令選單
O = 系統功能選單

1.圖形指令選單
  (上排左到右)
  A = 攻擊
  V = 檢視隊員狀態
  G = 撿取物品、購買物品、學習技能
  U = 使用物品、交易物品、丟棄物品
  S = 使用技能
  (下排左到右)
  M = 施展法術
  T = 交談
  L = 更換領隊
  C = 指定隊員
  X = 離開圖形指令選單
2.系統功能選單
  L = 讀檔
  S = 存檔
  Q = 離開遊戲
  P = 暫停遊戲
  M = 音樂開關
  X = 離開系統功能選單
