名稱：影之嶽／暗影之地 (Shadowlands，英文版)
簡介：Teque London 1992年出品的角色扮演遊戲，由Domark代理發行，台灣由智冠代理進口，
      以3D斜角畫面呈現，採團隊即時方式戰鬥。由於操作上相當繁瑣，為玩家所詬病 [待補]。
 
備註：本版為光碟版
保護：無
備註：光碟版的遊戲主程式檔較磁片版新（差異不明），其餘內容均相同（光碟版另外附有
      線上手冊）。
備註：電腦速度不可太快，否則一下子就陣亡了。
操作：

點選人物圖像的腳，再點擊地點 = 移動
點選人物圖像的手，再點擊敵人 = 攻擊
點選人物圖像的手，再點擊物品 = 拿取
點選人物圖像的頭，再點擊書本 = 閱讀
Esc = 切換人物顯示圖位置
H = 暫停遊戲
Q = 離開遊戲

說明：

1.注意火把必須熄滅後才能收入背包，否則會在背包裡繼續燃燒殆盡。
2.戰鬥後，生命會隨時間逐漸恢復（很快）。
