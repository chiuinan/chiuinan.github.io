名稱：百戰迷宮 (Dungeon Hack，英文版)
簡介：DreamForge 1993年出品的角色扮演遊戲，由SSI代理發行，台灣則由智冠代理進口。
      此遊戲可說是魔眼殺機系列的變形，因此操作方式和魔眼殺機相當類似，採用直角3D
      視覺呈現，以單人即時方式戰鬥，而地下城內的怪物也和魔眼殺機系列相同。故事則
      描寫主角接受女巫的委託，到地下城中殺死龍王，不過劇情在遊戲過程中並沒有影響，
      也可說是本遊戲完全亳無劇情可言，這點使得本遊戲根本比不上魔眼殺機系列。不過
      本遊戲最大的特色是它每次重新產生的地下城均不同，這也就是它被稱為百戰迷宮的
      原因：永遠都有新的地下城可供冒險。只是沒有劇情的支撐，單純的只是變化迷宮，
      玩久了還是會覺得很膩。如果喜歡地下城冒險的玩家，在沒有其他更好的遊戲玩時，
      倒可利用本遊戲來打發一些時間。
 
保護：文字密碼（密碼表），破解碼如下：

備註：光碟版安裝結果和磁片版相同。

        O - 直接取得新的物品，取得物品不定，且次數有限
        X - 增加經驗值

*** 攻防說明

一. 命中率

亂數  = 1～20
AC    = 防禦等級，初始值為10，愈低愈好，最低-10（後述）
THAC0 = 用來表示攻擊AC為0的敵人，命中所需亂數值，初始值為20，愈低愈好

命中所需亂數值 = 攻擊者THAC0 - 被攻擊者AC
命中率 = (20-THAC0+AC)*5%

例如：
  (1)隊員（THAC0=15）攻擊怪物（AC=3），則亂數值>=15-3=12便表示命中。
  (2)隊員（THAC0=16）攻擊怪物（AC=-2），則亂數值>=16+2=18才表示命中。

等級提升可以降低THAC0，各職業／等級的THAC0值為：

敏捷愈高，投擲或射擊時的命中率愈高，請參閱屬性說明。

魔法武器可以降低THAC0，例如+2長劍，攻擊時THAC0便減2。

5.法術修正

祝福術、祈禱術等也可以暫時降低THAC0一點。
