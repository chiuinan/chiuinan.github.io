新手提示
操作：

上下鍵 = 前進後退
左右鍵 = 左右轉
滑鼠左鍵 = 左手攻擊／拿取或放下或投擲物品／慢速捲頁
滑鼠右鍵 = 右手攻擊／拿取並自動歸入物品欄／快速捲頁
滑鼠中鍵或左右鍵一起按 = 使用物品
Tab = 目前地圖（取得日誌後）
F2 = 儲存遊戲
F3 = 文字顯示開關
F4 = 載入遊戲
F5/F6 = 降低／提高音樂音量
F7/F8 = 降低／提高音效音量
F9 = 音效開關
F10 = 離開遊戲
F11 = 亮度控制
F12 = 滑鼠靈敏度控制
Esc = 選項畫面／取消
O = 遊戲選項
I = 物品卷軸
M = 神秘魔鏡
J = 日誌
Space = 略過語音
Enter = 暫停語音
Shift-F8 = 座標顯示開關

提示：（作者－青衫）

1.怪物都有一定的活動範圍，因此打不過時可以先逃開，補完血後再回來打。
  尤其是房間內的怪，一定不會出房間，故可以開門後，在門前用投擲武器殺
  死。
2.重型武器可提升力量，輕型武器可提升敏捷，兩者都提升時才會提升健康，
  進而提升血量，故各種武器都要練，不可偏廢。尤其使用的武器技能若已是
  最高的第5級，繼續使用並無法提升力量或敏捷，必須特別注意。技能等級
  的提升視有效的攻擊次數而定，包括砍碎物品等，1級50次，2級100次，3級
  220次，4級450次，5級1000次（不同技能會略有差異），力量／敏捷的提升
  亦同。
3.物品卷軸可存放無限物品，因此可以撿的東西，最好都撿起來放入，以後可
  能都會用到。
4.所有可以打破的東西儘量打破，除了可以練武器技能外，有時也會掉出東西。
5.從妖精王國拿到Yoth Soggoth's Orb後，已無需擔心法力問題，最好儘量施
  展魔法盾、魔法護甲、迅捷術（搭配3倍、全體、延時），讓全隊一直保持在
  高保護狀態下，以減少受傷。
