名稱：勁爆水上滑板 (Wakeboarding Unleashed: Featuring Shaun Murray，英文版)
簡介：Beenox 2003年出品的滑水遊戲，由Aspyr代理發行 [待補]。



保護：光碟+序號（NHJ34VMG6CLV8ZLF）
備註：VirtualBox無法執行，虛擬機請使用VMware。
操作：

上下左右 = 移動（畫面上左下或右下箭頭即同時按左下或右下按鍵）
Pad-2 = 跳躍（Jump，即畫面上的J鍵）
Pad-4 = 抓握（Grab，即畫面上的G鍵），需跳到空中
Pad-6 = 翻滾（Invert，即畫面上的I鍵），需跳到空中
Pad-8 = 軌道滑行（Rail，即畫面上的R鍵），需跳到軌道
