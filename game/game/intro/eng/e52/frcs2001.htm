名稱：最終賽車2001／終極賽車：賽博空間2001 (Final Racing: Cyber Space 2001，英文版)
簡介：正先1997年出品的賽車遊戲，由智冠代理發行。由於遊戲為純英文，因此在國內乏人
      問津，在國外亦無名氣，加上問題一堆，難以執行，現今已成為相當罕見的遊戲 [待補]。



備註：本版為完整版修正版，可選8個賽道。
保護：光碟

4.CYBER SPACE

6.METROPOLIS

備註：原始光碟無法正常安裝，有三種方式可以進行安裝：

1.自行建立一個捷徑，目標為X:\Racing\RACING.EXE，開始位置為X:\，其中X為光碟所在磁
  碟機。執行後選擇Install即可安裝。
2.拷貝光碟裡的Racing目錄出來，並進行後述的Registry設定。
3.修正setup.exe：
  81 7C 24 44 05 00 04 00 74
  -- -- -- -- -- -- -- -- 7D

備註：非安裝移機者，必須進行下列Registry設定（假設光碟在D，安裝路徑為C:\Racing）：

[HKEY_CURRENT_USER\Software\Front FarEast\Racing]
"RacingCD"="D:\\"
"RacingDir"="C:\\Racing\\"

備註：完整版程式（211KB）必須光碟檔案非唯讀才能啟動，可將光碟檔案拷貝到某個目錄，
      再用subst命令模擬成磁碟後再安裝，但如此一來會沒有音樂。建議使用熱血版的程式
      （3.8MB），並進行後述的修正。

備註：XP或Win98系統需執行dxdiag關閉DirectDraw，然後以No Hardware Acceleration方式
      執行（或直接執行rmh.exe）。Win64請搭配dgVoodoo2或DxWnd執行（需要d3drm.dll）。
      在XP或Win64系統執行時，必須限制FPS，否則對手速度會異常的快，根本不可能獲勝。
      dgVoodoo2請在主視窗按滑鼠右鍵，選Show all sections，再點選GeneralExt，限制
      FPS limit為30。DxWnd則到Timing裡勾選Frame per Second為Limit，並填入delay為
      50。虛擬機+XP目前無法限制FPS，不建議使用。
操作：

左右 = 左右轉
上 = 加速
下 = 減速
AZ = 換檔
F1 = 切換視角
F3 = 切換輔助圖
F4 = 切換速度表

[44F984] = 目前滑鼠所在的功能（選擇賽道時），1左2右3上4下
[6236A8] = 滑鼠目前位置的圖點像素值
