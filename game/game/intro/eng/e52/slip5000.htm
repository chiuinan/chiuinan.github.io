名稱：立體狂飆 (Slipstream 5000，英文版)
簡介：Refinery 1995年出品的飛機競速戰鬥遊戲，由Gremlin代理發行 [待補]。
 
備註：本版為光碟版
保護：磁片版為手冊密碼，光碟版為光碟
備註：光碟版須在Windows上安裝，但可在DOS或Windows上執行。移機DOS時，須修改Config.ini
      裡的目錄名稱。Windows上執行上，須安裝VDMSound。
檔案：slip5000-disk.rar = 磁片版（已破解）
密技：在主畫面輸入Refinery，可以啟動作弊模式，可任選賽道（即便是灰色的），同時可
      買任何裝備（即便現金不夠）。
操作：

左右 = 左右轉
上 = 向下飛
下 = 向上飛
Space = 加速
Alt = 發射
Ctrl = 更換武器
F1～F5 = 變更視角
