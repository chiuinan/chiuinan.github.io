名稱：光速兔崽子／爵士兔 (Jazz Jackrabbit，英文版)
簡介：Epic 1994年出品的2D多向捲軸動作遊戲，採共享軟體方式銷售，台灣由宏申代理進口。
      最初共有6個章節，同年推出光碟版，再追加3個章節。同年又推出聖誕版「假日兔94
      （Holiday Hare '94）」，內含一個關卡，1995年繼續再推出一個聖誕版「假日兔95
      （Holiday Hare '95）」，內含2個關卡，其後光碟版均納入這3個關卡成為一個新的
      章節 [待補]。
 
備註：本版為最終光碟版（10個完整章節）
保護：磁片版為數字密碼，光碟版為光碟，最終光碟版完全安裝即可免光碟。磁片版破解碼：
      JAZZ.EXE
        A6 60 75 0D  (密碼隨便輸入)
        A4 -- -- 00
      或
      JAZZ.BIN
        0D 76 4C CD 21
        25 00 FF A3 E6
      修改後再將之改名為JAZZ.EXE，如此可不用輸入密碼。
檔案：jazz-disk.rar = 磁片版（6個章節）
      jazz94.rar = 假日兔94版
      jazz95.rar = 假日兔95版
修改：JAZZ.BIN（若有破解改名則為JAZZ.EXE）

1.子彈不限
  FF 8D E2 D6 A0 DC D8  (新磁片版)
  90 90 90 90 -- -- --
或
  FF 8D D6 D6 A0 CE D8  (舊磁片版)
  90 90 90 90 -- -- --

密技：遊戲中先按P暫停，再按Backspace鍵，然後再輸入下列字串（某些版本前面要加DD）：

BOUF = 無敵（暫時）
GUNHED = 獲得100個各種武器子彈和5個炸藥
BAD = 獲得小鳥
CSTRIKE = 獲得飛行板
SABLE = 我方超快速
DOOM = 增加難度（敵人變強變快）
LAMER = 跳關
HOOKER = 小關結束後，跳到Bonus關卡

操作：

上下左右 = 移動
Alt = 跳躍
Space = 射擊
Ctrl = 更換武器
P = 暫停遊戲
L = 讀檔
S = 存檔
