名稱：倫敦黑車手：警察也瘋狂／英倫狂飆之瘋狂警察 (London Racer: Police Madness，英文版)
簡介：Davilex 2005年出品的警車追捕罪犯動作模擬遊戲 [待補]。



保護：無
備註：VirtualBox無法執行，虛擬機請使用VMware。
操作：

左右 = 左右轉
上 = 加速
下 = 剎車
Z = 警燈
C = 射擊
X = 掃瞄
Enter = 重置
Esc = 暫停

說明：

1.遇可疑犯（綠箭頭），需按著掃瞄，確認是否為罪犯。
2.追逐罪犯（紅箭頭），需按著警燈，進行追緝。然後在罪犯車子進入瞄準圈時，進行射擊。
  罪犯車子毀損到一定程度，即可逮捕。
