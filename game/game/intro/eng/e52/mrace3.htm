名稱：瘋狂大飆車3：超級競賽／風之翼 (MegaRace 3: Nanotech Disaster，英文版)
簡介：Cryo 2001年出品的賽車遊戲，台灣由英特衛代理進口，但並未中文化 [待補]。



保護：無（需完全安裝）
備註：英特衛代理版日期為2001/11/28，安裝過程有中文，但遊戲內容仍為英文。
備註：VirtualBox無法執行，虛擬機請使用VMware（要設成螢幕相同色數）或Win64實機。
操作：遊戲不支援滑鼠，請使用鍵盤操作（可在OPTIONS中變更）。

上 = 加速
左右 = 轉彎
左Ctrl = 剎車
下 = 使用Bonus
左Shift = 使用模組
Q = 攻擊模組
D = 防禦模組
S = 速度模組
Space = 切換模組
F1 = 變更視角
