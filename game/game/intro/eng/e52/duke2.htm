名稱：毀滅公爵2 (Duke Nukem 2，英文版)
簡介：Apogee 1993年出品的2D多向捲軸射擊動作遊戲，共分為四個章節，其中第一章節做為
      共享軟體發佈。1997年收錄在「毀滅公爵殺戮合集（Kill-A-Ton Collection）」裡 [待補]。
 
保護：無
備註：本遊戲磁片版和光碟版內容是一樣的（光碟版多了一些廣告檔），另有Focus公司出品
      的Duke Nukem Plus光碟，除了相同的二代內容外，還加上毀滅公爵3D等遊戲的共享版。
密技：（只適用正式版）

同時按下EAT = 生命全滿，分數歸零
同時按下NUK = 獲得一個隨機武器，以及必要過關物品

操作：

Ctrl = 跳躍
Alt = 射擊
F1 = 系統選單
F2 = 存檔
F3 = 讀檔
P = 暫停遊戲
M = 音樂開關
S = 聲音開關
H = 提示開關
