名稱：雙截龍1 (Double Dragon，英文版)
簡介：Technos 1987年出品的打鬥清版動作遊戲，1988年由Arcadia移植至PC DOS，台灣則由
      智冠代理進口 [待補]。
  
備註：本版為1989年版，會明確詢問單打或雙打。
保護：無
備註：電腦速度不可太快，否則很快就陣亡了。
檔案：dd-old.rar = 1988年版（智冠版，按Pad5才會進入雙打）
操作：

      玩家1   玩家2
-------------------------
上      Q      上
下      A      下
左      O      左
右      P      右
攻擊  Space   Pad5

攻擊 = 直拳攻擊／撿拾／使用武器
攻擊+上 = 跳起（可再配合左右鍵）
攻擊+下 = 頭鎚
攻擊+後退 = 肘擊／過肩摔
攻擊+前進 = 側踢／抓人膝撞
攻擊+下+後退 = 旋風腳
攻擊+下+前進 = 跳踢
Esc = 結束遊戲
