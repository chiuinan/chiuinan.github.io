名稱：POD: Planet of Death (英文版)
簡介：Ubisoft 1997年出品的賽車遊戲，同年推出「回到地獄（Back to Hell）」擴展包，
      並與主程式合併為「黃金版（Gold）」 [待補]。



備註：本版為黃金版
保護：光碟
備註：XP以上系統執行可能會無音樂及音效。背景音樂會在開始賽程後才播放。
備註：最好使用DosBox或PCem+Win9x系統執行，否則遊戲速度會過快，難以操控。
備註：執行時若出現SetDisplayMode錯誤，或進入賽程時當機，可試著更換光碟上各種顯示
      模式的主程式看看。
操作：

一、操作鍵（可在Option裡修改）

左右鍵 = 方向盤
上鍵 = 加速
下鍵 = 減速
A = 換高速檔
Z = 換低速檔

二、功能鍵

F2 = 切換視角
F3 = 轉速表顯示開關
F4 = 時速表顯示開關
F5 = 雷達顯示開關
F6 = 損壞報告顯示開關
F7 = 各車狀況顯示開關
F8 = 用時顯示開關
Esc = 功能選單
Ctrl+Y = 重新開始
Ctrl+E = 查看其他車輛
+/- = 縮放視窗大小
Alt+Enter = 視窗／全螢幕開關
