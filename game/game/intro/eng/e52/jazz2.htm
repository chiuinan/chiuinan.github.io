名稱：光速兔崽子2／爵士兔2 (Jazz Jackrabbit 2，英文版)
簡介：Epic 1998年出品的2D多向捲軸動作遊戲，採共享軟體方式推廣，正式光碟版亦包含了
      這些共享軟體的關卡，同年推出獨立銷售的聖誕版「假日兔98（Holiday Hare '98）」。
      1999年推出「絕密檔案／秘密檔案（The Secret Files）」，增加一個章節；同年再
      推出假日兔98的增強版「聖誕紀事（The Christmas Chronicles）」，可選用Lori [待補]。



備註：本版為絕密檔案1.24版+聖誕紀事1.24版
保護：無
備註：絕密檔案版安裝完後，必須再執行Patch\patch.bat，才能在假日兔98選用Lori。此關
      卡並非實際的假日兔98關卡，純為推廣用，若真要玩需使用假日兔98或聖誕紀事光碟。
檔案：j2patch_123r.exe = 原始版1.23更新檔
操作：

左右 = 移動
Shift+左右 = 跑步
上 = 舉槍向上
下 = 趴下
Ctrl = 跳躍
Space = 射擊
Ctrl+下 = 特殊能力
Esc = 離開遊戲
