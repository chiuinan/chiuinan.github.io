名稱：驚魂塔 (Mystic Towers，英文版)
簡介：Animation FX 1994年出品的動作遊戲，由Apogee代理發行，台灣由智冠代理進口 [待補]。
 
保護：無
密技：遊戲中輸入BALDRIC，即可進入作弊模式，此時有下列按鍵：

Shift+R+二位數字 = 跳至指定房間（00～45）
Shift+H = 體力、食物、飲料加至最大值
Shift+K = 獲得所有鑰匙（逃生鑰匙除外）
Shift+W = 武器全滿
Shift+C = 金幣數變10

但只要使用前述按鍵，分數便會降至零。

操作：

上 = 上樓
BackSpace = 拉動（Pull）
Alt／滑鼠右鍵 = 跳（Jump）
Ctrl = 動作（Action），如撿起、開門、喝水、攻擊等
Space = 更換法術或武器（Select）
1～0 = 直接選定法術或武器（參見後面說明）
Esc = 回到主選單
F1 = 求助
F2 = 儲存遊戲
F3 = 讀取遊戲
F10 = 螢幕寬度切換
M = 音樂開關
S = 音效開關
+/- = 調整遊戲速度
P = 遊戲暫停

說明：

1.攻擊型法術
  1=冰凍術（Ice）
  2=硫磺術（Sulfur）
  3=施毒術（Venom Cloud）
  4=火球術（Fireball）
  5=閃電術（Lightning）
2.戰術型法術
  6=透視術（Reveal）
  7=醫療術（Heal）
  8=傳送術（Teleport）
  9=飄浮術（Levitate）
  0=炸彈（Bomb）
