名稱：龐克雙鱷 (Cool Croc Twins，英文版)
簡介：Empire 1991年出品的動作遊戲，台灣由智冠代理進口。1996年與「魔法少年」收錄在
      同一張光碟裡 [待補]。
 
保護：字串密碼，破解碼不明。
備註：光碟版內容為磁片版COOL.DAT更名為COOL.EXE直接執行（只有一個檔案）。
檔案：cct-cd.rar = 光碟版（含破解程式）
操作：

OP = 左右走（倒立時方向相反）
Space = 跳（可以壓死敵人）
Esc = 暫停（再按一次離開遊戲）
開頭動畫可連按兩次Esc略過
