名稱：機車越野賽 (Motocross／Suzuki's RM250 Motocross，英文版)
簡介：Dynamix 1989年出品的機車競速遊戲，由Gamestar代理發行，台灣由智冠代理進口 [待補]。
 
保護：無
操作：

左右 = 左右轉
G = 加速
B = 減速
G+Space = 換高速檔（一開始為空檔，最高到5檔）
B+Space = 換低速檔
Esc = 暫停遊戲
+- = 增減遊戲速度（調整比賽場的精細度）
Ctrl-J = 切換搖桿或鍵盤控制
Ctrl-R = 結束競賽，回選擇畫面
Ctrl-S = 音效開關
Ctrl-T = 音樂開關
Ctrl-Q = 離開遊戲
Tab = 跳到下個選項
