操作：

左右 = 向左右飛
上下 = 向上下飛
Enter/左Shift = 加速
Space = 發射機砲（需消耗燃料）
F1 = 顯示尚未行駛的距離
F2 = 暫停遊戲
F4 = 上下的控制鍵互換
F5 = 音效開關
Esc = 顯示排行榜畫面
Ctrl-Q = 離開遊戲
