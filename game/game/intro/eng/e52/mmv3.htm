名稱：微型賽車3／微型機器3 (Micro Machines 3，英文版)
簡介：ToolBox Design 1997年出品的微小車競速遊戲，由Codemasters代理發行，1998年移
      植至Windows [待補]。



保護：光碟
備註：XP以上系統安裝時，需設定setup.exe為Win9x相容。執行時亦需設定Intro.exe（D3D）
      或Intros.exe（軟體模擬）或micro.exe（實際遊戲程式）為Win9x相容。
備註：VirtualBox須搭配DxWnd視窗模式，否則無法支援遊戲賽車時的顯示模式320x240x32，
      或使用虛擬機VMware。
備註：非安裝移機者，需設定下列Registry（假設光碟在D，安裝目錄為C:\MMV3）：

[HKEY_LOCAL_MACHINE\SOFTWARE\Toolbox Design\Micro Machines V3]
[HKEY_LOCAL_MACHINE\SOFTWARE\WOW6432Node\Toolbox Design\Micro Machines V3]

"Drive"="D:\\"
"Path"="C:\\MMV3\\"

備註：進入遊戲後到顯示主選單需要一點時間，請耐心等候。
操作：

X,Pad2 = 左轉
C,Pad3 = 右轉
CapsLock,Ins = 加速
左Shift,Del = 剎車
A,Home = 射擊（Fire）
\,End = 跳躍
