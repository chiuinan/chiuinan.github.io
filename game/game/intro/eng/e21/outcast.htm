名稱：時空英豪 (Outcast，英文版)
簡介：Appeal 1999年出品的動作冒險遊戲，由Infogrames代理發行 [待補]。
 
備註：本版為1999/10/27版
保護：07/23版為光碟SafeDisc 1.20，10/27版已去除SafeDisc保護
操作：

左Ctrl/滑鼠左鍵 = 主動作（跳躍／攻擊／下潛），持續按著會開啟動作選單
左Shift/滑鼠右鍵 = 次動作（上升），持續按著為進入戰鬥模式
方向鍵 = 走動
左Alt+方向鍵 = 調整視角
右Alt+方向鍵 = 轉身
Space = 爬行切換
Home/End = 調整視線遠近（第一人稱／第三人稱）
PgUp/PgDn = 切換武器
0 = 空手
1～6 = 選擇武器
F1 = 按鍵說明
I = 資訊選單
B = 背包
N = 筆記
L = 字典
R/E = 增減聲音大小
D/F = 增減平衡
V/C = 增減滑鼠靈敏度
Tab = 地圖
+- = 地圖縮放
* = 調整地圖大小
Esc = 回到主選單
