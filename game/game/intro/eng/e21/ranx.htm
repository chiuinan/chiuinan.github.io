名稱：街頭小霸王 (RanX，英文版)
簡介：Ubisoft 1990年出品的動作冒險遊戲，台灣由智冠代理發行。說是動作冒險遊戲，但
      交談並不會獲得過關的必要資訊，因此也可歸類為純動作遊戲 [待補]。
 
保護：不明
操作：

1.移動

左右 = 移動
Home/PgUp = 向左右跳躍
左+Home/右+PgUp = 向左右飛躍
下 = 蹲下／坐車
上 = 進入旅館、地下鐵

2.攻擊

Space+Home/PgUp = 向左上／右上踢
Space+左/右 = 向左／右踢
Space+End/PgDn = 向左下／右下踢
Space+上(立即放開) = 查看口袋物品
Space+上(按住後放開) = 將物品丟出，距離依按住時間而定

3.其他

Enter = 進入遊戲選單
Esc = 跳出遊戲選單
Space = 選取遊戲選單
P = 暫停
