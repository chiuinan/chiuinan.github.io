名稱：魔城迷蹤1 (Ecstatica: A State of Mind，英文版)
簡介：Andrew Spencer 1994年出品的動作冒險遊戲，由Psygnosis代理發行，台灣則由松崗
      代理進口，但並未中文化。1997年推出Windows重製版 [待補]。

備註：本版為1994年DOS版
保護：光碟
備註：使用VirtualBox執行1997年Windows版時，需搭配DxWnd。
操作：

Pad-8,4,6,2 = 移動
Pad-7,9 = 攻擊
Pad-5 = 閃躲
Pad-1,3 = 撿拾
F1-F4 = 潛行模式
F5-F8 = 移動模式
F9-F12 = 跑步模式
Esc = 系統選單
C = 截圖 (Windows版)
