名稱：俠盜獵車手1／橫行霸道1／俠盜飛車 (Grand Theft Auto／Race N Chase，英文版)
簡介：DMA Design 1997年出品的動作冒險遊戲，由ASC代理發行。遊戲也同時包含了競速、
      第三人稱射擊、角色扮演等多種元素。1999年分別推出「倫敦1969（London 1969）」
      和「倫敦1961（London 1961）」兩個資料片，2004年將一、二代連同資料片，收錄在
      「經典收藏版（The Classics Collection）」光碟中 [待補]。



保護：原始版為光碟，經典收藏版為光碟SecuROM 5.03
備註：原始版遊戲可在DOS或Windows系統上執行，並採用音軌背景音樂；經典收藏版只能在
      Windows系統上執行，並改用檔案式背景音樂。
