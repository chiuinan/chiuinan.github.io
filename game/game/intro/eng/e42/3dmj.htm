名稱：3D Mahjongg (英文版)
簡介：Megaware 2000年出品的四川省麻將遊戲，也收錄在松崗代理的玩瘋樂2光碟（Family
      Collection 2）中。玩家必須拿掉左邊或右邊無相隣的牌，且上邊亦無牌壓著的相同
      牌組，直到解開所有牌組。遊戲採用3D方式顯示整個牌組，雖然可以自由旋轉視角，
      但還是不如2D方便。遊戲另一個缺點就是排列完全隨機，因此並非一定有解，常常必
      須不斷重排才能解開。不過遊戲並未限制時間、重排次數，因此最終還是可以解完。
      遊戲提供的排列形式也頗多種，拿來消磨時間也還不錯。



備註：本版為2002/07/30版，玩瘋樂2光碟裡的版本為2000/11/18
保護：無
操作：

Ctrl-Q = 重排
Ctrl-H = 提示
Ctrl-R = 悔一手
Z = 拉近
X = 拉遠
1～3 = 內定視角
Esc = 結束遊戲，回到主選單
F12 = 結束遊戲
上下左右鍵，或滑鼠右鍵按著可以調整3D視角
滑鼠移到畫面上方可出現遊戲選單
