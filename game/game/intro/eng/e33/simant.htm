名稱：模擬螞蟻 (SimAnt - The Electronic Ant Colony，英文版)
簡介：Maxis 1991年出品的模擬遊戲，曾榮獲1991年美國出版商協會頒發最佳
      教育軟體獎及最佳模擬獎。玩者可經由本遊戲了解各種螞蟻的生活習性，
      遊戲中也有黃、黑螞蟻對抗的競爭遊戲，只是整個遊戲的教育性質較重，
      想經由此遊戲獲得娛樂的玩者可能會有點失望。



保護：無。
檔案：simant_install.rar = 尚未安裝前的檔案。
備註：安裝後，系統會將安裝資訊附加在每個檔案的最後面，包括安裝日期、
      使用者名稱等資訊（有加密），共計256 byte。

操作：

一、遊戲模式



1.Tutorial Game：教學模式，採用互動式一步步教導玩家如何玩本遊戲。
2.Quick Game：快速遊戲，單一區域與紅螞蟻對抗，殺死紅螞蟻蟻后即結束。
3.Full Game：完整遊戲，多個區域（192個）與紅螞蟻、人類對抗，必須培養
  新蟻后，佔領70%以上區域，逼得屋主搬家才算獲勝。
4.Experimental：實驗模式，玩家可自由變更環境來觀察螞蟻的行為。
5.Load Game：載入之前的存檔，繼續遊戲。
6.Cancel：離開遊戲。

角色操控法（通常為黃蟻）：

1.對自己單擊滑鼠左鍵，可以在視窗中將自己置中。
2.對其他地點單擊滑鼠左鍵，可以移動操控對象。
3.滑鼠左鍵按著操控對象會出現操控選單。
4.滑鼠左鍵按著其他螞蟻，會顯示其資訊。

操控選單：

1.蟻后

Dig Hole（或按1）：挖掘蟻穴，雙擊滑鼠左鍵也有相同效果。
Start Brood（或按2）：開始產卵，玩家角色將切換成工蟻。
Help：顯示各選單作用。

2.工蟻、兵蟻

(1)對食物雙擊滑鼠左鍵，自動前往拿取或吃掉食物。
(2)對小石頭雙擊滑鼠左鍵，自動前往拿取。
(3)有拿取東西時，對地上雙擊滑鼠左鍵，即可放下。
(4)對紅蟻雙擊滑鼠左鍵，自動前往攻擊。

Recruit 5（或按1）：集結5隻螞蟻，可選用多次以集結更多螞蟻。
Recruit 10（或按2）：集結10隻螞蟻，可選用多次以集結更多螞蟻。
Release 1/2（或按3）：釋放一半螞蟻。
Release All（或按4）：釋放全部螞蟻。
Exchange（或按X）：更換玩家要操控的對象，包括蜘蛛。
Help：顯示各選單作用。

如果要搬食物，先集結螞蟻後，再到食物處釋放，釋放的螞蟻便會自動去搬食
物。如果要攻擊，亦先集結螞蟻後，再到敵人處釋放，釋放的螞蟻便會自動去
攻擊敵人（可先更換操控角色為兵蟻）。戰鬥勝負率如下（會有10%誤差）：

      工蟻    兵蟻    蟻后    雄議
----------------------------------------
工蟻  50/50   20/80   30/70   70/30
兵蟻  80/20   50/50   60/40   90/10
蟻后  70/30   40/60   50/50   80/20
雄議  30/70   10/90   20/80   50/50

3.蜘蛛

Chase（或按1）：追殺指定的螞蟻。雙擊滑鼠左鍵也有相同效果。
Hunt（或按2）：自動不斷追殺螞蟻（不定黑蟻或紅蟻）。
Exchange（或按X）：更換玩家要操控的對象。
Help：顯示各選單作用。

利用操控蜘蛛，玩家可以協助黑蟻來消滅紅蟻。又，如果玩家已在操控蜘蛛，
此時再更換操控對象為蜘蛛，則遊戲程式會因為找不到玩家在操控的螞蟻，而
另行產生出一隻黑蟻。可以利用此點來大量增加黑蟻。

九、Full Game玩法提示

1.先往右上走一段路，以便讓蟻穴靠近綠色食物，有利初期發展。
2.對地上雙擊滑鼠左鍵開始挖掘蟻穴，進入土中後，繼續對泥土雙擊滑鼠左鍵
  進行挖掘，不用挖很深。然後滑鼠左鍵按著自己，選Start Brood開始產卵，
  玩家角色切換成工蟻。
3.繼續挖掘蟻穴，稍微擴大一些後，調整蟻種控制視窗，將雄蟻比例調低。
4.切換至地表，將顯示框移到綠色食物處，雙擊綠色食物，黃蟻會自動移動到
  點擊處搬取食物（若健康度過低，會自動吃掉，此時要再重新點取）。
5.切換到黑蟻穴，雙擊通道，黃蟻會移動到點擊處放下食物。
6.等有其他工蟻出現時，便開始切換到紅蟻穴，滑鼠左鍵點擊兩下紅蟻后，讓
  黃蟻開始攻擊蟻后。最好先暫停，點選各螞蟻，找出兵蟻後，再交換成兵蟻。
7.紅蟻后死亡後（可能需要攻擊多次），要再清除紅蟻卵，避免繼續孵化。接
  著調高雄蟻比例，並繼續清除現存紅蟻。
8.到庭院圖選擇Mate（交配），等出現新蟻后，再Place（置放）到其他區域
  （三格以內）。如果有其他紅蟻佔領區域，則要儘量往紅蟻佔領區方向置放。
9.重複步驟8，不斷產出新蟻后置放到各區域發展。等到與紅蟻區重疊時，再
  雙擊進入該區協助清除紅蟻。等佔領區域夠多時，便能看到結局。

十、教育模式過程

1.雙擊泥土，挖掘蟻穴。再次雙擊泥土，挖掘蟻穴。
2.點擊上面草地，移往地表。
3.點擊其他土地，進行移動。點擊自己，進行顯示置中。
4.將滑鼠移出螢幕外，以捲動視窗。
5.選擇Windows-Map選單，切換至地圖視窗。將顯示方框移到綠色食物上，再
  選擇Windows-Edit選單，切換至編輯視窗。移動到食物旁，雙擊食物吃掉。
6.雙擊搬起食物，移動回蟻穴，雙擊通道將之放下。
7.回到地表，雙擊小石頭搬起，再雙擊任一地方放下。
8.按著自己，選擇Recruit 10，集結一些螞蟻。按下0並移動，施放警戒嗅跡。
9.按著自己，選擇Exchange，再挑選任一隻黑蟻。
10.切換至地圖視窗，按下Yard鈕顯示庭院全景。再按下House鈕顯示屋內全景。
11.按下Patch鈕切回鳥瞰圖，再按下B鈕開啟行為控制視窗，移動中間圓點。
12.切換至地圖視窗，按下？鈕，顯示功能說明。
13.按下右下狀況圖，顯示蟻種資訊。
14.按下C鈕開啟蟻種控制視窗。
15.切換至編輯視窗，按左邊黃蟻鈕，進行顯示置中。
16.按著自己，選擇Recruit 10共五次。往右移到紅蟻巢，殺掉紅蟻后。
